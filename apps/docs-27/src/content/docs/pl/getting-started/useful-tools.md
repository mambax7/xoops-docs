| Oprogramowanie | Opis |
| --- | --- |
| 7Zip www.7zip.org | Program open source do pakowania/rozpakowywania plików. |
| WinSCP www.WinSCP.net | WinSCP to otwarty klient SFTP i klient FTP dla Windows. Jego główną funkcją jest bezpieczny transfer plików między komputerem lokalnym i zdalnym. |
| WAMP Server www.wampserver.com | WampServer to środowisko deweloperskie sieci web dla Windows. Pozwala tworzyć aplikacje internetowe za pomocą Apache, PHP i bazy danych MySQL. Posiada również PHPMyAdmin do łatwego zarządzania bazami danych. WampServer instaluje się automatycznie (instalator) i jego użycie jest bardzo intuicyjne. Będziesz mógł dostrajać swój serwer bez nawet dotykania plików ustawień. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP to łatwa w instalacji dystrybucja Apache zawierająca MySQL, PHP i Perl. XAMPP jest naprawdę bardzo łatwy do zainstalowania i użycia - wystarczy pobrać, rozpakować i uruchomić. |
| KeyPass www.keepass.info | KeePass to bezpłatny menedżer haseł o otwartym kodzie źródłowym, który pomaga w bezpiecznym zarządzaniu hasłami. Możesz umieścić wszystkie hasła w jednej bazie danych, która jest zablokowana jednym kluczem głównym lub plikiem klucza. Dlatego musisz pamiętać tylko jedno hasło główne lub wybrać plik klucza, aby odblokować całą bazę danych. Bazy danych są szyfrowane przy użyciu najlepszych i najbezpieczniejszych algorytmów szyfrowania, które są obecnie znane (AES i Twofish). |

| Package Web Browsers [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === Aplikacje USB ==== - Firefox + rozszerzenia webdevelopper i XooFoo (wersja stabilna i wersja rozwojowa), IETester 0.4.5, Opera (wersja stabilna i wersja rozwojowa), OperaMobile Emulator, Safari, Google Chrome + rozszerzenie webdevelopper (wersja stabilna i wersja rozwojowa), Maxthon (wersja stabilna i wersja rozwojowa), SeaMonkey, Avant browser, Flock i Elinks |
| Uniform Server [http://uniformserver.com](http://uniformserver.com) | "Dlaczego używać Uniform Server zamiast XAMPP lub WAMP lub EASYPHP? 1. Bezpieczeństwo 2. Lekka waga (xampp 51MB, XAMPlite 28MB. Uniform Server 9.95MB). 3. Łatwiejsze w użyciu. 4. XAMPP (na przykład) ma wiele niepotrzebnych funkcji, zwłaszcza dla społeczności Xoops. Pomimo małego rozmiaru Uniform Server, ma wszystko czego potrzebujemy./ /Instalacja Uniform Server da ci w ostatnich wersjach: - Bazę danych MySQL. - phpMyAdmin do administrowania bazą danych MySQL. - Hosting Apache. - PHP (Ma ponad 100 poprawek błędów więcej niż to, co otrzymujesz podczas pobierania XAMPP). - eAccelerator, aby przyspieszyć witrynę. |
| Uniform Server by XooFoo [http://goo.gl/LGx3V](http://goo.gl/LGx3V) | Uniform Server zoptymalizowany dla XOOPS |
| Mini Server (zaledwie 5MB) by XooFoo [http://goo.gl/Ke0LW](http://goo.gl/Ke0LW) | Podstawowy przenośny serwer produkcyjny z dodaną mocą PHP, umożliwiający dynamiczne strony internetowe oprócz statycznych stron htm i html. Zaprojektowany, aby umożliwić wiele serwerów działających na tym samym PC lub dyskach USB, będą one wymagać oddzielnego wirtualnego dysku automatycznie utworzonego. Obsługa MySQL jest włączona, co dodatkowo zwiększa możliwości tego serwera. |
