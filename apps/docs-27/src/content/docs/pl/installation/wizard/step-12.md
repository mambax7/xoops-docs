Ta strona zbiera Twój wybór dla początkowego motywu nowej witryny. Motywy kontrolują wizualny charakter Twojej witryny. Motywy mogą być zmieniane w dowolnym momencie, jeśli chcesz.

Po wpisaniu żądanych informacji i poprawieniu wszelkich problemów kliknij przycisk "Kontynuuj", aby przejść dalej.

![XOOPS Installer Select Theme](/xoops-docs/2.7/img/installation/installer-12.png)

## Dane zebrane w tym kroku

### Wybierz motyw domyślny

#### Motyw domyślny

Wybierz początkowy motyw dla witryny z listy. Zrzut ekranu każdego motywu jest wyświetlany w momencie jego wybrania.

XOOPS 2.7.0 jest dostarczany z kilkoma motywami front-end poza kutią:

* **default** — minimalny motyw bazowy, przydatny jako punkt wyjścia do dostosowania.
* **xbootstrap5** — motyw Bootstrap 5, zalecany dla większości nowych witryn.
* **xswatch5** — warianty Bootswatch 5; zbiór stylizowanych podmotywów opartych na Bootstrap 5.
* **xtailwind** — motyw oparty na Tailwind CSS + DaisyUI z 35 paletami i interaktywnością Alpine.js
* **xtailwind2** — artystycznie ukierunkowany motyw siostrzany z wyselekcjonowanymi paletami i udoskonalonym stylem modułu

Możesz zmienić motyw w dowolnym momencie z obszaru administracyjnego, a dodatkowe motywy mogą być przesyłane do katalogu `themes/`.
