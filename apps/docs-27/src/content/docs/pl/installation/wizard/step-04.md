title: "Ustawienia ścieżki"
---

Ta strona zbiera informacje o systemie plików i adresie internetowym, które XOOPS potrzebuje do działania. Instalator spróbuje zgadnąć te informacje na podstawie adresu URL, którego używasz do dostępu, oraz lokalizacji uruchomionych skryptów PHP.

Po przejrzeniu i poprawieniu wszelkich problemów kliknij przycisk "Kontynuuj", aby przejść dalej.

![XOOPS Installer Path Settings](/xoops-docs/2.7/img/installation/installer-04-02.png)

## Dane zebrane w tym kroku

### Ścieżki fizyczne XOOPS

#### Ścieżka fizyczna głównego katalogu dokumentów XOOPS

Ścieżka fizyczna do katalogu dokumentów XOOPS (serwowanego) BEZ końcowego ukośnika

#### Katalog plików danych XOOPS

Ścieżka fizyczna do katalogu plików danych XOOPS (zapisywalnego) BEZ końcowego ukośnika. Umieść folder poza katalogiem dokumentów XOOPS, aby był bezpieczny.

#### Katalog biblioteki XOOPS

Ścieżka fizyczna do katalogu biblioteki XOOPS BEZ końcowego ukośnika. Umieść folder poza katalogiem dokumentów XOOPS, aby był bezpieczny.

### Lokalizacje internetowe

#### Lokalizacja witryny (adres URL)

Główny adres URL, który będzie używany do dostępu do instalacji XOOPS. XOOPS będzie używać go do budowania wszystkich adresów URL wewnątrz XOOPS. Jeśli chcesz witrynę _https_, upewnij się, że ją określisz. Jeśli chcesz lub nie chcesz wiodącego _www._ w domenie witryny, upewnij się, że określisz ją jak chcesz.

Domyślnie będzie to adres URL używany do uruchomienia instalatora.

#### Domena pliku cookie dla witryny

Domena do ustawiania plików cookie. Może być pusta, pełny host z adresu URL (www.example.com), lub zarejestrowana domena bez poddomen (example.com) do udostępniania między poddomenami (www.example.com i blog.example.com).

## Błędy

Jeśli jakiekolwiek wprowadzone ścieżki nie mogą być znalezione, zostanie wyświetlony błąd. Popraw wszelkie problemy przed kontynuowaniem.

![XOOPS Installer Path Settings Error](/xoops-docs/2.7/img/installation/installer-04-01.png)
