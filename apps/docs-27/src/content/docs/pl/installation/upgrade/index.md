Ta sekcja zawiera instrukcje aktualizacji istniejącej witryny XOOPS. Kreator aktualizacji zachowuje dane, moduły i motywy, jednocześnie aktualizując pliki jądra.

## Ścieżka aktualizacji

| Krok | Strona | Co robić |
|------|--------|------------|
| 1 | [Przygotowania](./step-01/) | Kopia zapasowa, przejrzyj dziennik zmian, pobierz nowe wydanie |
| 2 | [Sprawdzenie preflightu](./preflight/) | Weryfikuj kompatybilność przed uruchomieniem aktualizacji |
| 3 | [Uruchomienie aktualizacji](./step-02/) | Wykonaj kreatora aktualizacji |
| 4 | [Rozwiązywanie problemów](./step-03/) | Rozwiąż typowe problemy aktualizacji |
| 5 | [Po aktualizacji](./step-04/) | Weryfikuj, aktualizuj moduły, wyczyść pamięć podręczną |

:::tip
Zawsze utwórz **pełną kopię zapasową bazy danych i kopię zapasową pliku** przed uaktualnieniem. Aktualizacja jest nieodwracalna bez kopii zapasowej.
:::
