Ta razdelek vas vodi skozi nadgradnjo obstoječega mesta XOOPS. Čarovnik za nadgradnjo med posodabljanjem osnovnih datotek ohrani vaše podatke, module in teme.

## Pot nadgradnje

| korak | Stran | Kaj storiti |
|------|------|------------|
| 1 | [Pripravki](./step-01/) | Varnostno kopiranje, pregled dnevnika sprememb, prenos nove izdaje |
| 2 | [Preverjanje pred letom](./preflight/) | Pred zagonom nadgradnje preverite združljivost |
| 3 | [Izvajanje nadgradnje](./step-02/) | Zaženite čarovnika za nadgradnjo |
| 4 | [Odpravljanje težav](./step-03/) | Razrešite pogoste težave pri nadgradnji |
| 5 | [Po nadgradnji](./step-04/) | Preveri, posodobi module, počisti predpomnilnik |

:::namig
Pred nadgradnjo vedno naredite **polno varnostno kopijo baze podatkov in varnostno kopijo datotek**. Nadgradnja je nepovratna brez varnostne kopije.
:::
