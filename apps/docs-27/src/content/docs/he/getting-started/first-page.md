# יצירת הדף הראשון שלך ב-XOOPS

למד כיצד ליצור, לעצב ולפרסם את קטע התוכן הראשון שלך ב-XOOPS.

## הבנת תוכן XOOPS

### מהו Page/Post?

ב-XOOPS, התוכן מנוהל באמצעות מודולים. סוגי התוכן הנפוצים ביותר הם:

| הקלד | תיאור | מקרה שימוש |
|---|---|---|
| **עמוד** | תוכן סטטי | אודותינו, יצירת קשר, שירותים |
| **Post/Article** | תוכן עם חותמת זמן | חדשות, פוסטים בבלוג |
| **קטגוריה** | ארגון תוכן | תוכן הקשור לקבוצה |
| **תגובה** | משוב משתמשים | אפשר אינטראקציה של מבקרים |

מדריך זה מכסה יצירת page/article בסיסי באמצעות מודול תוכן ברירת המחדל של XOOPS.

## גישה לעורך התוכן

### מלוח הניהול

1. היכנס לפאנל הניהול: `http://your-domain.com/xoops/admin/`
2. נווט אל **תוכן > דפים** (או מודול התוכן שלך)
3. לחץ על "הוסף דף חדש" או "פוסט חדש"

### ממשק קצה (אם מופעל)

אם ה-XOOPS שלך מוגדר לאפשר יצירת תוכן בחזית:

1. היכנס כמשתמש רשום
2. עבור לפרופיל שלך
3. חפש את האפשרות "שלח תוכן".
4. בצע את אותם השלבים הבאים

## ממשק עורך תוכן

עורך התוכן כולל:


## מדריך שלב אחר שלב: יצירת הדף הראשון שלך

### שלב 1: גש לעורך התוכן

1. בלוח הניהול, לחץ על **תוכן > דפים**
2. לחץ על **"הוסף דף חדש"** או על **"צור"**
3. תראה את עורך התוכן

### שלב 2: הזן את כותרת העמוד

בשדה "כותרת", הזן את שם הדף שלך:

```
Title: Welcome to Our Website
```

שיטות עבודה מומלצות לכותרים:
- ברור ותיאורי
- כלול מילות מפתח אם אפשר
- 50-60 תווים אידיאלי
- הימנע מ-ALL CAPS (קשה לקריאה)
- היה ספציפי (לא "עמוד 1")

### שלב 3: בחר קטגוריה

בחר היכן לארגן את התוכן הזה:

```
Category: [Dropdown ▼]
```

האפשרויות עשויות לכלול:
- כללי
- חדשות
- בלוג
- הודעות
- שירותים

אם קטגוריות לא קיימות, בקש מהמנהל ליצור אותן.

### שלב 4: כתוב את התוכן שלך

לחץ באזור עורך התוכן והקלד את הטקסט שלך.

#### עיצוב טקסט בסיסי

השתמש בסרגל הכלים של העורך:

| כפתור | פעולה | תוצאה |
|---|---|---|
| **ב** | מודגש | **טקסט מודגש** |
| *אני* | נטוי | *טקסט נטוי* |
| <u>U</u> | קו תחתון | <u>טקסט עם קו תחתון</u> |

#### שימוש ב-HTML

XOOPS מאפשר תגי HTML בטוחים. דוגמאות נפוצות:

```html
<!-- Paragraphs -->
<p>This is a paragraph.</p>

<!-- Headings -->
<h1>Main Heading</h1>
<h2>Subheading</h2>

<!-- Lists -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<!-- Bold and Italic -->
<strong>Bold text</strong>
<em>Italic text</em>

<!-- Links -->
<a href="https://example.com">Link text</a>

<!-- Line breaks -->
<br>

<!-- Horizontal rule -->
<hr>
```

#### בטוח HTML דוגמאות

**תגים מומלצים:**
- פסקאות: `<p>`, `<br>`
- כותרות: `<h1>` עד `<h6>`
- טקסט: `<strong>`, `<em>`, `<u>`
- רשימות: `<ul>`, `<ol>`, `<li>`
- קישורים: `<a href="">`
- ציטוטים בלוק: `<blockquote>`
- טבלאות: `<table>`, `<tr>`, `<td>`

**הימנע מהתגים האלה** (ייתכן שיהיו מושבתים מטעמי אבטחה):
- סקריפטים: `<script>`
- סגנונות: `<style>`
- Iframes: `<iframe>` (אלא אם כן מוגדר)
- טפסים: `<form>`, `<input>`

### שלב 5: הוסף תמונות

#### אפשרות 1: הוסף תמונה URL

שימוש בעורך:

1. לחץ על הלחצן **הוסף תמונה** (סמל תמונה)
2. הזן תמונה URL: `https://example.com/image.jpg`
3. הזן טקסט חלופי: "תיאור תמונה"
4. לחץ על "הוסף"

HTML שווה ערך:

```html
<img src="https://example.com/image.jpg" alt="Description">
```

#### אפשרות 2: העלה תמונה

1. העלה תמונה ל-XOOPS תחילה:
   - עבור אל **תוכן > מנהל מדיה**
   - העלה את התמונה שלך
   - העתק את התמונה URL

2. בעורך התוכן, הכנס באמצעות URL (למעלה השלבים)

#### שיטות עבודה מומלצות לתמונה

- השתמש בגדלי קבצים מתאימים (בצע אופטימיזציה של תמונות)
- השתמש בשמות קבצים תיאוריים
- כלול תמיד טקסט חלופי (נגישות)
- פורמטים נתמכים: JPG, PNG, GIF, WebP
- רוחב מומלץ: 600-800 פיקסלים לתוכן

### שלב 6: הטמעת מדיה

#### הטמע סרטון מ-YouTube

```html
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  frameborder="0" allowfullscreen>
</iframe>
```

החלף את `VIDEO_ID` במזהה הסרטון של YouTube.

**כדי למצוא מזהה סרטון YouTube:**
1. פתח סרטון ביוטיוב
2. URL הוא: `https://www.youtube.com/watch?v=VIDEO_ID`
3. העתק את המזהה (תווים אחרי `v=`)

#### הטמע וידאו מ-Vimeo

```html
<iframe src="https://player.vimeo.com/video/VIDEO_ID"
  width="640" height="360" frameborder="0"
  allow="autoplay; fullscreen" allowfullscreen>
</iframe>
```

### שלב 7: הוסף מטא תיאור

בשדה "תיאור", הוסף סיכום קצר:

```
Description: Learn how to get started with our website.
This page provides an overview of our services and how we can help you.
```

**שיטות מומלצות לתיאור מטא:**
- 150-160 תווים
- כלול מילות מפתח עיקריות
- צריך לסכם במדויק את התוכן
- משמש בתוצאות של מנוע החיפוש
- הפוך אותו למשכנע (משתמשים רואים זאת)

### שלב 8: הגדר אפשרויות פרסום

#### פרסם סטטוס

בחר סטטוס פרסום:

```
Status: ☑ Published
```

אפשרויות:
- **פורסם:** גלוי לציבור
- **טיוטה:** גלויה רק למנהלים
- **בהמתנה לסקירה:** ממתין לאישור
- ** בארכיון:** מוסתר אך נשמר

#### נראות

הגדר מי יכול לראות את התוכן הזה:

```
Visibility: ☐ Public
           ☐ Registered Users Only
           ☐ Private (Admin Only)
```

#### תאריך פרסום

הגדר מתי התוכן הופך גלוי:

```
Publish Date: [Date Picker] [Time]
```

השאר בתור "עכשיו" כדי לפרסם מיד.

#### אפשר הערות

הפעל או השבת הערות מבקרים:

```
Allow Comments: ☑ Yes
```

אם מופעל, מבקרים יכולים להוסיף משוב.

### שלב 9: שמור את התוכן שלך

אפשרויות שמירה מרובות:

```
[Publish Now]  [Save as Draft]  [Schedule]  [Preview]
```

- **פרסם עכשיו:** הפוך לגלוי מיד
- **שמור כטיוטה:** שמור על פרטיות לעת עתה
- **לוח זמנים:** פרסם בעתיד date/time
- **תצוגה מקדימה:** ראה איך זה נראה לפני השמירה

לחץ על בחירתך:

```
Click [Publish Now]
```

### שלב 10: אמת את הדף שלך

לאחר הפרסום, אמת את התוכן שלך:

1. עבור אל דף הבית של האתר שלך
2. נווט לאזור התוכן שלך
3. חפש את הדף החדש שלך שנוצר
4. לחץ כדי להציג אותו
5. בדוק:
   - [ ] התוכן מוצג כהלכה
   - [ ] מופיעות תמונות
   - [ ] העיצוב נראה טוב
   - [ ] קישורים עובדים
   - [ ] הכותרת והתיאור נכונים

בצע אופטימיזציה של התוכן שלך למנועי חיפוש:

### כותרת

- 50-60 תווים
- ייחודי לכל עמוד

- כלול מילת מפתח באופן טבעי

- כוון ל-300+ מילים עבור מאמרים

- שמור על URLs קצר ותיאורי
- השתמש במקפים כדי להפריד בין מילים
- הימנע מתווים מיוחדים

## פתרון בעיות ביצירת תוכן

### התוכן לא מופיע

**סימפטום:** הדף שפורסם לא מופיע באתר

**פתרון:**
1. בדוק את סטטוס הפרסום: צריך להיות "מפורסם"
2. בדוק את תאריך הפרסום: צריך להיות נוכחי או עבר
3. בדוק נראות: צריך להיות "ציבורי"
4. נקה cache: ניהול > כלים > נקה cache
5. בדוק הרשאות: לקבוצת משתמשים חייבת להיות גישה

### העיצוב לא עובד

**סימפטום:** HTML תגים או עיצוב מופיעים כטקסט

**פתרון:**
1. ודא ש-HTML מופעל בהגדרות המודול
2. השתמש בתחביר HTML תקין
3. סגור את כל התגים: `<p>Text</p>`
4. השתמש בתגיות מותרות בלבד
5. השתמש בישויות HTML: `&lt;` עבור `<`, `&amp;` עבור `&`

### תמונות לא מוצגות

**סימפטום:** תמונות מציגות סמל שבור

**פתרון:**
1. ודא שהתמונה URL נכונה
2. בדוק שקובץ התמונה קיים
3. ודא הרשאות מתאימות לתמונה
4. נסה להעלות תמונה ל-XOOPS במקום זאת
5. בדוק אם יש חסימה חיצונית (ייתכן שצריך CORS)

### בעיות בקידוד תווים

**סימפטום:** תווים מיוחדים מופיעים כקשקוש

**פתרון:**
1. שמור את הקובץ כקידוד UTF-8
2. ודא שערכת התווים של העמוד היא UTF-8
3. הוסף לראש HTML: `<meta charset="UTF-8">`
4. הימנע מהעתק-הדבק מ-Word (השתמש בטקסט רגיל)

## שיטות עבודה מומלצות לזרימת תוכן

### תהליך מומלץ

1. **כתוב תחילה בעורך:** השתמש בעורך תוכן מנהל
2. **תצוגה מקדימה לפני פרסום:** לחץ על לחצן תצוגה מקדימה
3. **הוסף מטא נתונים:** כותרת מלאה, תיאור, תגים
4. **שמור כטיוטה תחילה:** שמור כטיוטה כדי למנוע אובדן עבודה
5. **סקירה סופית:** קרא שוב לפני הפרסום
6. **פרסם:** לחץ על פרסם כשתהיה מוכן
7. **אמת:** בדוק באתר חי
8. **ערוך במידת הצורך:** בצע תיקונים במהירות

### בקרת גרסה

שמור תמיד גיבויים:

1. **לפני שינויים גדולים:** שמור כגרסה חדשה או כגיבוי
2. **ארכיון תוכן ישן:** שמור גרסאות שלא פורסמו
3. **תאריך את הטיוטות שלך:** השתמש בשם ברור: "Page-Draft-2025-01-28"
