## סקירה כללית

Publisher הוא מודול ניהול התוכן המובהק עבור XOOPS, שהתפתח מ-SmartSection והפך לפתרון הבלוג והחדשות העשיר ביותר בתכונות. הוא מספק כלים מקיפים ליצירה, ארגון ופרסום תוכן עם תמיכה מלאה בזרימת עבודה עריכה.

**דרישות:**
- XOOPS 2.5.10+
- PHP 7.1+ (PHP 8.x מומלץ)

## 🚀 התחלה מהירה

### 1. התקנה
```bash
# Download from GitHub
git clone https://github.com/XoopsModules25x/publisher.git

# Copy to modules directory
cp -r publisher /path/to/xoops/htdocs/modules/
```
לאחר מכן התקן דרך XOOPS Admin → מודולים → התקן.

### 2. צור את הקטגוריה הראשונה שלך

1. עבור אל **אדמין → Publisher → קטגוריות**
2. לחץ על **הוסף קטגוריה**
3. מלא:
   - **שם**: חדשות
   - **תיאור**: חדשות ועדכונים אחרונים
   - **תמונה**: העלה תמונת קטגוריה
4. שמור

### 3. צור את המאמר הראשון שלך

1. עבור אל **אדמין → Publisher → מאמרים**
2. לחץ על **הוסף מאמר**
3. מלא:
   - **כותרת**: ברוכים הבאים לאתר שלנו
   - **קטגוריה**: חדשות
   - **תוכן**: תוכן המאמר שלך
4. הגדר **סטטוס**: פורסם
5. שמור

## 🔄 הגירה

### מ-SmartSection

Publisher כולל כלי העברה מובנה:

1. עבור אל **אדמין ← Publisher ← ייבוא**
2. בחר **SmartSection** כמקור
3. בחר אפשרויות ייבוא:
   - קטגוריות
   - מאמרים
   - הערות
4. לחץ על **ייבוא**

### מתוך מודול חדשות

1. עבור אל **אדמין ← Publisher ← ייבוא**
2. בחר **חדשות** כמקור
3. קטגוריות מפה
4. לחץ על **ייבוא**
