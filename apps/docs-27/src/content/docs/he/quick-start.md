## דרישות

| רכיב | מינימום | מומלץ |
|------------|------------------------|----------------|
| PHP | 8.2 | 8.4+ |
| MySQL | 5.7 | 8.0+ |
| MariaDB | 10.4 | 10.11+ |
| שרת אינטרנט | Apache 2.4 / Nginx 1.20 | יציב אחרון |

## הורד

הורד את המהדורה האחרונה מ-[GitHub מהדורות](https://github.com/XOOPS/XoopsCore27/releases).
```bash
# Or clone directly
git clone https://github.com/XOOPS/XoopsCore27.git mysite
cd mysite
```
## שלבי התקנה

1. **העלה קבצים** לשורש המסמך של שרת האינטרנט שלך (למשל `public_html/`).
2. **צור מסד נתונים MySQL** ומשתמש עם הרשאות מלאות עליו.
3. **פתחו את הדפדפן שלכם** ונווט לדומיין שלכם - תוכנית ההתקנה XOOPS מתחילה אוטומטית.
4. **עקוב אחר האשף בן 5 השלבים** - הוא מגדיר נתיבים, יוצר טבלאות ומגדיר את חשבון הניהול שלך.
5. **מחק את התיקיה `install/`** כאשר תתבקש. זה חובה לביטחון.

## ודא את ההתקנה

לאחר ההגדרה, בקר ב:

- **עמוד קדמי:** `https://yourdomain.com/`
- **פאנל ניהול:** `https://yourdomain.com/xoops_data/` *(נתיב שבחרת במהלך ההתקנה)*
