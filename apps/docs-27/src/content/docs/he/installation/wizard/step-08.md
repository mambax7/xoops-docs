דף זה מציג את התוצאות של יצירת הטבלאות XOOPS במסד הנתונים.

לאחר בדיקה ותיקון של בעיות כלשהן, בחר בלחצן "המשך" כדי להמשיך.

## על הצלחה

הקטע _יצירת טבלאות מסד נתונים_ מציג את טבלאות מסד הנתונים שנוצרו.

![XOOPS Installer Table Creation](/xoops-docs/2.7/img/installation/installer-08.png)

## שגיאות

כל שגיאה מצביעה על בעיות בהרשאה הקשורות לחשבון המשתמש MySQL במסד הנתונים and/or MySQL מסד הנתונים שהוזן בשלבים הקודמים. אתה יכול לחזור למקטעים אלה באמצעות אפשרויות חיבור מסד נתונים ותצורת מסד נתונים בתפריט.
