דף זה אוסף את המידע המשמש ליצירת חשבון המנהל הראשי (AKA מנהל האתר).

לאחר הזנת המידע המבוקש ותיקון כל בעיה, בחר בלחצן "המשך" כדי להמשיך.

![XOOPS Installer Initial Settings](/xoops-docs/2.7/img/installation/installer-09.png)

## נתונים שנאספו בשלב זה

### חשבון מנהל

#### התחברות למנהל מערכת

זה יהיה שם המשתמש שישמש את המנהל לכניסה למערכת XOOPS.

#### דואר אלקטרוני של מנהל מערכת

זוהי כתובת הדוא"ל המשמשת את מערכת XOOPS כדי לתקשר עם המנהל. (שים לב, זה נדרש עבור סיסמה recovery/reset.)

#### סיסמת מנהל

זוהי הסיסמה לשימוש עבור חשבון המנהל,

#### אשר את הסיסמה

השתמש בשדה זה כדי לאשר את סיסמת המנהל. זה חייב להתאים לשדה _סיסמת מנהל_.

### חוזק סיסמה

סעיף זה מציג את החוזק היחסי של _סיסמת המנהל_.

#### מחולל סיסמאות

אתה יכול להשתמש בחלק זה כדי ליצור ולהשתמש בסיסמה אקראית עבור החשבון. בחר _Generate_ כדי להציג סיסמה אקראית. בחר _העתק_ כדי להעתיק את הסיסמה שנוצרה אל _סיסמת מנהל_.
