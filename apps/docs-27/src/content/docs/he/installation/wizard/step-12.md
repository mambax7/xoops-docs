#### ערכת נושא ברירת מחדל

בחר נושא ראשוני עבור האתר שלך מהרשימה. צילום מסך של כל נושא מוצג כאשר הוא נבחר.

XOOPS 2.7.0 נשלח עם מספר ערכות נושא חזיתיות מהקופסה:

* **ברירת מחדל** - ערכת נושא בסיסית מינימלית, שימושית כנקודת התחלה להתאמה אישית.
* **xbootstrap5** - ערכת נושא Bootstrap 5, מומלץ לרוב האתרים החדשים.
* **xswatch5** - גרסאות Bootswatch 5; קבוצה של ערכות משנה מעוצבות על גבי Bootstrap 5.
* **xtailwind** — Tailwind CSS ערכת נושא + DaisyUI עם 35 פלטות ואינטראקטיביות Alpine.js
* **xtailwind2** - נושא אחים בימוי אמנות עם פלטות אצרות וסגנון מודול מעודן

ניתן לשנות את הנושא בכל עת מאזור הניהול, וניתן להעלות ערכות נושא נוספות לספריית `themes/`.
