אשף ההתקנה XOOPS מנחה אותך בתהליך בן 15 שלבים שמגדיר את מסד הנתונים שלך, יוצר את חשבון הניהול ומכין את האתר שלך לשימוש ראשון.

## לפני שתתחיל

- [העלית XOOPS לשרת שלך](/xoops-docs/2.7/installation/ftp-upload/) או הגדרת סביבה מקומית
- אתה [אימתת את הדרישות](/xoops-docs/2.7/installation/requirements/)
- יש לך את האישורים של מסד הנתונים שלך מוכנים

## צעדי אשף

| שלב | מסך | מה קורה |
|------|--------|--------------|
| 1 | [בחירת שפה](./step-01/) | בחר שפת התקנה |
| 2 | [ברוך הבא](./step-02/) | הסכם רישיון |
| 3 | [בדיקת תצורה](./step-03/) | PHP/server בדיקת סביבה |
| 4 | [הגדרת נתיב](./step-04/) | הגדר נתיב שורש ו-URL |
| 5 | [חיבור למסד נתונים](./step-05/) | הזן מארח מסד נתונים, משתמש, סיסמה |
| 6 | [תצורת מסד נתונים](./step-06/) | הגדר שם מסד נתונים וקידומת טבלה |
| 7 | [שמור תצורה](./step-07/) | כתוב mainfile.php |
| 8 | [יצירת שולחן](./step-08/) | צור את סכימת מסד הנתונים |
| 9 | [הגדרות ראשוניות](./step-09/) | שם האתר, דוא"ל מנהל |
| 10 | [הכנסת נתונים](./step-10/) | אכלס נתוני ברירת מחדל |
| 11 | [תצורת אתר](./step-11/) | URL, אזור זמן, שפה |
| 12 | [בחר ערכת נושא](./step-12/) | בחר ערכת נושא ברירת מחדל |
| 13 | [התקנת מודול](./step-13/) | התקן מודולים מצורפים |
| 14 | [ברוך הבא](./step-14/) | הודעת ההתקנה הושלמה |
| 15 | [ניקוי](./step-15/) | הסר את תיקיית ההתקנה |

:::זהירות[אבטחה]
לאחר השלמת האשף, **מחק או שנה את שם התיקיה `install/`** — שלב 15 מנחה אותך בתהליך. השארת אותו נגיש מהווה סיכון אבטחה.
:::
