Esta página recopila la información usada para establecer el estado inicial de tu nuevo sitio XOOPS. Esto incluye cosas como títulos y valores de etiquetas meta HTML.

Estos son solo los valores iniciales para tu sitio. La configuración puede ser cambiada en cualquier momento en las preferencias de administración de tu nuevo sitio XOOPS.

Quizás quieras usar estas configuraciones iniciales para limitar lo que será visto públicamente hasta que tu sitio esté completamente configurado.

Después de introducir la información solicitada y corregir cualquier problema, selecciona el botón "Continuar" para proceder.

![Configuración del Sitio del Instalador de XOOPS](/xoops-docs/2.7/img/installation/installer-11.png)

## Datos Recopilados en Este Paso

### Configuración General

#### Nombre del sitio

El nombre, o título, para tu sitio.

#### Eslogan para tu sitio

Un eslogan, o subtítulo, para tu sitio.

#### Modo de depuración

Esto te permite entrar a tu nuevo sitio con información de depuración mostrada. Mientras que un sitio web en ejecución debería tener esto apagado, puede ser muy útil si hay algún problema con la instalación. _Habilitar depuración (modo inline)_ es el valor más comúnmente usado.

#### ¿Apagar tu sitio?

Selecciona _sí_ para apagar tu sitio. En este estado, inmediatamente después de la instalación solo la cuenta de admin recién creada podrá acceder al sitio. Esto te permitirá configurar tu sitio a tu satisfacción antes de permitir acceso público.

### Etiquetas Meta y Pie de Página

Las etiquetas meta HTML contienen datos describiendo un sitio. Algunos módulos pueden establecer etiquetas meta específicas de página. Los valores de etiqueta introducidos aquí serán usados si no son reemplazados por valores específicos de página.

#### Palabras clave meta

Una lista de palabras clave relacionadas con el contenido del sitio. Estas pueden jugar un papel en SEO, pero usa cuidadosamente. "Spamming" con palabras clave puede tener un efecto negativo en la colocación del motor de búsqueda.

#### Descripción meta

Una breve descripción de tu sitio.

#### Autor meta

El autor predeterminado del sitio en forma de nombre, dirección de correo, nombre de empresa o URL.
