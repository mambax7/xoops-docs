---
title: "Limpieza"
---

Por propósitos de seguridad, deberías remover el directorio _install_ de tu sitio. El instalador habrá renombrado el directorio a _installremove_ seguido de una cadena aleatoria larga.

Puedes usar un cliente FTP, como FileZilla para eliminar el directorio.
