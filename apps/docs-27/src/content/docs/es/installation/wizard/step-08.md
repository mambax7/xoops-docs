---
title: "Creación de Tabla"
---

Esta página muestra los resultados de crear las tablas de XOOPS en la base de datos.

Después de revisar y corregir cualquier problema, selecciona el botón "Continuar" para proceder.

## En Caso de Éxito

La sección _Creación de tablas de base de datos_ muestra las tablas de base de datos que fueron creadas.

![Creación de Tabla del Instalador de XOOPS](/xoops-docs/2.7/img/installation/installer-08.png)

## Errores

Cualquier error indica problemas con los permisos relacionados con la cuenta de usuario de MySQL y/o la base de datos de MySQL introducida en los pasos anteriores. Puedes volver a esas secciones usando las opciones de Conexión de Base de Datos y Configuración de Base de Datos en el menú.
