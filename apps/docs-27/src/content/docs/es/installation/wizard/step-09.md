Esta página recopila la información usada para crear la cuenta principal de Administrador (AKA Webmaster).

Después de introducir la información solicitada y corregir cualquier problema, selecciona el botón "Continuar" para proceder.

![Configuración Inicial del Instalador de XOOPS](/xoops-docs/2.7/img/installation/installer-09.png)

## Datos Recopilados en Este Paso

### Cuenta del administrador

#### Inicio de sesión del admin

Este será el nombre de usuario usado por el administrador para iniciar sesión en el sistema XOOPS.

#### Email del admin

Esta es la dirección de correo usada por el sistema XOOPS para comunicarse con el administrador. (Nota, esto es requerido para recuperación/restablecimiento de contraseña.)

#### Contraseña del admin

Esta es la contraseña a usar para la cuenta del administrador.

#### Confirmar contraseña

Usa este campo para confirmar la contraseña del administrador. Debe coincidir con el campo _Contraseña del admin_.

### Fortaleza de contraseña

Esta sección muestra la fortaleza relativa de la _Contraseña del admin_.

#### Generador de contraseña

Puedes usar esta sección para generar y usar una contraseña aleatoria para la cuenta. Selecciona _Generar_ para mostrar una contraseña aleatoria. Selecciona _Copiar_ para copiar la contraseña generada a _Contraseña del admin_.
