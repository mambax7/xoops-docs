Esta página recopila la información sobre la cuenta de usuario de base de datos que XOOPS usará.

Después de introducir la información solicitada y corregir cualquier problema, selecciona el botón "Continuar" para proceder.

![Conexión de Base de Datos del Instalador de XOOPS](/xoops-docs/2.7/img/installation/installer-05.png)

## Datos Recopilados en Este Paso

### Conexión del servidor

#### Base de Datos

Actualmente, _mysql_ es la única opción disponible.

#### Nombre de host del servidor

Nombre de host del servidor de base de datos. Si no estás seguro, _localhost_ funciona en la mayoría de los casos.

#### Nombre de usuario

La cuenta de usuario de base de datos que XOOPS debe usar para conectarse al servidor de base de datos.

#### Contraseña

Contraseña de la cuenta de usuario de base de datos introducida anteriormente.

#### Usar conexión persistente

Por defecto es 'No'. Déjalo en blanco si no estás seguro.
