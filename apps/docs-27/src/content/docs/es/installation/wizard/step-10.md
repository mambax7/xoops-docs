Esta página muestra los resultados de crear las tablas de XOOPS en la base de datos.

Después de revisar y corregir cualquier problema, selecciona el botón "Continuar" para proceder.

## En Caso de Éxito

La sección _Guardando tu configuración en la base de datos_ muestra el número de entradas insertadas en la base de datos por tabla.

Debajo de esto está el resultado de la generación de la clave de licencia.
