---
title: "Configuración de Ruta"
---

Esta página recopila la información sobre el sistema de archivos y dirección web que XOOPS necesita para operar. El instalador intentará adivinar esta información en base a la URL que utilizas para acceder a él, y las ubicaciones de los scripts PHP en ejecución.

Después de revisar y corregir cualquier problema, selecciona el botón "Continuar" para proceder.

![Configuración de Ruta del Instalador de XOOPS](/xoops-docs/2.7/img/installation/installer-04-02.png)

## Datos Recop ilados en Este Paso

### Rutas físicas de XOOPS

#### Ruta física de la raíz de documentos de XOOPS

Ruta física del directorio de documentos (servido) de XOOPS SIN barra inclinada al final

#### Directorio de archivos de datos de XOOPS

Ruta física del directorio de archivos de datos (escribible) de XOOPS SIN barra inclinada al final. Ubica la carpeta fuera de la raíz del documento de XOOPS para hacerla segura.

#### Directorio de biblioteca de XOOPS

Ruta física del directorio de biblioteca de XOOPS SIN barra inclinada al final. Ubica la carpeta fuera de la raíz del documento de XOOPS para hacerla segura.

### Ubicaciones web

#### Ubicación del sitio web (URL)

URL principal que se usará para acceder a tu instalación de XOOPS. XOOPS usará esto para construir todas las URLs dentro de XOOPS. Si quieres un sitio _https_, asegúrate de especificarlo. Si quieres/no quieres un _www._ inicial en tu dominio del sitio, asegúrate de especificarlo como quieres.

Esto por defecto será la URL usada para lanzar el instalador.

#### Dominio de Cookie para el Sitio Web

Dominio para establecer cookies. Puede estar en blanco, el host completo de la URL (www.example.com), o el dominio registrado sin subdominios (example.com) para compartir entre subdominios (www.example.com y blog.example.com.)

## Errores

Si alguna de las rutas introducidas no se puede encontrar, se mostrará un error. Corrige cualquier problema antes de continuar.

![Error de Configuración de Ruta del Instalador de XOOPS](/xoops-docs/2.7/img/installation/installer-04-01.png)
