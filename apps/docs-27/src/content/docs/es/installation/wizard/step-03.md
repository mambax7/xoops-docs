---
title: "Comprobación de Configuración"
---

_Comprobando la configuración de tu servidor_ es una revisión del entorno en el que se ejecuta el instalador.

Si se reportan problemas en la sección _Requisitos_, estos necesitan ser corregidos, de lo contrario XOOPS _no_ funcionará.

La sección _Extensiones Recomendadas_ indica extensiones PHP adicionales que XOOPS, o módulos de XOOPS, pueden usar para proporcionar funciones adicionales. Puedes proceder con la instalación, incluso si las extensiones no están disponibles. Pero, si puedes habilitarlas, eso es recomendado.

Después de revisar, selecciona el botón "Continuar" para proceder.
