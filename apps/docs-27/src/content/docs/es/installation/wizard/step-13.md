Esta página recopila tu opción para los módulos a instalar en tu nuevo sitio. Los módulos pueden ser añadidos o removidos en cualquier momento, si lo deseas.

Después de introducir la información solicitada y corregir cualquier problema, selecciona el botón "Continuar" para proceder.

![Instalación de Módulos del Instalador de XOOPS](/xoops-docs/2.7/img/installation/installer-13.png)

## Datos Recopilados en Este Paso

### Módulos Disponibles

Selecciona "Sí" para instalar un módulo. Puedes seleccionar ninguno, cualquiera o todos los módulos.

Estos son los módulos incluidos con una instalación estándar de XOOPS 2.7.0.

* Mensajería Privada - _mensajería privada mejorada entre usuarios_
* Perfil de Usuario - _perfiles de usuario mejorados/personalizados_
* Protector - _seguridad mejorada del sitio, controles de spam y mitigación de ataques_
* DebugBar - _una barra de herramientas de depuración para desarrollo y preparación basada en Symfony VarDumper; útil para desarrolladores de módulos y para diagnosticar problemas de producción, pero normalmente no necesaria en un sitio en vivo público_

El módulo _Sistema_ siempre es instalado y no está listado aquí como una opción.

#### Nota

Puedes añadir módulos al directorio de módulos al subir los archivos de XOOPS antes de comenzar la instalación, y también estarán disponibles en este paso.

#### Recomendación Fuerte

**Todos los sitios de XOOPS** deberían instalar el módulo _Protector_.
