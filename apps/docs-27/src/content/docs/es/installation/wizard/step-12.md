Esta página recopila tu opción para el tema inicial para tu nuevo sitio. Los temas controlan el carácter visual de tu sitio. Los temas pueden ser cambiados en cualquier momento, si lo deseas.

Después de introducir la información solicitada y corregir cualquier problema, selecciona el botón "Continuar" para proceder.

## Datos Recopilados en Este Paso

### Selecciona el tema predeterminado

#### Tema predeterminado

Selecciona un tema inicial para tu sitio de la lista. Una captura de pantalla de cada tema se muestra mientras es seleccionado.

XOOPS 2.7.0 viene con varios temas de frontend fuera de la caja:

* **default** — un tema de línea base minimalista, útil como punto de partida para personalización.
* **xbootstrap5** — Tema Bootstrap 5, recomendado para la mayoría de sitios nuevos.
* **xswatch5** — Variantes de Bootswatch 5; un conjunto de subtemas estilizados sobre Bootstrap 5.
* **xtailwind** — Tema basado en Tailwind CSS + DaisyUI con 35 paletas e interactividad Alpine.js
* **xtailwind2** — tema hermano dirigido por arte con paletas curadas y estilo de módulo refinado

Puedes cambiar el tema en cualquier momento desde el área de administración, y temas adicionales pueden ser subidos al directorio `themes/`.
