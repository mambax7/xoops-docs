| Software | Descripción |
| --- | --- |
| 7Zip www.7zip.org | Programa de código abierto para comprimir/descomprimir archivos. |
| WinSCP www.WinSCP.net | WinSCP es un cliente SFTP de código abierto y un cliente FTP para Windows. Su función principal es la transferencia segura de archivos entre una computadora local y una computadora remota. |
| WAMP Server www.wampserver.com | WampServer es un entorno de desarrollo web de Windows. Le permite crear aplicaciones web con Apache, PHP y la base de datos MySQL. También viene con PHPMyAdmin para administrar fácilmente sus bases de datos. WampServer se instala automáticamente (instalador) y su uso es muy intuitivo. Podrá ajustar su servidor sin ni siquiera tocar los archivos de configuración. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP es una distribución Apache fácil de instalar que contiene MySQL, PHP y Perl. XAMPP es realmente muy fácil de instalar y usar - simplemente descargue, extraiga e inicie. |
| KeyPass www.keepass.info | KeePass es un gestor de contraseñas de código abierto gratuito que le ayuda a administrar sus contraseñas de manera segura. Puede poner todas sus contraseñas en una base de datos, que se bloquea con una clave maestra o un archivo de clave. Entonces solo tiene que recordar una sola contraseña maestra o seleccionar el archivo de clave para desbloquear toda la base de datos. Las bases de datos se cifran utilizando los mejores algoritmos de cifrado más seguros conocidos actualmente (AES y Twofish). |
| Package Web Tools [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === Aplicaciones USB ==== - 7zip, Filezilla, Kompozer, DataBase Browser, Opera, PicPick, Winmerge, Gimp, Notepad++, OikoCSSeditor, SubCommander, ToDoList, Toucan, Xenu, TeamViewer, Pidgin, FastImageOptimizer, VirtualWin y MiniServer (con el último Apache - MySQL - PHP, y Adminer para administrar la base de datos) |
| Package Web Browsers [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === Aplicaciones USB ==== - Firefox + extensiones webdevelper y XooFoo (versión estable y desarrollo), IETester 0.4.5, Opera (versión estable y desarrollo), OperaMobile Emulator, Safari, Google Chrome + extensión webdevelper (versión estable y desarrollo), Maxthon (versión estable y desarrollo), SeaMonkey, navegador Avant, Flock y Elinks |
| Uniform Server [http://uniformserver.com](http://uniformserver.com) | "¿Por qué usar Uniform Server en lugar de XAMPP o WAMP o EASYPHP? 1. Seguridad 2. Ligero (xampp 51MB, XAMPlite 28MB. Uniform Server 9.95MB). 3. Más fácil de usar. 4. XAMPP (por ejemplo) tiene muchas características innecesarias, especialmente para la comunidad de Xoops. A pesar del pequeño tamaño de Uniform Server, tiene todo lo que necesitamos./ /Instalar Uniform Server le dará, en últimas versiones: - Base de datos MySQL. - phpMyAdmin para administrar la base de datos MySQL. - Alojamiento Apache. - PHP (Tiene más de 100 correcciones de errores de las que obtiene al descargar XAMPP). - eAccelerator para hacer que el sitio web sea más rápido. |
| Uniform Server por XooFoo [http://goo.gl/LGx3V](http://goo.gl/LGx3V) | Uniform Server optimizado para XOOPS |
| Mini Server (solo 5MB) por XooFoo [http://goo.gl/Ke0LW](http://goo.gl/Ke0LW) | Un servidor de producción portátil básico con el poder añadido de PHP permitiendo páginas web dinámicas además de páginas htm y html estáticas. Diseñado para permitir que varios servidores ejecuten en la misma PC o unidades USB, estos requerirán una unidad virtual separada creada automáticamente. El soporte de MySQL está habilitado mejorando aún más la capacidad de este servidor. |
