## Acerca de XOOPS CMS

XOOPS es un Sistema de Gestión de Contenido (CMS) y programa de portal web que permite a los administradores crear fácilmente sitios web dinámicos con excelente contenido y muchas características excepcionales. Es una herramienta ideal para desarrollar sitios web de comunidad dinámicos pequeños a grandes, portales intra-empresa, portales corporativos, blogs y mucho más. Se puede instalar en cualquier servidor con un servidor web capaz de ejecutar PHP (p. ej., Apache o Nginx) y una base de datos compatible con MySQL.

**Sitio web del proyecto principal:** [https://xoops.org](https://xoops.org)

XOOPS se publica bajo los términos de la Licencia Pública General de GNU (GPL) y es gratuita para usar y modificar. Puede redistribuirla siempre que cumpla con los términos de distribución de la GPL.

### Por qué XOOPS significa

XOOPS es un acrónimo de **Extensible Object Oriented Portal System** (Sistema de Portal Orientado a Objetos Extensible). Aunque comenzó como un sistema de portal, ahora es un Sistema de Gestión de Contenido completamente completo que puede servir desde pequeños sitios personales hasta instalaciones empresariales grandes.

### Pronunciación

La pronunciación estándar de XOOPS sigue las reglas del inglés y se pronuncia `[zoo'ps]`.

### Características clave

**Impulsado por Base de Datos**

XOOPS utiliza una base de datos relacional (MySQL, 5.7.8 o posterior) para almacenar los datos requeridos para ejecutar un sistema de gestión de contenido basado en web.

**Completamente Modularizado**

Los módulos se pueden instalar, desinstalar, activar o desactivar con un clic a través del sistema de administración de módulos XOOPS.

**Personalización**

Los usuarios registrados pueden editar sus perfiles, seleccionar temas del sitio, cargar avatares personalizados y más.

**Gestión de Usuarios**

XOOPS ofrece búsqueda de usuarios por varios criterios, correo electrónico y un sistema de mensajería privada basado en plantillas.

**Soporte Mundial**

XOOPS es creado y mantenido por un equipo de voluntarios de todo el mundo. La comunidad XOOPS tiene sitios de soporte en múltiples idiomas.

**Soporte de Idioma Multibyte**

XOOPS es totalmente compatible con idiomas multibyte, incluyendo japonés, chino simplificado y tradicional, coreano y árabe (con soporte RTL agregado en 2.7.0).

**Permisos de Grupo Versátiles**

Un sistema de permisos poderoso y fácil de usar permite a los administradores establecer permisos por grupo.

**Interfaz Temática Personalizable**

El sistema de temas permite tanto a administradores como a usuarios cambiar la apariencia de todo el sitio. La distribución 2.7.0 se envía con temas modernos basados en Bootstrap-5 y Tailwind.
