---

Dieser Abschnitt führt Sie durch die Aktualisierung einer bestehenden XOOPS-Website. Der Upgrade-Assistent behält Ihre Daten, Module und Themes bei, während die Kerndateien aktualisiert werden.

## Upgrade-Pfad

| Schritt | Seite | Was zu tun ist |
|------|------|------------|
| 1 | [Vorbereitung](./step-01/) | Sicherung, Changelog überprüfen, neue Version herunterladen |
| 2 | [Preflight-Check](./preflight/) | Kompatibilität vor dem Upgrade überprüfen |
| 3 | [Upgrade ausführen](./step-02/) | Upgrade-Assistent ausführen |
| 4 | [Troubleshooting](./step-03/) | Häufige Upgrade-Probleme beheben |
| 5 | [Nach dem Upgrade](./step-04/) | Überprüfen, Module aktualisieren, Cache leeren |

:::tip
Machen Sie immer eine **vollständige Datenbank-Sicherung und Datei-Sicherung**, bevor Sie ein Upgrade durchführen. Das Upgrade ist ohne Sicherung nicht umkehrbar.
:::
