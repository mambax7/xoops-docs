Diese Seite erfasst Informationen zum Datenbank-Benutzerkonto, das XOOPS verwenden wird.

Geben Sie die angeforderten Informationen ein und korrigieren Sie nach Bedarf Probleme, und wählen Sie dann die Schaltfläche "Weiter", um fortzufahren.

![XOOPS-Installationsprogramm-Datenbankverbindung](/xoops-docs/2.7/img/installation/installer-05.png)

## In diesem Schritt erfasste Daten

### Server-Verbindung

#### Datenbank

Derzeit ist _mysql_ die einzige verfügbare Option.

#### Server-Hostname

Hostname des Datenbankservers. Wenn Sie unsicher sind, funktioniert _localhost_ in den meisten Fällen.

#### Benutzername

Das Datenbank-Benutzerkonto, das XOOPS zur Verbindung mit dem Datenbankserver verwenden soll.

#### Passwort

Passwort des oben eingegebenen Datenbank-Benutzerkontos.

#### Persistente Verbindung verwenden

Die Standardeinstellung ist "Nein". Lassen Sie dies leer, wenn Sie unsicher sind.
