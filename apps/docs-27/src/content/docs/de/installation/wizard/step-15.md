---
title: "Bereinigung"
---

Aus Sicherheitsgründen sollten Sie das Verzeichnis _install_ von Ihrer Website entfernen. Das Installationsprogramm wird das Verzeichnis in _installremove_ gefolgt von einer langen Zufallszeichenfolge umbenannt haben.

Sie können einen FTP-Client wie FileZilla verwenden, um das Verzeichnis zu löschen.
