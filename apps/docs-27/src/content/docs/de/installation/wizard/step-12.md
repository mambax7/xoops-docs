Diese Seite erfasst Ihre Wahl für das anfängliche Design für Ihre neue Website. Designs steuern das visuelle Aussehen Ihrer Website. Designs können jederzeit bei Bedarf geändert werden.

Wählen Sie nach Eingabe der angeforderten Informationen und Korrektur etwaiger Probleme die Schaltfläche "Weiter", um fortzufahren.

![XOOPS Installer Design auswählen](/xoops-docs/2.7/img/installation/installer-12.png)

## In diesem Schritt erfasste Daten

### Wählen Sie das Standarddesign

#### Standarddesign

Wählen Sie ein anfängliches Design für Ihre Website aus der Liste. Ein Screenshot jedes Designs wird angezeigt, wenn es ausgewählt wird.

XOOPS 2.7.0 wird mit mehreren Frontend-Designs ab Werk geliefert:

* **default** — ein minimales Basis-Design, das als Ausgangspunkt für Anpassungen nützlich ist.
* **xbootstrap5** — Bootstrap 5-Design, empfohlen für die meisten neuen Websites.
* **xswatch5** — Bootswatch 5-Varianten; eine Reihe von gestylten Unter-Designs auf Basis von Bootstrap 5.
* **xtailwind** — Tailwind CSS-basiertes Design + DaisyUI mit 35 Paletten und Alpine.js-Interaktivität
* **xtailwind2** — kunstdirektiertes Schwester-Design mit kuratierten Paletten und verfeinertem Modul-Styling

Sie können das Design jederzeit aus dem Verwaltungsbereich ändern, und zusätzliche Designs können in das Verzeichnis `themes/` hochgeladen werden.
