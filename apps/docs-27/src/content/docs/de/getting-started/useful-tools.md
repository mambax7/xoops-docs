| Software | Beschreibung |
| --- | --- |
| 7Zip www.7zip.org | Open-Source-Programm zum Zippen/Entzippen von Dateien. |
| WinSCP www.WinSCP.net | WinSCP ist ein Open-Source-SFTP-Client und FTP-Client für Windows. Seine Hauptfunktion ist die sichere Dateiübertragung zwischen einem lokalen und einem Remote-Computer. |
| WAMP Server www.wampserver.com | WampServer ist eine Windows-Webentwicklungsumgebung. Es ermöglicht Ihnen, Webanwendungen mit Apache, PHP und der MySQL-Datenbank zu erstellen. Es wird auch mit PHPMyAdmin geliefert, um Ihre Datenbanken einfach zu verwalten. WampServer wird automatisch installiert (Installer), und die Verwendung ist sehr intuitiv. Sie können Ihren Server anpassen, ohne die Einstellungsdateien zu berühren. |
| XAMPP  [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP ist eine einfach zu installierende Apache-Distribution, die MySQL, PHP und Perl enthält. XAMPP ist wirklich sehr einfach zu installieren und zu verwenden - einfach herunterladen, extrahieren und starten. |
| KeyPass www.keepass.info | KeePass ist ein kostenloser Open-Source-Passwort-Manager, der Ihnen dabei hilft, Ihre Passwörter auf sichere Weise zu verwalten. Sie können alle Ihre Passwörter in eine Datenbank eingeben, die mit einem Master-Schlüssel oder einer Schlüsseldatei gesperrt ist. Sie müssen sich also nur ein Passwort merken oder die Schlüsseldatei auswählen, um die gesamte Datenbank zu entsperren. Die Datenbanken werden mit den besten und sichersten derzeit bekannten Verschlüsselungsalgorithmen verschlüsselt (AES und Twofish). |
| Package Web Tools [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === USB Apps ==== - 7zip, Filezilla, Kompozer, DataBase Browser, Opera, PicPick, Winmerge, Gimp, Notepad++, OikoCSSeditor, SubCommander, ToDoList, Toucan, Xenu, TeamViewer, Pidgin, FastImageOptimizer, VirtualWin und MiniServer (mit letztem Apache - MySQL - PHP und Adminer zur Verwaltung der Datenbank) |
| Package Web Browsers [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === USB Apps ==== - Firefox + webdevelopper und XooFoo Erweiterungen (stabile und Entwicklungsversion), IETester 0.4.5, Opera (stabile und Entwicklungsversion), OperaMobile Emulator, Safari, Google Chrome + webdevelopper Erweiterung (stabile und Entwicklungsversion), Maxthon (stabile und Entwicklungsversion), SeaMonkey, Avant Browser, Flock und Elinks |
| Uniform Server [http://uniformserver.com](http://uniformserver.com) | „Warum Uniform Server statt XAMPP oder WAMP oder EASYPHP verwenden? 1. Sicherheit 2. Leichtgewicht (xampp 51 MB, XAMPlite 28 MB. Uniform Server 9,95 MB). 3. Einfacher zu verwenden. 4. XAMPP (zum Beispiel) hat viele unnötige Funktionen, besonders für die Xoops-Community. Trotz der geringen Größe von Uniform Server hat es alles, was wir brauchen./ /Die Installation von Uniform Server gibt Ihnen in den letzten Versionen: - MySQL-Datenbank. - phpMyAdmin zur Verwaltung der MySQL-Datenbank. - Apache-Hosting. - PHP (Hat über 100 Fehlerkorrektionen mehr als das, was Sie beim Download von XAMPP erhalten). - eAccelerator, um die Website schneller zu machen. |
| Uniform Server by XooFoo [http://goo.gl/LGx3V](http://goo.gl/LGx3V) | Uniform Server optimiert für XOOPS |
| Mini Server (nur 5 MB) by XooFoo [http://goo.gl/Ke0LW](http://goo.gl/Ke0LW) | Ein grundlegender tragbarer Produktionsserver mit der zusätzlichen Leistung von PHP, die dynamische Webseiten zusätzlich zu statischen htm- und html-Seiten ermöglicht. Entworfen, um mehrere Server auf demselben PC oder USB-Discs zu ermöglichen, benötigen diese ein separates virtuelles Laufwerk, das automatisch erstellt wird. MySQL-Unterstützung ist aktiviert und verbessert damit die Funktionen dieses Servers weiter. |
