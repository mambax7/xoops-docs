---
title: "Vérification de la configuration"
---

La _Vérification de la configuration de votre serveur_ est un examen de l'environnement dans lequel le programme d'installation s'exécute.

Si des problèmes sont signalés dans la section des _Exigences_, ils doivent être corrigés, sinon XOOPS _ne_ fonctionnera _pas_.

La section _Extensions recommandées_ indique des extensions PHP supplémentaires que XOOPS, ou les modules XOOPS, peuvent utiliser pour fournir des fonctions supplémentaires. Vous pouvez continuer l'installation même si les extensions ne sont pas disponibles. Mais, si vous pouvez activer ces extensions, c'est recommandé.

Après examen, sélectionnez le bouton "Continuer" pour continuer.
