Cette page collecte votre choix pour les modules à installer dans votre nouveau site. Les modules peuvent être ajoutés ou supprimés à tout moment, si souhaité.

Après avoir entré les informations demandées et corrigé tout problème, sélectionnez le bouton "Continuer" pour continuer.

![Installation du module du programme d'installation XOOPS](/xoops-docs/2.7/img/installation/installer-13.png)

## Données collectées dans cette étape

### Modules disponibles

Sélectionnez "Oui" pour installer un module. Vous pouvez ne sélectionner aucun, certains ou tous les modules.

Ce sont les modules inclus dans une installation XOOPS 2.7.0 standard.

* Messagerie privée - _messagerie privée améliorée entre les utilisateurs_
* Profil utilisateur - _profils utilisateur améliorés/personnalisés_
* Protector - _sécurité du site améliorée, contrôles du spam et atténuation des attaques_
* DebugBar - _une barre d'outils de débogage de développement et d'intermédiaire basée sur Symfony VarDumper; utile pour les développeurs de modules et pour diagnostiquer les problèmes de production, mais généralement pas nécessaire sur un site public en direct_

Le module _System_ est toujours installé et n'est pas répertorié ici comme une option.

#### Note

Vous pouvez ajouter des modules au répertoire des modules lors du téléchargement des fichiers XOOPS avant de démarrer l'installation, et ils seront également disponibles lors de cette étape.

#### Recommandation forte

**Tous les sites XOOPS** doivent installer le module _Protector_.
