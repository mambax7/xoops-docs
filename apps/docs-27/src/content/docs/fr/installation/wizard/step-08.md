---
title: "Création de table"
---

Cette page affiche les résultats de la création des tables XOOPS dans la base de données.

Après examen et correction de tout problème, sélectionnez le bouton "Continuer" pour continuer.

## En cas de succès

La section _Création des tables de la base de données_ affiche les tables de la base de données qui ont été créées.

![Création de table du programme d'installation XOOPS](/xoops-docs/2.7/img/installation/installer-08.png)

## Erreurs

Toute erreur indique des problèmes avec les permissions relatifs au compte d'utilisateur de la base de données MySQL et/ou à la base de données MySQL saisie aux étapes précédentes. Vous pouvez revenir à ces sections en utilisant les options Connexion à la base de données et Configuration de la base de données dans le menu.
