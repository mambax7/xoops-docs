title: "Bienvenue"

Le _Bienvenue sur votre site XOOPS_ couvre brièvement quelques ressources XOOPS disponibles et offre quelques suggestions pour gérer votre site.

Après examen, sélectionnez le bouton "Continuer" pour accéder à votre nouveau site.
