Cette page collecte votre choix du thème initial de votre nouveau site. Les thèmes contrôlent le caractère visuel de votre site. Les thèmes peuvent être modifiés à tout moment, si souhaité.

Après avoir entré les informations demandées et corrigé tout problème, sélectionnez le bouton "Continuer" pour continuer.

![Sélection du thème du programme d'installation XOOPS](/xoops-docs/2.7/img/installation/installer-12.png)

## Données collectées dans cette étape

### Sélectionner le thème par défaut

#### Thème par défaut

Sélectionnez un thème initial pour votre site dans la liste. Une capture d'écran de chaque thème est affichée lorsqu'il est sélectionné.

XOOPS 2.7.0 est fourni avec plusieurs thèmes frontaux prêts à l'emploi :

* **default** — un thème de base minimal, utile comme point de départ pour la personnalisation.
* **xbootstrap5** — Thème Bootstrap 5, recommandé pour la plupart des nouveaux sites.
* **xswatch5** — Variantes Bootswatch 5; un ensemble de sous-thèmes stylisés au-dessus de Bootstrap 5.
* **xtailwind** — Thème basé sur Tailwind CSS + DaisyUI avec 35 palettes et interactivité Alpine.js
* **xtailwind2** — Thème frère dirigé par l'art avec des palettes curées et un style de module affiné

Vous pouvez modifier le thème à tout moment depuis la zone d'administration, et des thèmes supplémentaires peuvent être téléchargés dans le répertoire `themes/`.
