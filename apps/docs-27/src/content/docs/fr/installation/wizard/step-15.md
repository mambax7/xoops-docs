---
title: "Nettoyage"
---

Pour des raisons de sécurité, vous devez supprimer le répertoire _install_ de votre site. Le programme d'installation aura renommé le répertoire en _installremove_ suivi d'une longue chaîne aléatoire.

Vous pouvez utiliser un client FTP, tel que FileZilla pour supprimer le répertoire.
