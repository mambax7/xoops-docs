title: "Paramètres initiaux"
---

Cette page collecte les informations utilisées pour créer le compte administrateur principal (alias Webmaster).

Après avoir entré les informations demandées et corrigé tout problème, sélectionnez le bouton "Continuer" pour continuer.

![Paramètres initiaux du programme d'installation XOOPS](/xoops-docs/2.7/img/installation/installer-09.png)

## Données collectées dans cette étape

### Compte administrateur

#### Connexion administrateur

Ce sera le nom d'utilisateur utilisé par l'administrateur pour se connecter au système XOOPS.

#### Email administrateur

C'est l'adresse email utilisée par le système XOOPS pour communiquer avec l'administrateur. (Remarque : ceci est requis pour la récupération/réinitialisation du mot de passe.)

#### Mot de passe administrateur

C'est le mot de passe à utiliser pour le compte administrateur.

#### Confirmer le mot de passe

Utilisez ce champ pour confirmer le mot de passe administrateur. Il doit correspondre au champ _Mot de passe administrateur_.

### Force du mot de passe

Cette section affiche la force relative du _Mot de passe administrateur_.

#### Générateur de mot de passe

Vous pouvez utiliser cette section pour générer et utiliser un mot de passe aléatoire pour le compte. Sélectionnez _Générer_ pour afficher un mot de passe aléatoire. Sélectionnez _Copier_ pour copier le mot de passe généré dans _Mot de passe administrateur_.
