title: "Connexion à la base de données"
---

Cette page collecte les informations sur le compte d'utilisateur de la base de données que XOOPS utilisera.

Après avoir entré les informations demandées et corrigé tout problème, sélectionnez le bouton "Continuer" pour continuer.

![Connexion à la base de données du programme d'installation XOOPS](/xoops-docs/2.7/img/installation/installer-05.png)

## Données collectées dans cette étape

### Connexion au serveur

#### Base de données

Actuellement, _mysql_ est la seule option disponible.

#### Nom d'hôte du serveur

Nom d'hôte du serveur de base de données. Si vous n'êtes pas sûr, _localhost_ fonctionne dans la plupart des cas.

#### Nom d'utilisateur

Le compte d'utilisateur de la base de données que XOOPS doit utiliser pour se connecter au serveur de base de données.

#### Mot de passe

Mot de passe du compte d'utilisateur de la base de données saisi ci-dessus.

#### Utiliser une connexion persistante

Par défaut, 'Non'. Laissez ce champ vide si vous n'êtes pas sûr.
