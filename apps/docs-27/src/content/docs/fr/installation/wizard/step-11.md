---
title: "Configuration du site"
---

Cette page collecte les informations utilisées pour définir l'état initial de votre nouveau site XOOPS. Cela inclut des éléments comme les titres et les valeurs des balises meta HTML.

Ce ne sont que les valeurs initiales de votre site. Le paramètre peut être modifié à tout moment dans les préférences d'administration de votre nouveau site XOOPS.

Vous pouvez souhaiter utiliser ces paramètres initiaux pour limiter ce qui sera visible publiquement jusqu'à ce que votre site soit entièrement configuré.

Après avoir entré les informations demandées et corrigé tout problème, sélectionnez le bouton "Continuer" pour continuer.

![Configuration du site du programme d'installation XOOPS](/xoops-docs/2.7/img/installation/installer-11.png)

## Données collectées dans cette étape

### Paramètres généraux

#### Nom du site

Le nom, ou titre, de votre site.

#### Slogan de votre site

Un slogan, ou sous-titre, pour votre site.

#### Mode de débogage

Cela vous permet d'entrer votre nouveau site avec les informations de débogage affichées. Bien qu'un site Web en cours d'exécution doit avoir ceci désactivé, il peut être très utile s'il y a des problèmes avec l'installation. _Enable debug (inline mode)_ est la valeur la plus couramment utilisée.

#### Désactiver votre site ?

Sélectionnez _yes_ pour désactiver votre site. Dans cet état, immédiatement après l'installation, seul le compte administrateur nouvellement créé pourra accéder au site. Cela vous permettra de configurer votre site à votre satisfaction avant d'autoriser l'accès public.

### Méta-balises et pied de page

Les balises meta HTML contiennent les données décrivant un site. Certains modules peuvent définir les méta-balises spécifiques à la page. Les valeurs de balise saisies ici seront utilisées si elles ne sont pas remplacées par les valeurs spécifiques à la page.

#### Mots-clés méta

Une liste de mots-clés liés au contenu du site. Ceux-ci peuvent jouer un rôle dans le référencement, mais utilisez-les avec prudence. La "publicité excessive" avec les mots-clés peut avoir un effet négatif sur le classement des moteurs de recherche.

#### Description méta

Une brève description de votre site.

#### Auteur méta

L'auteur par défaut du site sous la forme d'un nom, une adresse email, un nom d'entreprise ou une URL.

#### Copyright méta

Une déclaration de droits d'auteur pour votre site.

### Paramètres des informations utilisateur

#### Autoriser les nouvelles inscriptions d'utilisateurs ?

Sélectionnez oui pour accepter les nouvelles inscriptions d'utilisateurs, ou non si l'enregistrement sera privé.
