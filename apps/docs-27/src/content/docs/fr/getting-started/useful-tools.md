| Logiciels | Description |
| --- | --- |
| 7Zip www.7zip.org | Programme open source pour compresser/décompresser les fichiers. |
| WinSCP www.WinSCP.net | WinSCP est un client SFTP et FTP open source pour Windows. Sa fonction principale est le transfert sécurisé de fichiers entre un ordinateur local et un ordinateur distant. |
| WAMP Server www.wampserver.com | WampServer est un environnement de développement Web Windows. Il vous permet de créer des applications Web avec Apache, PHP et la base de données MySQL. Il est livré avec PHPMyAdmin pour gérer facilement vos bases de données. WampServer s'installe automatiquement (programme d'installation), et son utilisation est très intuitive. Vous pourrez affiner votre serveur sans même toucher aux fichiers de paramètres. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP est une distribution Apache facile à installer contenant MySQL, PHP et Perl. XAMPP est vraiment très facile à installer et à utiliser - il suffit de télécharger, extraire et démarrer. |
| KeyPass www.keepass.info | KeePass est un gestionnaire de mots de passe gratuit et open source, qui vous aide à gérer vos mots de passe de manière sécurisée. Vous pouvez mettre tous vos mots de passe dans une base de données, qui est verrouillée avec une clé maître ou un fichier clé. Vous n'avez donc à mémoriser qu'un seul mot de passe maître ou à sélectionner le fichier clé pour déverrouiller la base de données entière. Les bases de données sont chiffrées à l'aide des meilleurs et des algorithmes de chiffrement les plus sûrs actuellement connus (AES et Twofish). |
| Package Web Tools [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === Applications USB ==== - 7zip, Filezilla, Kompozer, DataBase Browser, Opera, PicPick, Winmerge, Gimp, Notepad++, OikoCSSeditor, SubCommander, ToDoList, Toucan, Xenu, TeamViewer, Pidgin, FastImageOptimizer, VirtualWin et MiniServer (avec le dernier Apache - MySQL - PHP, et Adminer pour administrer la base de données) |
| Package Web Browsers [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === Applications USB ==== - Firefox + webdevelpper et extensions XooFoo (version stable et de développement), IETester 0.4.5, Opera (version stable et de développement), OperaMobile Emulator, Safari, Google Chrome + extension webdevelpper (version stable et de développement), Maxthon (version stable et de développement), SeaMonkey, Avant browser, Flock et Elinks |
| Uniform Server [http://uniformserver.com](http://uniformserver.com) | "Pourquoi utiliser Uniform Server au lieu de XAMPP ou WAMP ou EASYPHP? 1. Sécurité 2. Léger (xampp 51MB, XAMPlite 28MB. Uniform Server 9.95MB). 3. Plus facile à utiliser. 4. XAMPP (par exemple) a beaucoup de fonctionnalités inutiles, en particulier pour la communauté Xoops. Malgré la petite taille d'Uniform Server, il a tout ce dont nous avons besoin./ /L'installation d'Uniform Server vous donnera, dans les dernières versions : - Base de données MySQL. - phpMyAdmin pour administrer la base de données MySQL. - Hébergement Apache. - PHP (a plus de 100 corrections de bogues que ce que vous obtenez en téléchargeant XAMPP). - eAccelerator pour rendre le site Web plus rapide. |
| Uniform Server par XooFoo [http://goo.gl/LGx3V](http://goo.gl/LGx3V) | Uniform Server optimisé pour XOOPS |
| Mini Server (juste 5MB) par XooFoo [http://goo.gl/Ke0LW](http://goo.gl/Ke0LW) | Un serveur de production portable de base avec la puissance supplémentaire de PHP permettant des pages Web dynamiques en plus des pages htm et html statiques. Conçu pour permettre l'exécution de plusieurs serveurs sur le même PC ou des disques USB, ceux-ci nécessiteront un lecteur virtuel séparé créé automatiquement. Le support MySQL est activé, améliorant davantage les capacités de ce serveur. |
