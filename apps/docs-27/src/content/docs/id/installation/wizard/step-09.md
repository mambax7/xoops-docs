Halaman ini mengumpulkan informasi yang digunakan untuk membuat akun Administrator utama (AKA Webmaster).

Setelah memasukkan informasi yang diminta dan memperbaiki masalah apa pun, pilih tombol "Lanjutkan" untuk melanjutkan.

![XOOPS Installer Initial Settings](/xoops-docs/2.7/img/installation/installer-09.png)

## Data yang Dikumpulkan pada Langkah Ini

### Akun administrator

#### Login Admin

Ini akan menjadi nama pengguna yang digunakan oleh administrator untuk masuk ke sistem XOOPS.

#### Email Admin

Ini alamat email yang digunakan oleh sistem XOOPS untuk berkomunikasi dengan administrator. (Catatan, ini diperlukan untuk kata sandi recovery/reset.)

#### Kata sandi Admin

Ini adalah kata sandi yang digunakan untuk akun administrator,

#### Konfirmasi kata sandi

Gunakan bidang ini untuk mengonfirmasi kata sandi administrator. Itu harus cocok dengan bidang _Kata sandi Admin_.

### Kekuatan kata sandi

Bagian ini menampilkan kekuatan relatif dari _kata sandi Admin_.

#### Pembuat Kata Sandi

Anda dapat menggunakan bagian ini untuk membuat dan menggunakan kata sandi acak untuk akun tersebut. Pilih _Hasilkan_ untuk menampilkan kata sandi acak. Pilih _Salin_ untuk menyalin kata sandi yang dibuat ke _Kata sandi Admin_.
