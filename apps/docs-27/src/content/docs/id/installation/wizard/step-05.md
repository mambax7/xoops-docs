Halaman ini mengumpulkan informasi tentang akun pengguna database yang akan digunakan XOOPS.

Setelah memasukkan informasi yang diminta dan memperbaiki masalah apa pun, pilih tombol "Lanjutkan" untuk melanjutkan.

![XOOPS Installer Database Connection](/xoops-docs/2.7/img/installation/installer-05.png)

## Data yang Dikumpulkan pada Langkah Ini

### Koneksi server

#### Basis Data

Saat ini, _mysql_ adalah satu-satunya pilihan yang tersedia.

#### Nama host server

Nama host server database. Jika Anda tidak yakin, _localhost_ berfungsi dalam banyak kasus.

#### Nama pengguna

Akun pengguna database yang harus digunakan XOOPS untuk menyambung ke server database.

#### Kata Sandi

Kata sandi akun pengguna database dimasukkan di atas.

#### Gunakan koneksi persisten

Standarnya adalah 'Tidak'. Biarkan kosong jika Anda tidak yakin.
