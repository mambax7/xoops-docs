---
title: "Pemilihan Bahasa"
---

Distribusi utama XOOPS hadir dengan dukungan bahasa Inggris. Dukungan untuk lokal tambahan disediakan oleh [XOOPS Situs dukungan lokal](https://xoops.org/modules/xoopspartners/). Dukungan ini bisa datang dalam bentuk distribusi yang disesuaikan, atau file tambahan untuk ditambahkan ke distribusi utama.

Terjemahan XOOPS disimpan di [transifex](https://www.transifex.com/xoops/public/)

Jika Penginstal XOOPS Anda memiliki dukungan bahasa tambahan, Anda dapat memilih bahasa di sini.

Pilih tombol "Lanjutkan" untuk melanjutkan.
