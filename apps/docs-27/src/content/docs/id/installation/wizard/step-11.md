Halaman ini mengumpulkan informasi yang digunakan untuk menyetel status awal situs XOOPS baru Anda. Ini mencakup hal-hal seperti judul dan nilai tag meta HTML.

Ini hanyalah nilai awal untuk situs Anda. Pengaturan dapat diubah kapan saja di preferensi administrasi situs XOOPS baru Anda.

Anda mungkin ingin menggunakan pengaturan awal ini untuk membatasi apa yang akan dilihat secara publik hingga situs Anda sepenuhnya siap.

Setelah memasukkan informasi yang diminta dan memperbaiki masalah apa pun, pilih tombol "Lanjutkan" untuk melanjutkan.

![XOOPS Installer Site Configuration](/xoops-docs/2.7/img/installation/installer-11.png)

## Data yang Dikumpulkan pada Langkah Ini

### Pengaturan Umum

#### Nama situs

Nama, atau judul, untuk situs Anda.

#### Slogan untuk situs Anda

Slogan, atau subjudul, untuk situs Anda.

#### Modus debug

Ini memungkinkan Anda memasuki situs baru Anda dengan informasi debug yang ditampilkan. Meskipun situs web yang sedang berjalan seharusnya menonaktifkannya, ini bisa sangat berguna jika ada masalah apa pun dengan instalasi. _Aktifkan debug (mode sebaris)_ adalah nilai yang paling umum digunakan.

#### Matikan situs Anda?

Pilih _yes_ untuk mematikan situs Anda. Dalam keadaan ini, segera setelah instalasi, hanya akun admin yang baru dibuat yang dapat mengakses situs tersebut. Ini akan memungkinkan Anda mengonfigurasi situs sesuai keinginan Anda sebelum mengizinkan akses publik.

### Meta Tag dan Footer

Tag meta HTML berisi data yang menjelaskan sebuah situs. Beberapa module mungkin mengatur tag meta khusus halaman. Nilai tag yang dimasukkan di sini akan digunakan jika tidak diganti dengan nilai spesifik laman.

#### Kata Kunci Meta

Daftar kata kunci yang berhubungan dengan konten situs. Ini mungkin berperan dalam SEO, tetapi gunakan dengan hati-hati. "Spamming" dengan kata kunci dapat berdampak negatif pada penempatan mesin pencari.

#### Deskripsi Meta

Deskripsi singkat tentang situs Anda.

#### Penulis Meta

Default pembuat situs berupa nama, alamat email, nama perusahaan atau URL.

#### Meta Hak Cipta

Pernyataan hak cipta untuk situs Anda.

### Pengaturan Info Pengguna

#### Izinkan pendaftaran pengguna baru?

Pilih ya untuk menerima pendaftaran pengguna baru, atau tidak jika pendaftaran bersifat pribadi.
