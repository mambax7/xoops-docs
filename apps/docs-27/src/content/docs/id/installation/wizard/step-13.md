Halaman ini mengumpulkan pilihan Anda untuk module yang akan dipasang di situs baru Anda. module dapat ditambahkan atau dihapus kapan saja, jika diinginkan.

Setelah memasukkan informasi yang diminta dan memperbaiki masalah apa pun, pilih tombol "Lanjutkan" untuk melanjutkan.

![XOOPS Installer Modules Installation](/xoops-docs/2.7/img/installation/installer-13.png)

## Data yang Dikumpulkan pada Langkah Ini

### module yang Tersedia

Pilih "Ya" untuk menginstal module. Anda tidak boleh memilih satu pun, salah satu, atau semua module.

Ini adalah module yang disertakan dengan instalasi standar XOOPS 2.7.0.

* Pesan Pribadi - _peningkatan pesan pribadi antar pengguna_
* Profil Pengguna - profil pengguna _enhanced/custom_
* Pelindung - _peningkatan keamanan situs, kontrol spam, dan mitigasi serangan_
* DebugBar - _sebuah toolbar debug pengembangan dan pementasan berdasarkan Symfony VarDumper; berguna untuk pengembang module dan untuk mendiagnosis masalah produksi, tetapi biasanya tidak diperlukan di situs langsung yang dapat dilihat publik_

module _System_ selalu diinstal dan tidak tercantum di sini sebagai opsi.

#### Catatan

Anda dapat menambahkan module ke direktori module saat mengunggah file XOOPS sebelum memulai instalasi, dan module tersebut juga akan tersedia pada langkah ini.

#### Rekomendasi Kuat

**Semua situs XOOPS** harus menginstal module _Protector_.
