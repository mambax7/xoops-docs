---
title: "Pemeriksaan Konfigurasi"
---

_Memeriksa konfigurasi server Anda_ adalah tinjauan lingkungan tempat penginstal berjalan.

Jika ada masalah yang dilaporkan di bagian _Persyaratan_, masalah ini perlu diperbaiki, jika tidak, XOOPS akan _tidak_ berfungsi.

Bagian _Ekstensi yang disarankan_ menunjukkan ekstensi PHP tambahan yang dapat digunakan oleh module XOOPS, atau XOOPS, untuk menyediakan fungsi tambahan. Anda dapat melanjutkan penginstalan, meskipun ekstensi tidak tersedia. Namun, jika Anda dapat mengaktifkannya, itu disarankan.

Setelah meninjau, pilih tombol "Lanjutkan" untuk melanjutkan.
