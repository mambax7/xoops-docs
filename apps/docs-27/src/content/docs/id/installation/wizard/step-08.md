---
title: "Pembuatan Tabel"
---

Halaman ini menampilkan hasil pembuatan tabel XOOPS pada database.

Setelah meninjau dan memperbaiki masalah apa pun, pilih tombol "Lanjutkan" untuk melanjutkan.

## Pada Sukses

Bagian _Pembuatan tabel database_ memperlihatkan tabel database yang telah dibuat.

![XOOPS Installer Table Creation](/xoops-docs/2.7/img/installation/installer-08.png)

## Kesalahan

Kesalahan apa pun menunjukkan masalah dengan izin yang berkaitan dengan akun pengguna basis data MySQL and/or MySQL basis data yang dimasukkan pada langkah sebelumnya. Anda dapat kembali ke bagian tersebut menggunakan opsi Koneksi database dan Konfigurasi database di menu.
