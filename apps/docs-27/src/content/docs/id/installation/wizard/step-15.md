---
title: "Pembersihan"
---

Untuk tujuan keamanan, Anda harus menghapus direktori _install_ dari situs Anda. Pemasang akan mengganti nama direktori menjadi _installremove_ diikuti dengan string acak yang panjang.

Anda dapat menggunakan klien FTP, seperti FileZilla untuk menghapus direktori.
