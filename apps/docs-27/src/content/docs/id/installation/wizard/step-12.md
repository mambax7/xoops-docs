Halaman ini mengumpulkan pilihan Anda untuk theme awal situs baru Anda. theme mengontrol karakter visual situs Anda. theme dapat diubah sewaktu-waktu, jika diinginkan.

Setelah memasukkan informasi yang diminta dan memperbaiki masalah apa pun, pilih tombol "Lanjutkan" untuk melanjutkan.

## Data yang Dikumpulkan pada Langkah Ini

### Pilih theme bawaan

#### theme bawaan

Pilih theme awal untuk situs Anda dari daftar. Tangkapan layar setiap theme ditampilkan saat dipilih.

XOOPS 2.7.0 dikirimkan dengan beberapa theme front-end yang siap pakai:

* **default** — theme dasar minimal, berguna sebagai titik awal penyesuaian.
* **xbootstrap5** — theme Bootstrap 5, direkomendasikan untuk sebagian besar situs baru.
* **xswatch5** — Bootswatch 5 varian; satu set sub-theme bergaya di atas Bootstrap 5.
* **xtailwind** — theme berbasis Tailwind CSS + DaisyUI dengan 35 palet dan interaktivitas Alpine.js
* **xtailwind2** — theme saudara yang diarahkan secara seni dengan palet pilihan dan gaya module yang disempurnakan

Anda dapat mengubah theme kapan saja dari area administrasi, dan theme tambahan dapat diunggah ke direktori `themes/`.
