---
title: "Meningkatkan XOOPS"
description: "Cara meningkatkan instalasi XOOPS yang sudah ada dengan aman ke versi yang lebih baru."
---

Bagian ini memandu Anda dalam mengupgrade situs XOOPS yang sudah ada. Wizard pemutakhiran mempertahankan data, module, dan theme Anda saat memperbarui file core.

## Jalur peningkatan

| Langkah | Halaman | Apa yang harus dilakukan |
|------|------|------------|
| 1 | [Persiapan](./step-01/) | Cadangkan, tinjau log perubahan, unduh rilis baru |
| 2 | [Pemeriksaan Pra-Penerbangan](./preflight/) | Verifikasi kompatibilitas sebelum menjalankan pemutakhiran |
| 3 | [Menjalankan Peningkatan](./step-02/) | Jalankan wizard pemutakhiran |
| 4 | [Pemecahan Masalah](./step-03/) | Selesaikan masalah peningkatan umum |
| 5 | [Setelah Peningkatan](./step-04/) | Verifikasi, perbarui module, hapus cache |

:::tip
Selalu buat **cadangan basis data lengkap dan cadangan file** sebelum melakukan peningkatan. Peningkatan ini tidak dapat diubah tanpa cadangan.
:::
