| 7Zip www.7zip.org | Program Sumber Terbuka ke file zip/unzip. |
| WinSCP www.WinSCP.net | WinSCP adalah klien SFTP open source dan klien FTP untuk Windows. Fungsi utamanya adalah transfer file yang aman antara komputer lokal dan komputer jarak jauh. |
| Server WAMP www.wampserver.com | WampServer adalah lingkungan pengembangan web Windows. Ini memungkinkan Anda membuat aplikasi web dengan Apache, PHP dan database MySQL. Itu juga dilengkapi dengan PHPMyAdmin untuk mengelola database Anda dengan mudah.  WampServer menginstal secara otomatis (installer), dan penggunaannya sangat intuitif. Anda akan dapat menyetel server Anda bahkan tanpa menyentuh file pengaturan. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP adalah distribusi Apache yang mudah diinstal yang berisi MySQL, PHP dan Perl. XAMPP sangat mudah dipasang dan digunakan - cukup unduh, ekstrak, dan mulai. |
| KeyPass www.keepass.info | KeePass adalah pengelola kata sandi sumber terbuka gratis, yang membantu Anda mengelola kata sandi dengan cara yang aman. Anda dapat memasukkan semua kata sandi Anda ke dalam satu database, yang dikunci dengan satu kunci master atau file kunci. Jadi, Anda hanya perlu mengingat satu kata sandi utama atau memilih file kunci untuk membuka kunci seluruh database. Basis data dienkripsi menggunakan algoritma enkripsi terbaik dan teraman yang dikenal saat ini (AES dan Twofish). |
