## Tentang CMS XOOPS

XOOPS adalah Sistem Manajemen Konten (CMS) dan program portal web yang memungkinkan administrator dengan mudah membuat situs web dinamis dengan konten hebat dan banyak fitur luar biasa. Ini adalah alat yang ideal untuk mengembangkan situs komunitas dinamis kecil hingga besar, portal intra-perusahaan, portal perusahaan, weblog, dan banyak lagi. Itu dapat diinstal pada host mana pun dengan server web berkemampuan PHP (misalnya Apache atau Nginx) dan database yang kompatibel dengan MySQL.

**Situs web proyek utama:** [https://xoops.org](https://xoops.org)

XOOPS dirilis di bawah ketentuan GNU General Public License (GPL) dan bebas untuk digunakan dan dimodifikasi. Anda dapat mendistribusikannya kembali selama Anda mematuhi ketentuan distribusi GPL.

### Apa kepanjangan dari XOOPS

XOOPS adalah akronim dari **Sistem Portal Berorientasi Objek eXtensible**. Meskipun dimulai sebagai sistem portal, kini menjadi Sistem Manajemen Konten berfitur lengkap yang dapat melayani situs pribadi kecil melalui instalasi perusahaan besar.

### Pengucapan

Pengucapan standar XOOPS mengikuti aturan bahasa Inggris dan diucapkan `[zoo'ps]`.

### Fitur utama

**Berbasis database**

XOOPS menggunakan database relasional (MySQL, 5.7.8 atau lebih baru) untuk menyimpan data yang diperlukan untuk menjalankan sistem manajemen konten berbasis web.

**Sepenuhnya termodulasi**

module dapat diinstal, dihapus, diaktifkan, atau dinonaktifkan dengan satu klik melalui sistem administrasi module XOOPS.

**Personalisasi**

Pengguna terdaftar dapat mengedit profil mereka, memilih theme situs, mengunggah avatar khusus, dan banyak lagi.

**Manajemen pengguna**

XOOPS menawarkan pencarian antar pengguna berdasarkan berbagai kriteria, email, dan sistem pesan pribadi berbasis template.

**Dukungan di seluruh dunia**

XOOPS dibuat dan dikelola oleh tim sukarelawan dari seluruh dunia. Komunitas XOOPS memiliki situs dukungan dalam berbagai bahasa.

**Dukungan bahasa multibita**

XOOPS sepenuhnya mendukung bahasa multibyte, termasuk Jepang, Cina Sederhana dan Tradisional, Korea, dan Arab (dengan dukungan RTL ditambahkan di 2.7.0).

**Izin grup serbaguna**

Sistem izin yang kuat dan mudah digunakan memungkinkan administrator untuk mengatur izin berdasarkan grup.

**Antarmuka skinnable berbasis theme**

Sistem theme memungkinkan administrator dan pengguna mengubah tampilan seluruh situs. Distribusi 2.7.0 dikirimkan dengan theme modern berbasis Bootstrap-5 dan Tailwind.
