## 關於 XOOPS CMS

XOOPS 是一個內容管理系統 (CMS) 和網路入口程式，可讓管理員輕鬆建立具有豐富內容和許多傑出功能的動態網站。它是開發小型至大型動態社群網站、公司內部入口、企業入口、網誌等的理想工具。可以在任何具有 PHP 功能的網頁伺服器 (如 Apache 或 Nginx) 和 MySQL 相容資料庫的主機上安裝。

**主要專案網站：** [https://xoops.org](https://xoops.org)

XOOPS 以 GNU 通用公用授權 (GPL) 條款發佈，可免費使用和修改。只要您遵守 GPL 的發佈條款，您可以轉售它。

### XOOPS 代表什麼

XOOPS 是 **eXtensible Object Oriented Portal System** (可擴充物件導向入口系統) 的縮寫。雖然它最初是作為入口系統開始的，但現在是一個功能完整的內容管理系統，可以提供小型個人網站至大型企業安裝。

### 發音

XOOPS 的標準發音遵循英文規則，發音為 `[zoo'ps]`。

### 主要功能

**資料庫驅動**

XOOPS 使用關聯式資料庫 (MySQL 5.7.8 或更新版本) 來儲存執行網路型內容管理系統所需的資料。

**完全模組化**

模組可以透過 XOOPS 模組管理系統按一下滑鼠進行安裝、卸載、啟用或停用。

**個人化**

註冊使用者可以編輯他們的個人檔案、選擇網站主題、上傳自訂頭像等。

**使用者管理**

XOOPS 提供按各種條件搜尋使用者、電子郵件和以樣板為基礎的私人訊息系統。

**全球支援**

XOOPS 由來自世界各地的志願者團隊建立和維護。XOOPS 社群在多種語言中具有支援網站。

**多位元組語言支援**

XOOPS 完全支援多位元組語言，包括日語、簡體中文和繁體中文、韓語以及阿拉伯語 (在 2.7.0 中新增了 RTL 支援)。

**多功能群組權限**

強大且使用者易用的權限系統允許管理員按群組設定權限。

**基於主題的可換膚介面**

主題系統可讓管理員和使用者變更整個網站的外觀。2.7.0 發行版附帶最新的 Bootstrap-5 和基於 Tailwind 的主題。
