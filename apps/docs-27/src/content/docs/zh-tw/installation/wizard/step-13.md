此頁面收集您對要在新網站中安裝的模組選擇。如果需要，可以隨時新增或移除模組。

輸入所需資訊並修正任何問題後，選擇「繼續」按鈕以繼續。

![XOOPS 安裝程式模組安裝](/xoops-docs/2.7/img/installation/installer-13.png)

## 此步驟中收集的資料

### 可用模組

選擇「是」以安裝模組。您可以選擇無、任何或所有模組。

這些是標準 XOOPS 2.7.0 安裝隨附的模組。

* 私人訊息 - _使用者之間的增強型私人訊息傳遞_
* 使用者資料 - _增強/自訂使用者資料_
* 保護器 - _增強的網站安全性、垃圾郵件控制和攻擊緩解_
* DebugBar - _基於 Symfony VarDumper 的開發和測試偵錯工具欄；對模組開發人員和診斷生產問題很有用，但通常在面向公眾的實時網站上不需要_

_系統_ 模組始終安裝，此處未列為選項。

#### 注意

在啟動安裝前，您可以在上傳 XOOPS 檔案時將模組新增到模組目錄，它們也將在此步驟中可用。

#### 強烈建議

**所有 XOOPS 網站** 應安裝 _保護器_ 模組。
