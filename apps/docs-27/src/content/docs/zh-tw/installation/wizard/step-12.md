此頁面收集您對新網站初始佈景主題的選擇。佈景主題控制您網站的視覺特徵。如果需要，可以隨時變更佈景主題。

輸入所需資訊並修正任何問題後，選擇「繼續」按鈕以繼續。

![XOOPS 安裝程式選擇佈景主題](/xoops-docs/2.7/img/installation/installer-12.png)

## 此步驟中收集的資料

### 選擇預設佈景主題

#### 預設佈景主題

從清單中為網站選擇初始佈景主題。選擇時會顯示每個佈景主題的螢幕截圖。

XOOPS 2.7.0 開箱即用地隨附多個前端佈景主題：

* **default** — 最小基準佈景主題，作為自訂的起點很有用。
* **xbootstrap5** — Bootstrap 5 佈景主題，建議大多數新網站使用。
* **xswatch5** — Bootswatch 5 變體；一組在 Bootstrap 5 之上的樣式化子佈景主題。
* **xtailwind** — 基於 Tailwind CSS 的佈景主題 + 具有 35 個調色盤和 Alpine.js 互動性的 DaisyUI
* **xtailwind2** — 藝術導向的姊妹佈景主題，具有策劃的調色盤和精緻的模組樣式

您可以隨時從管理區域變更佈景主題，並可將其他佈景主題上傳到 `themes/` 目錄。
