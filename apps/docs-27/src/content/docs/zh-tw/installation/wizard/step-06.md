## 此步驟中收集的資料

### 資料庫

#### 資料庫名稱

XOOPS 應使用的主機上的資料庫名稱。在上一步中輸入的資料庫使用者應在此資料庫上具有所有權限。安裝程式將嘗試建立此資料庫（如果不存在）。

#### 資料表前綴

此前綴將新增至 XOOPS 建立的所有新資料表的名稱中。這有助於避免名稱衝突（如果資料庫與其他應用程式共用）。唯一的前綴也使得更難猜測資料表名稱，這具有安全優勢。如果您不確定，只需保留預設值

#### 資料庫字元集

安裝程式預設為 `utf8mb4`，它支援完整的 Unicode 範圍，包括表情符號和補充字元。您可以在此選擇不同的字元集，但建議在幾乎所有語言和地區設定中使用 `utf8mb4`，除非您有特定的理由來變更。

#### 資料庫排序

預設情況下，排序欄位留空。當空白時，MySQL 套用所選字元集的預設排序（對於 `utf8mb4`，通常為 `utf8mb4_general_ci` 或 `utf8mb4_0900_ai_ci`，具體取決於 MySQL 版本）。如果您需要特定的排序 — 例如要符合現有的資料庫 — 請在此選擇。否則，留空是建議的選擇。
