本節介紹如何升級現有XOOPS網站。升級精靈在更新核心文件時保留您的數據、模塊和主題。

## 升級路徑

| 步驟 | 頁面 | 要執行的操作 |
|------|------|------------|
| 1 | [準備](./step-01/) | 備份、查看更新日誌、下載新版本 |
| 2 | [飛行前檢查](./preflight/) | 在運行升級前驗證兼容性 |
| 3 | [運行升級](./step-02/) | 執行升級精靈 |
| 4 | [故障排除](./step-03/) | 解決常見升級問題 |
| 5 | [升級後](./step-04/) | 驗證、更新模塊、清除緩存 |

:::tip
始終在升級前進行**完整數據庫備份和文件備份**。沒有備份，升級是不可逆的。
:::
