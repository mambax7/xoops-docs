| 軟體 | 描述 |
| --- | --- |
| 7Zip www.7zip.org | 開源程式以壓縮/解壓縮檔案。 |
| WinSCP www.WinSCP.net | WinSCP 是 Windows 的開源 SFTP 用戶端和 FTP 用戶端。其主要功能是本地電腦和遠端電腦之間的安全檔案傳輸。 |
| WAMP Server www.wampserver.com | WampServer 是 Windows 網頁開發環境。它允許您使用 Apache、PHP 和 MySQL 資料庫建立網頁應用程式。它也附帶 PHPMyAdmin 以輕鬆管理您的資料庫。WampServer 自動安裝（安裝程式），使用非常直觀。您將能夠調整您的伺服器，甚至無需觸及設定檔案。 |
| XAMPP  [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP 是一個簡單易用的 Apache 發佈版，包含 MySQL、PHP 和 Perl。XAMPP 非常易於安裝和使用 - 只需下載、解壓並啟動。 |
| KeyPass www.keepass.info | KeePass 是一個免費的開源密碼管理員，可幫助您以安全的方式管理密碼。您可以將所有密碼放在一個資料庫中，該資料庫使用一個主金鑰或金鑰檔案鎖定。因此，您只需要記住一個主密碼或選擇金鑰檔案來解鎖整個資料庫。使用目前已知的最佳和最安全的加密演算法（AES 和 Twofish）進行加密。 |
| Package Web Tools [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === USB 應用程式 ==== - 7zip、Filezilla、Kompozer、DataBase Browser、Opera、PicPick、Winmerge、Gimp、Notepad++、OikoCSSeditor、SubCommander、ToDoList、Toucan、Xenu、TeamViewer、Pidgin、FastImageOptimizer、VirtualWin 和 MiniServer（帶有最新 Apache - MySQL - PHP，以及用於管理資料庫的 Adminer） |
| Package Web Browsers [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === USB 應用程式 ==== - Firefox + webdevelopper 和 XooFoo 擴充功能（穩定版和開發版本）、IETester 0.4.5、Opera（穩定版和開發版本）、OperaMobile Emulator、Safari、Google Chrome + webdevelopper 擴充功能（穩定版和開發版本）、Maxthon（穩定版和開發版本）、SeaMonkey、Avant 瀏覽器、Flock 和 Elinks |

| Mini Server (just 5MB) by XooFoo [http://goo.gl/Ke0LW](http://goo.gl/Ke0LW) | 一個基本的便攜式生產伺服器，具有 PHP 的附加功能，允許除了靜態 htm 和 html 頁面外還有動態網頁。設計為允許在同一臺 PC 或 USB 光碟上執行多個伺服器，這些伺服器需要一個自動建立的單獨虛擬磁碟機。MySQL 支援已啟用，進一步增強了此伺服器的功能。 |
