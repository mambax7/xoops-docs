In dit gedeelte wordt u begeleid bij het upgraden van een bestaande XOOPS-site. De upgradewizard bewaart uw gegevens, modules en thema's terwijl de kernbestanden worden bijgewerkt.

## Upgradepad

| Stap | Pagina | Wat te doen |
|------|------|------------|
| 1 | [Voorbereidingen](./step-01/) | Back-up maken, changelog bekijken, nieuwe release downloaden |
| 2 | [Preflight-controle](./preflight/) | Controleer de compatibiliteit voordat u de upgrade |
| 3 | [De upgrade uitvoeren](./step-02/) | Voer de upgradewizard uit |
| 4 | [Problemen oplossen](./step-03/) | Veelvoorkomende upgradeproblemen oplossen |
| 5 | [Na de upgrade](./step-04/) | Verifiëren, modules bijwerken, cache wissen |

:::tip
Maak altijd een **volledige databaseback-up en bestandsback-up** voordat u een upgrade uitvoert. De upgrade is onomkeerbaar zonder back-up.
:::
