| Software | Beschrijving |

| 7Zip www.7zip.org | Open Source-programma om bestanden te zip/unzippen. |
| WinSCP www.WinSCP.net | WinSCP is een open source SFTP-client en FTP-client voor Windows. De belangrijkste functie is de veilige bestandsoverdracht tussen een lokale en een externe computer. |
| WAMP-server www.wampserver.com | WampServer is een Windows-webontwikkelomgeving. Hiermee kunt u webapplicaties maken met Apache, PHP en de MySQL-database. Het wordt ook geleverd met PHPMyAdmin om uw databases eenvoudig te beheren.  WampServer wordt automatisch geïnstalleerd (installatieprogramma) en het gebruik ervan is zeer intuïtief. U kunt uw server afstemmen zonder zelfs maar de instellingsbestanden aan te raken. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP is een eenvoudig te installeren Apache-distributie die MySQL, PHP en Perl bevat. XAMPP is echt heel eenvoudig te installeren en te gebruiken - gewoon downloaden, uitpakken en starten. |
| KeyPass www.keepass.info | KeePass is een gratis open source wachtwoordbeheerder, waarmee u uw wachtwoorden op een veilige manier kunt beheren. U kunt al uw wachtwoorden in één database plaatsen, die wordt afgesloten met één hoofdsleutel of een sleutelbestand. U hoeft dus slechts één enkel hoofdwachtwoord te onthouden of het sleutelbestand te selecteren om de hele database te ontgrendelen. De databases zijn gecodeerd met behulp van de beste en veiligste encryptie-algoritmen die momenteel bekend zijn (AES en Twofish). |
| Webtools pakket [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === USB Apps ==== - 7zip, Filezilla, Kompozer, DataBase Browser, Opera, PicPick, Winmerge, Gimp, Notepad++, OikoCSSeditor, SubCommander, ToDoList, Toucan, Xenu, TeamViewer, Pidgin, FastImageOptimizer, VirtualWin en MiniServer (met laatste Apache - MySQL - PHP en beheerder om de database te beheren) |
| Webbrowsers pakket [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === USB Apps ==== - Firefox + webdevelopper en XooFoo extensies (stabiele en ontwikkelingsversie), IETester 0.4.5, Opera (stabiele en ontwikkelingsversie), OperaMobile Emulator, Safari, Google Chrome + webdevelopper extensie (stabiele en ontwikkelingsversie), Maxthon (stabiele en ontwikkelingsversie), SeaMonkey, Avant browser, Flock en Elinks |
