---
title: "Tablo Oluşturma"
---
Bu sayfada veritabanında XOOPS tablolarının oluşturulmasının sonuçları görüntülenir.

Sorunları inceleyip düzelttikten sonra devam etmek için "Devam Et" düğmesini seçin.

## Başarı Üzerine

_Veritabanı tabloları oluşturma_ bölümü, oluşturulan database tablolarını gösterir.

![XOOPS Installer Table Creation](/xoops-docs/2.7/img/installation/installer-08.png)

## Hatalar

Herhangi bir hata, önceki adımlarda girilen MySQL database user hesabı and/or MySQL veritabanıyla ilgili izin gerektiren sorunları gösterir. Menüdeki database bağlantısı ve database yapılandırma seçeneklerini kullanarak bu bölümlere dönebilirsiniz.
