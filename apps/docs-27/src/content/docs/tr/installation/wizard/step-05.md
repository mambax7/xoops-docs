Bu sayfa XOOPS'in kullanacağı database user hesabı hakkındaki bilgileri toplar.

İstenilen bilgileri girdikten ve sorunları düzelttikten sonra devam etmek için "Devam Et" düğmesini seçin.

![XOOPS Installer Database Connection](/xoops-docs/2.7/img/installation/installer-05.png)

## Bu Adımda Toplanan Veriler

### Sunucu bağlantısı

#### database

Şu anda mevcut tek seçenek _mysql_'dir.

#### Sunucu ana bilgisayar adı

database sunucusunun ana bilgisayar adı. Emin değilseniz çoğu durumda _localhost_ işe yarar.

#### user adı

XOOPS'ün database sunucusuna bağlanmak için kullanması gereken database user hesabı.

#### Şifre

Yukarıda girilen database user hesabının şifresi.

#### Kalıcı bağlantı kullan

Varsayılan 'Hayır'dır. Emin değilseniz burayı boş bırakın.
