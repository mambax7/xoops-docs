XOOPS kurulum sihirbazı, veritabanınızı yapılandıran, yönetici hesabını oluşturan ve sitenizi ilk kullanıma hazırlayan 15 adımlı bir süreçte size yol gösterir.

## Başlamadan önce

- [XOOPS'yi sunucunuza yüklediniz](/xoops-docs/2.7/installation/ftp-upload/) veya yerel bir ortam kurdunuz
- [Gereksinimleri doğruladınız](/xoops-docs/2.7/installation/requirements/)
- database kimlik bilgileriniz hazır

## Sihirbaz adımları

| Adım | Ekran | Ne olur |
|------|--------|-------------|
| 1 | [Dil Seçimi](./step-01/) | Kurulum dilini seçin |
| 2 | [Hoş Geldiniz](./step-02/) | Lisans sözleşmesi |
| 3 | [Yapılandırma Kontrolü](./step-03/) | PHP/server ortam kontrolü |
| 4 | [Yol Ayarı](./step-04/) | Kök yolunu ayarlayın ve URL |
| 5 | [database Bağlantısı](./step-05/) | database ana bilgisayarını, kullanıcıyı, şifreyi girin |
| 6 | [database Yapılandırması](./step-06/) | database adını ve tablo önekini ayarlayın |
| 7 | [Yapılandırmayı Kaydet](./step-07/) | mainfile.php yazın |
| 8 | [Tablo Oluşturma](./step-08/) | database şemasını oluşturun |
| 9 | [Başlangıç ​​Ayarları](./step-09/) | Site adı, yönetici e-postası |
| 10 | [Veri Ekleme](./step-10/) | Varsayılan verileri doldur |
| 11 | [Site Yapılandırması](./step-11/) | URL, saat dilimi, dil |
| 12 | [theme Seçin](./step-12/) | Varsayılan bir theme seçin |
| 13 | [module Kurulumu](./step-13/) | Birlikte verilen modülleri yükleyin |
| 14 | [Hoş Geldiniz](./step-14/) | Kurulum tamamlandı mesajı |
| 15 | [Temizleme](./step-15/) | Yükleme klasörünü kaldırın |

:::dikkat[Güvenlik]
Sihirbazı tamamladıktan sonra **`install/` klasörünü silin veya yeniden adlandırın** — 15. adım bu konuda size yol gösterecektir. Erişilebilir bırakmak bir güvenlik riskidir.
:::
