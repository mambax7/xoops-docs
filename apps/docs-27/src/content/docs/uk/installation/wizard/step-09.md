На цій сторінці збирається інформація, яка використовується для створення основного облікового запису адміністратора (він же веб-майстер).

Після введення потрібної інформації та виправлення будь-яких проблем натисніть кнопку «Продовжити», щоб продовжити.

![XOOPS Початкові параметри інсталятора](/xoops-docs/2.7/img/installation/installer-09.png)

## Дані, зібрані на цьому кроці

### Обліковий запис адміністратора

#### Логін адміністратора

Це буде ім’я користувача, яке використовує адміністратор для входу в систему XOOPS.

#### Електронна пошта адміністратора

Це електронна адреса, яка використовується системою XOOPS для зв’язку з адміністратором. (Зверніть увагу, це потрібно для пароля recovery/reset.)

#### Пароль адміністратора

Це пароль для облікового запису адміністратора,

#### Підтвердьте пароль

Використовуйте це поле для підтвердження пароля адміністратора. Він має збігатися з полем _Пароль адміністратора_.

### Надійність пароля

У цьому розділі відображається відносна надійність _паролю адміністратора_.

#### Генератор паролів

Ви можете використовувати цей розділ, щоб створити та використовувати випадковий пароль для облікового запису. Виберіть _Generate_, щоб показати випадковий пароль. Виберіть _Копіювати_, щоб скопіювати згенерований пароль у _Пароль адміністратора_.
