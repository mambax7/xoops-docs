На цій сторінці зібрано ваш вибір початкової теми для вашого нового сайту. Теми контролюють візуальний характер вашого сайту. За бажанням теми можна змінити в будь-який час.

Після введення потрібної інформації та виправлення будь-яких проблем натисніть кнопку «Продовжити», щоб продовжити.

![XOOPS Installer Select Theme](/xoops-docs/2.7/img/installation/installer-12.png)

## Дані, зібрані на цьому кроці

### Виберіть тему за замовчуванням

#### Тема за замовчуванням

Виберіть початкову тему для свого сайту зі списку. Скріншот кожної теми відображається під час її вибору.

XOOPS 2.7.0 поставляється з декількома інтерфейсними темами з коробки:

* **за замовчуванням** — мінімальна базова тема, корисна як відправна точка для налаштування.
* **xbootstrap5** — тема Bootstrap 5, рекомендована для більшості нових сайтів.
* **xswatch5** — варіанти Bootswatch 5; набір стилізованих підтем поверх Bootstrap 5.
* **xtailwind** — тема на основі Tailwind CSS + DaisyUI з 35 палітрами та інтерактивністю Alpine.js
* **xtailwind2** — мистецька рідна тема з підібраними палітрами та вдосконаленим стилем модуля

Ви можете будь-коли змінити тему в області адміністрування, а додаткові теми можна завантажити в каталог `themes/`.
