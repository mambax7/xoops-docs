На цій сторінці збирається інформація про базу даних, яку використовуватиме XOOPS.

Після введення потрібної інформації та виправлення будь-яких проблем натисніть кнопку «Продовжити», щоб продовжити.

![XOOPS Конфігурація бази даних інсталятора](/xoops-docs/2.7/img/installation/installer-06.png)

## Дані, зібрані на цьому кроці

### База даних

#### Database name

Ім’я бази даних на хості, яку має використовувати XOOPS. Користувач бази даних, введений на попередньому кроці, повинен мати всі привілеї щодо цієї бази даних. Інсталятор спробує створити цю базу даних, якщо вона не існує.

#### Table prefix

Цей префікс буде додано до назв усіх нових таблиць, створених XOOPS. Це допомагає уникнути конфліктів імен, якщо база даних використовується спільно з іншими програмами. Унікальний префікс також ускладнює вгадування імен таблиць, що має переваги безпеки. Якщо ви не впевнені, просто збережіть значення за умовчанням

#### Набір символів бази даних

Інсталятор за замовчуванням використовує `utf8mb4`, який підтримує повний діапазон Unicode, включаючи emoji та додаткові символи. Тут можна вибрати інший набір символів, але `utf8mb4` рекомендовано практично для всіх мов і локалей, і його слід залишити як є, якщо у вас немає конкретних причин змінити його.

#### Database collation

Поле зіставлення за замовчуванням залишається порожнім. Якщо поле пусте, MySQL застосовує сортування за замовчуванням для будь-якого набору символів, вибраного вище (для `utf8mb4` це зазвичай `utf8mb4_general_ci` або `utf8mb4_0900_ai_ci`, залежно від версії MySQL). Якщо вам потрібне певне зіставлення — наприклад, щоб відповідати існуючій базі даних — виберіть його тут. В іншому випадку рекомендуємо залишити його порожнім.
