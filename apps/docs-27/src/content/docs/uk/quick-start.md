## Вимоги

| Компонент | Мінімум | Рекомендовано |
|------------|------------------------|--------------|
| PHP | 8.2 | 8,4+ |
| MySQL | 5.7 | 8.0+ |
| MariaDB | 10.4 | 10.11+ |
| Веб-сервер | Apache 2.4 / Nginx 1.20 | Останній стабільний |

## Завантажити

Завантажте останню версію з [GitHub Releases](https://github.com/XOOPS/XoopsCore27/releases).
```bash
# Or clone directly
git clone https://github.com/XOOPS/XoopsCore27.git mysite
cd mysite
```
## Етапи встановлення

1. **Завантажте файли** в корінь документа веб-сервера (наприклад, `public_html/`).
2. **Створіть базу даних MySQL** і користувача з повними правами на неї.
3. **Відкрийте браузер** і перейдіть до свого домену — інсталятор XOOPS запускається автоматично.
4. **Дотримуйтеся 5-крокового майстра** — він налаштовує шляхи, створює таблиці та налаштовує обліковий запис адміністратора.
5. **Видаліть папку `install/`**, коли буде запропоновано. Це обов’язково для безпеки.

## Перевірте встановлення

Після налаштування відвідайте:

- **Головна сторінка:** `https://yourdomain.com/`
- **Панель адміністратора:** `https://yourdomain.com/xoops_data/` *(шлях, який ви вибрали під час встановлення)*
