| 7Zip www.7zip.org | Програма з відкритим кодом для файлів zip/unzip. |
| WinSCP www.WinSCP.net | WinSCP — це клієнт SFTP і FTP з відкритим кодом для Windows. Його основною функцією є безпечна передача файлів між локальним і віддаленим комп’ютером. |
| Сервер WAMP www.wampserver.com | WampServer — це середовище веб-розробки Windows. Він дозволяє створювати веб-додатки з Apache, PHP і базою даних MySQL. Він також поставляється з PHPMyAdmin для легкого керування базами даних.  WampServer встановлюється автоматично (інсталятор), і його використання дуже інтуїтивно зрозуміле. Ви зможете налаштувати свій сервер, навіть не торкаючись файлів налаштувань. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP — це простий у встановленні дистрибутив Apache, що містить MySQL, PHP і Perl. XAMPP дійсно дуже простий у встановленні та використанні - просто завантажте, розпакуйте та запустіть. |
| KeyPass www.keepass.info | KeePass — це безкоштовний менеджер паролів із відкритим вихідним кодом, який допомагає вам безпечно керувати своїми паролями. Ви можете помістити всі свої паролі в одну базу даних, яка заблокована одним головним ключем або файлом ключів. Тож вам потрібно лише запам’ятати один головний пароль або вибрати файл ключа, щоб розблокувати всю базу даних. Бази даних зашифровано з використанням найкращих і найбезпечніших відомих на даний момент алгоритмів шифрування (AES і Twofish). |
