Ovaj odjeljak vodi vas kroz nadogradnju postojeće XOOPS stranice. Čarobnjak za nadogradnju čuva vaše podatke, modules i themes dok ažurira osnovne datoteke.

## Put nadogradnje

| Korak | Stranica | Što učiniti |
|------|------|------------|
| 1 | [Pripreme](./step-01/) | Izrada sigurnosne kopije, pregled dnevnika promjena, preuzimanje novog izdanja |
| 2 | [Provjera prije leta](./preflight/) | Provjerite kompatibilnost prije pokretanja nadogradnje |
| 3 | [Pokretanje nadogradnje](./step-02/) | Izvršite čarobnjaka za nadogradnju |
| 4 | [Rješavanje problema](./step-03/) | Rješavanje uobičajenih problema s nadogradnjom |
| 5 | [Nakon nadogradnje](./step-04/) | Potvrdi, ažuriraj modules, očisti cache |

:::napojnica
Prije nadogradnje uvijek napravite **potpunu sigurnosnu kopiju baze podataka i sigurnosnu kopiju datoteke**. Nadogradnja je nepovratna bez sigurnosne kopije.
:::
