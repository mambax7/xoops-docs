---
title: "Postavljanje staze"
---
Ova stranica prikuplja informacije o datotečnom sustavu i web adresi koje XOOPS treba za rad. Instalater će pokušati pogoditi ove informacije na temelju URL koji koristite za pristup i lokacija pokrenutih PHP skripti.

Nakon pregleda i ispravljanja problema odaberite gumb "Nastavi" za nastavak.

![XOOPS Postavke staze programa za instalaciju](/xoops-docs/2.7/img/installation/installer-04-02.png)

## Podaci prikupljeni u ovom koraku

### XOOPS Fizičke staze

#### XOOPS dokumentira korijenski fizički put

Fizička staza do direktorija dokumenata XOOPS (servirani) BEZ kose crte na kraju

#### XOOPS direktorij podatkovnih datoteka

Fizički put do direktorija podatkovnih datoteka XOOPS (za pisanje) BEZ kose crte na kraju. Pronađite mapu izvan korijena dokumenta XOOPS kako biste je osigurali.

#### XOOPS direktorij knjižnice

Fizički put do direktorija biblioteke XOOPS BEZ kose crte na kraju. Pronađite mapu izvan korijena dokumenta XOOPS kako biste je osigurali.

### Web lokacije

#### Lokacija web stranice (URL)

Glavni URL koji će se koristiti za pristup vašoj instalaciji XOOPS. XOOPS će ovo koristiti za izgradnju svih URL unutar XOOPS. Ako želite _https_ stranicu, svakako je navedite. Ako želite/ne želite vodeći _www._ na domeni svoje web stranice, svakako ga odredite kako želite.

Ovo će prema zadanim postavkama biti URL koji se koristi za pokretanje instalacijskog programa.

#### Domena kolačića za web stranicu

Domena za postavljanje kolačića. Može biti prazno, cijeli host iz URL (www.example.com) ili registrirana domena bez poddomena (example.com) za dijeljenje između poddomena (www.example.com i blog.example.com.)

## Pogreške

Ako se bilo koja od unesenih staza ne može pronaći, prikazat će se pogreška. Ispravite sve probleme prije nastavka.
