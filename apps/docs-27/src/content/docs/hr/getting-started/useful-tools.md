| Softver | Opis |
| --- | --- |
| 7Zip www.7zip.org | Program otvorenog koda za komprimiranje/raspakiranje datoteka. |
| WinSCP www.WinSCP.net | WinSCP je SFTP klijent i FTP klijent otvorenog koda za Windows. Njegova glavna funkcija je siguran prijenos datoteka između lokalnog i udaljenog računala. |
| WAMP poslužitelj www.wampserver.com | WampServer je Windows okruženje za web razvoj. Omogućuje vam izradu web aplikacija s Apacheom, PHP i MySQL bazom podataka. Također dolazi s PHPMyAdmin za jednostavno upravljanje vašim bazama podataka. WampServer se instalira automatski (installer), a njegovo korištenje je vrlo intuitivno. Moći ćete podesiti svoj poslužitelj čak i bez dodirivanja datoteka s postavkama. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP je Apache distribucija koja se lako instalira i sadrži MySQL, PHP i Perl. XAMPP je stvarno vrlo jednostavan za instalaciju i korištenje - samo preuzmite, izdvojite i pokrenite. |
| KeyPass www.keepass.info | KeePass je besplatni upravitelj lozinki otvorenog koda koji vam pomaže da upravljate svojim lozinkama na siguran način. Sve svoje lozinke možete staviti u jednu bazu podataka, koja je zaključana jednim glavnim ključem ili datotekom ključa. Dakle, morate zapamtiti samo jednu glavnu lozinku ili odabrati ključnu datoteku za otključavanje cijele baze podataka. Baze podataka su šifrirane korištenjem najboljih i najsigurnijih trenutno poznatih algoritama šifriranja (AES i Twofish). |
