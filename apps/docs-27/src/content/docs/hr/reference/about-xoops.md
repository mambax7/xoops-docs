## O XOOPS CMS

XOOPS je sustav za upravljanje sadržajem (CMS) i program web portala koji administrators omogućuje jednostavnu izradu dinamičkih web stranica s izvrsnim sadržajem i mnogim izvrsnim značajkama. To je idealan alat za razvoj malih i velikih dinamičnih web stranica zajednice, portala unutar poduzeća, korporativnih portala, web dnevnika i još mnogo toga. Može se instalirati na bilo koji host s web poslužiteljem koji podržava PHP (npr. Apache ili Nginx) i bazom podataka kompatibilnom sa MySQL.

**Glavna web stranica projekta:** [https://xoops.org](https://xoops.org)

XOOPS je objavljen pod uvjetima GNU Opće javne licence (GPL) i slobodan je za korištenje i modificiranje. Možete ga dalje distribuirati sve dok se pridržavate uvjeta distribucije GPL-a.

### Što XOOPS označava

XOOPS je akronim za **EXtensible Object Oriented Portal System**. Iako je započeo kao portalski sustav, sada je potpuno opremljen sustav upravljanja sadržajem koji može opsluživati ​​male osobne stranice kroz velike poslovne instalacije.

### Izgovor

Standardni izgovor XOOPS slijedi engleska pravila i izgovara se `[zoo'ps]`.

### Ključne značajke

**Upravljano bazom podataka**

XOOPS koristi relacijsku bazu podataka (MySQL, 5.7.8 ili novija) za pohranjivanje podataka potrebnih za pokretanje sustava za upravljanje sadržajem koji se temelji na webu.

**Potpuno modulariziran**

moduli se mogu instalirati, deinstalirati, aktivirati ili deaktivirati klikom preko XOOPS modula administration sustava.

**Personalizacija**

Registrirani korisnici mogu uređivati svoje profile, odabrati stranicu themes, učitati prilagođene avatare i još mnogo toga.

**Upravljanje korisnicima**

XOOPS nudi pretraživanje korisnika prema različitim kriterijima, e-poštu i sustav privatnih poruka temeljen na predlošcima.

**Podrška u cijelom svijetu**

XOOPS kreirao je i održava tim volontera iz cijelog svijeta. Zajednica XOOPS ima stranice za podršku u više languages.

**Podrška za višebajtni language**

XOOPS u potpunosti podržava višebajtni languages, uključujući japanski, pojednostavljeni i tradicionalni kineski, korejski i arapski (s RTL podrškom dodanom u 2.7.0).

**Svestrane grupne dozvole**

Snažan i jednostavan sustav dopuštenja omogućuje administrators postavljanje dopuštenja prema grupi.

**Sučelje koje se može skinuti na temelju teme**

Sustav tema omogućuje i administrators i korisnicima promjenu izgleda cijele stranice. Distribucija 2.7.0 isporučuje se s modernim Bootstrap-5 i themes temeljenim na Tailwindu.
