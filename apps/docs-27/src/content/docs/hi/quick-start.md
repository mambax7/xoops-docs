## आवश्यकताएँ

| घटक | न्यूनतम | अनुशंसित |
|---------------|
| PHP | 8.2 | 8.4+ |
| MySQL | 5.7 | 8.0+ |
| MariaDB | 10.4 | 10.11+ |
| वेब सर्वर | अपाचे 2.4 / नग्नेक्स 1.20 | नवीनतम स्थिर |

## डाउनलोड करें

[GitHub रिलीज़](https://github.com/XOOPS/XoopsCore27/releases) से नवीनतम रिलीज़ डाउनलोड करें।

```bash
# Or clone directly
git clone https://github.com/XOOPS/XoopsCore27.git mysite
cd mysite
```

## स्थापना चरण

1. अपने वेब सर्वर दस्तावेज़ रूट पर **फ़ाइलें अपलोड करें** (उदाहरण के लिए `public_html/`)।
2. **एक MySQL डेटाबेस बनाएं** और उस पर पूर्ण विशेषाधिकार वाला एक उपयोगकर्ता बनाएं।
3. **अपना ब्राउज़र खोलें** और अपने डोमेन पर नेविगेट करें - XOOPS इंस्टॉलर स्वचालित रूप से प्रारंभ हो जाता है।
4. **5-चरणीय विज़ार्ड का पालन करें** - यह पथ कॉन्फ़िगर करता है, तालिकाएँ बनाता है, और आपका व्यवस्थापक खाता सेट करता है।
5. **संकेत मिलने पर `install/` फ़ोल्डर हटाएं**। सुरक्षा की दृष्टि से यह अनिवार्य है.

## स्थापना सत्यापित करें

सेटअप के बाद, यहां जाएं:

- **मुख्य पृष्ठ:** `https://yourdomain.com/`
- **एडमिन पैनल:** `https://yourdomain.com/xoops_data/` *(इंस्टॉल के दौरान आपके द्वारा चुना गया पथ)*
