> XOOPS सीएमएस के लिए प्रमुख समाचार और ब्लॉग प्रकाशन मॉड्यूल।

---

## अवलोकन

प्रकाशक XOOPS के लिए निश्चित सामग्री प्रबंधन मॉड्यूल है, जो SmartSection से विकसित होकर सबसे अधिक सुविधा संपन्न ब्लॉग और समाचार समाधान बन गया है। यह पूर्ण संपादकीय वर्कफ़्लो समर्थन के साथ सामग्री बनाने, व्यवस्थित करने और प्रकाशित करने के लिए व्यापक उपकरण प्रदान करता है।

**आवश्यकताएँ:**
- XOOPS 2.5.10+
- PHP 7.1+ (PHP 8.x अनुशंसित)

---

## 🌟 मुख्य विशेषताएं

### सामग्री प्रबंधन
- **श्रेणियाँ और उपश्रेणियाँ** - श्रेणीबद्ध सामग्री संगठन
- **रिच टेक्स्ट एडिटिंग** - एकाधिक WYSIWYG संपादक समर्थित
- **फ़ाइल अनुलग्नक** - लेखों में फ़ाइलें संलग्न करें
- **छवि प्रबंधन** - पृष्ठ और श्रेणी छवियां
- **फ़ाइल रैपिंग** - फ़ाइलों को लेख के रूप में लपेटें

### प्रकाशन वर्कफ़्लो
- **निर्धारित प्रकाशन** - भविष्य की प्रकाशन तिथियां निर्धारित करें
- **समाप्ति तिथियाँ** - सामग्री स्वतः समाप्त हो जाती है
- **संयम** - संपादकीय अनुमोदन वर्कफ़्लो
- **ड्राफ्ट प्रबंधन** - कार्य प्रगति पर सहेजें

### प्रदर्शन एवं टेम्पलेट्स
- **चार बेस टेम्प्लेट** - एकाधिक डिस्प्ले लेआउट
- **कस्टम टेम्पलेट** - अपने स्वयं के डिज़ाइन बनाएं
- **एसईओ अनुकूलन** - खोज इंजन अनुकूल URL
- **रिस्पॉन्सिव डिज़ाइन** - मोबाइल-रेडी आउटपुट

### उपयोगकर्ता सहभागिता
- **रेटिंग** - आलेख रेटिंग प्रणाली
- **टिप्पणियाँ** - पाठक चर्चाएँ
- **सामाजिक साझाकरण** - सामाजिक नेटवर्क पर साझा करें

### अनुमतियाँ
- **प्रस्तुति नियंत्रण** - लेख कौन प्रस्तुत कर सकता है
- **फ़ील्ड-स्तरीय अनुमतियाँ** - समूह द्वारा फ़ॉर्म फ़ील्ड को नियंत्रित करें
- **श्रेणी अनुमतियाँ** - प्रति श्रेणी अभिगम नियंत्रण
- **मॉडरेशन अधिकार** - वैश्विक मॉडरेशन सेटिंग्स

## 🚀 त्वरित शुरुआत

### 1. स्थापना

```bash
# Download from GitHub
git clone https://github.com/XoopsModules25x/publisher.git

# Copy to modules directory
cp -r publisher /path/to/xoops/htdocs/modules/
```

फिर XOOPS एडमिन → मॉड्यूल → इंस्टॉल के माध्यम से इंस्टॉल करें।

### 2. अपनी पहली श्रेणी बनाएं

1. **एडमिन → प्रकाशक → श्रेणियाँ** पर जाएँ
2. **श्रेणी जोड़ें** पर क्लिक करें
3. भरें:
   - **नाम**: समाचार
   - **विवरण**: नवीनतम समाचार और अपडेट
   - **छवि**: श्रेणी छवि अपलोड करें
4. सहेजें

### 3. अपना पहला लेख बनाएं

1. **एडमिन → प्रकाशक → लेख** पर जाएँ
2. **लेख जोड़ें** पर क्लिक करें
3. भरें:
   - **शीर्षक**: हमारी साइट पर आपका स्वागत है
   - **श्रेणी**: समाचार
   - **सामग्री**: आपके लेख की सामग्री
4. सेट **स्थिति**: प्रकाशित
5. सहेजें

## ⚙️ कॉन्फ़िगरेशन विकल्प

### सामान्य सेटिंग्स

| सेटिंग | विवरण | डिफ़ॉल्ट |
|--|----|---|
| संपादक | WYSIWYG संपादक का उपयोग करें | XOOPS डिफ़ॉल्ट |
| प्रति पृष्ठ आइटम | प्रति पृष्ठ दिखाए गए लेख | 10 |
| ब्रेडक्रम्ब दिखाएँ | नेविगेशन ट्रेल प्रदर्शित करें | हाँ |
| रेटिंग की अनुमति दें | आलेख रेटिंग सक्षम करें | हाँ |
| टिप्पणियों की अनुमति दें | लेख टिप्पणियाँ सक्षम करें | हाँ |

### एसईओ सेटिंग्स

| सेटिंग | विवरण | डिफ़ॉल्ट |
|--|----|---|
| एसईओ URL | अनुकूल URL सक्षम करें | नहीं |
| URL पुनर्लेखन | अपाचे mod_rewrite | कोई नहीं |
| मेटा कीवर्ड | स्वचालित रूप से कीवर्ड जेनरेट करें | हाँ |

### अनुमतियाँ मैट्रिक्स

| अनुमति | अनाम | पंजीकृत | संपादक | एडमिन |

| लेख सबमिट करें | ✗ | ✓ | ✓ | ✓ |
| स्वयं के लेख संपादित करें | ✗ | ✓ | ✓ | ✓ |
| सभी लेख संपादित करें | ✗ | ✗ | ✓ | ✓ |

## 🔄प्रवास

### से SmartSectionप्रकाशक में एक अंतर्निहित माइग्रेशन टूल शामिल है:

1. **एडमिन → प्रकाशक → आयात** पर जाएँ
2. स्रोत के रूप में **SmartSection** चुनें
3. आयात विकल्प चुनें:
   - श्रेणियाँ
   - लेख
   - टिप्पणियाँ
4. **आयात** पर क्लिक करें

### न्यूज़ मॉड्यूल से

1. **एडमिन → प्रकाशक → आयात** पर जाएँ
2. स्रोत के रूप में **समाचार** चुनें
3. मानचित्र श्रेणियां
4. **आयात** पर क्लिक करें
