# XOOPS में अपना पहला पेज बनाना

XOOPS में अपनी सामग्री का पहला भाग बनाना, प्रारूपित करना और प्रकाशित करना सीखें।

## XOOPS सामग्री को समझना

### पेज/पोस्ट क्या है?

XOOPS में, सामग्री को मॉड्यूल के माध्यम से प्रबंधित किया जाता है। सबसे आम सामग्री प्रकार हैं:

| प्रकार | विवरण | केस का प्रयोग करें |
|---|---|---|
| **पेज** | स्थैतिक सामग्री | हमारे बारे में, संपर्क, सेवाएँ |
| **पोस्ट/लेख** | समय-मुद्रांकित सामग्री | समाचार, ब्लॉग पोस्ट |
| **श्रेणी** | सामग्री संगठन | समूह संबंधी सामग्री |
| **टिप्पणी** | उपयोगकर्ता प्रतिक्रिया | आगंतुकों से बातचीत की अनुमति दें |

यह मार्गदर्शिका XOOPS' डिफ़ॉल्ट सामग्री मॉड्यूल का उपयोग करके एक मूल पृष्ठ/लेख बनाने को कवर करती है।

## सामग्री संपादक तक पहुँचना

### एडमिन पैनल से

1. एडमिन पैनल में लॉग इन करें: `http://your-domain.com/xoops/admin/`
2. **सामग्री > पेज** (या आपका सामग्री मॉड्यूल) पर नेविगेट करें
3. "नया पेज जोड़ें" या "नया पोस्ट" पर क्लिक करें

### फ्रंटएंड (यदि सक्षम हो)

यदि आपका XOOPS फ्रंटएंड सामग्री निर्माण की अनुमति देने के लिए कॉन्फ़िगर किया गया है:

1. पंजीकृत उपयोगकर्ता के रूप में लॉग इन करें
2. अपनी प्रोफ़ाइल पर जाएँ
3. "सामग्री सबमिट करें" विकल्प देखें
4. नीचे दिए गए समान चरणों का पालन करें

## सामग्री संपादक इंटरफ़ेस

सामग्री संपादक में शामिल हैं:

```
┌─────────────────────────────────────┐

├─────────────────────────────────────┤
│                                     │
│ Title: [________________]           │
│                                     │

│                                     │

│                                     │
└─────────────────────────────────────┘
```

## चरण-दर-चरण मार्गदर्शिका: अपना पहला पृष्ठ बनाना

### चरण 1: सामग्री संपादक तक पहुंचें

1. एडमिन पैनल में, **सामग्री > पेज** पर क्लिक करें
2. **"नया पेज जोड़ें"** या **"बनाएं"** पर क्लिक करें
3. आपको सामग्री संपादक दिखाई देगा

### चरण 2: पृष्ठ शीर्षक दर्ज करें

"शीर्षक" फ़ील्ड में, अपना पृष्ठ नाम दर्ज करें:

```
Title: Welcome to Our Website
```

शीर्षकों के लिए सर्वोत्तम अभ्यास:
- स्पष्ट और वर्णनात्मक
- यदि संभव हो तो कीवर्ड शामिल करें
- 50-60 अक्षर आदर्श
- सभी से बचें CAPS (पढ़ना कठिन)
- विशिष्ट बनें ("पेज 1" नहीं)

### चरण 3: श्रेणी चुनें

चुनें कि इस सामग्री को कहां व्यवस्थित करना है:

```
Category: [Dropdown ▼]
```

विकल्पों में शामिल हो सकते हैं:
- सामान्य
- समाचार
- ब्लॉग
- घोषणाएँ
- सेवाएँ

यदि श्रेणियां मौजूद नहीं हैं, तो व्यवस्थापक से उन्हें बनाने के लिए कहें।

### चरण 4: अपनी सामग्री लिखें

सामग्री संपादक क्षेत्र में क्लिक करें और अपना टेक्स्ट टाइप करें।

#### मूल पाठ स्वरूपण

संपादक टूलबार का उपयोग करें:

| बटन | कार्रवाई | परिणाम |
|---|---|---|
| **बी** | बोल्ड | **बोल्ड टेक्स्ट** |
| *मैं* | इटैलिक | *इटैलिक टेक्स्ट* |
| <u>U</u> | रेखांकित करें | <u>रेखांकित पाठ</u> |

#### HTML का उपयोग करना

XOOPS सुरक्षित HTML टैग की अनुमति देता है। सामान्य उदाहरण:

```html
<!-- Paragraphs -->
<p>This is a paragraph.</p>

<!-- Headings -->
<h1>Main Heading</h1>
<h2>Subheading</h2>

<!-- Lists -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<!-- Bold and Italic -->
<strong>Bold text</strong>
<em>Italic text</em>

<!-- Links -->
<a href="https://example.com">Link text</a>

<!-- Line breaks -->
<br>

<!-- Horizontal rule -->
<hr>
```

#### सुरक्षित HTML उदाहरण

**अनुशंसित टैग:**
- पैराग्राफ: `<p>`, `<br>`
- शीर्षक: `<h1>` से `<h6>`
- पाठ: `<strong>`, `<em>`, `<u>`
- सूचियाँ: `<ul>`, `<ol>`, `<li>`
- लिंक: `<a href="">`
- ब्लॉककोट: `<blockquote>`
- टेबल्स: `<table>`, `<tr>`, `<td>`

**इन टैग से बचें** (सुरक्षा के लिए अक्षम किया जा सकता है):
- स्क्रिप्ट: `<script>`
- शैलियाँ: `<style>`
- Iframes: `<iframe>` (जब तक कॉन्फ़िगर न किया गया हो)
- फॉर्म: `<form>`, `<input>`

### चरण 5: छवियाँ जोड़ें

#### विकल्प 1: छवि URL डालें

संपादक का उपयोग करना:

1. **छवि सम्मिलित करें** बटन (छवि आइकन) पर क्लिक करें
2. छवि URL दर्ज करें: `https://example.com/image.jpg`
3. वैकल्पिक पाठ दर्ज करें: "छवि का विवरण"
4. "सम्मिलित करें" पर क्लिक करें

HTML समकक्ष:

```html
<img src="https://example.com/image.jpg" alt="Description">
```

#### विकल्प 2: छवि अपलोड करें

1. सबसे पहले XOOPS पर छवि अपलोड करें:
   - **सामग्री > मीडिया प्रबंधक** पर जाएँ
   - अपनी छवि अपलोड करें
   - छवि URL कॉपी करें

2. सामग्री संपादक में, URL का उपयोग करके डालें (उपरोक्त चरण)

#### छवि सर्वोत्तम अभ्यास

- उचित फ़ाइल आकार का उपयोग करें (छवियों को अनुकूलित करें)
- वर्णनात्मक फ़ाइल नाम का प्रयोग करें
- हमेशा वैकल्पिक टेक्स्ट (पहुंच-योग्यता) शामिल करें
- समर्थित प्रारूप: JPG, PNG, GIF, WebP
- अनुशंसित चौड़ाई: सामग्री के लिए 600-800 पिक्सेल

### चरण 6: मीडिया एम्बेड करें

#### YouTube से वीडियो एम्बेड करें

```html
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  frameborder="0" allowfullscreen>
</iframe>
````VIDEO_ID` को YouTube वीडियो आईडी से बदलें।

**YouTube वीडियो आईडी ढूंढने के लिए:**
1. YouTube पर वीडियो खोलें
2. URL है: `https://www.youtube.com/watch?v=VIDEO_ID`
3. आईडी कॉपी करें (`v=` के बाद के अक्षर)

#### Vimeo से वीडियो एम्बेड करें

```html
<iframe src="https://player.vimeo.com/video/VIDEO_ID"
  width="640" height="360" frameborder="0"
  allow="autoplay; fullscreen" allowfullscreen>
</iframe>
```

### चरण 7: मेटा विवरण जोड़ें

"विवरण" फ़ील्ड में, एक संक्षिप्त सारांश जोड़ें:

```
Description: Learn how to get started with our website.
This page provides an overview of our services and how we can help you.
```

**मेटा विवरण सर्वोत्तम अभ्यास:**
- 150-160 अक्षर
- मुख्य कीवर्ड शामिल करें
- सामग्री का सटीक सारांश प्रस्तुत करना चाहिए
- खोज इंजन परिणामों में उपयोग किया जाता है
- इसे आकर्षक बनाएं (उपयोगकर्ता इसे देखें)

### चरण 8: प्रकाशन विकल्प कॉन्फ़िगर करें

#### स्थिति प्रकाशित करें

प्रकाशन स्थिति चुनें:

```
Status: ☑ Published
```

विकल्प:
- **प्रकाशित:** जनता के लिए दृश्यमान
- **ड्राफ्ट:** केवल व्यवस्थापकों के लिए दृश्यमान
- **लंबित समीक्षा:** अनुमोदन की प्रतीक्षा में
- **संग्रहीत:** छिपा हुआ लेकिन रखा हुआ

#### दृश्यता

सेट करें कि इस सामग्री को कौन देख सकता है:

```
Visibility: ☐ Public
           ☐ Registered Users Only
           ☐ Private (Admin Only)
```

#### प्रकाशन तिथि

सामग्री दृश्यमान होने पर सेट करें:

```
Publish Date: [Date Picker] [Time]
```

तुरंत प्रकाशित करने के लिए "अभी" छोड़ें।

#### टिप्पणियों की अनुमति दें

विज़िटर टिप्पणियाँ सक्षम या अक्षम करें:

```
Allow Comments: ☑ Yes
```

यदि सक्षम किया गया है, तो आगंतुक फीडबैक जोड़ सकते हैं।

### चरण 9: अपनी सामग्री सहेजें

एकाधिक सहेजें विकल्प:

```
[Publish Now]  [Save as Draft]  [Schedule]  [Preview]
```

- **अभी प्रकाशित करें:** तुरंत दृश्यमान बनाएं
- **ड्राफ्ट के रूप में सहेजें:** अभी निजी रखें
- **अनुसूची:** भविष्य की तारीख/समय पर प्रकाशित करें
- **पूर्वावलोकन:** सहेजने से पहले देखें कि यह कैसा दिखता है

अपनी पसंद पर क्लिक करें:

```
Click [Publish Now]
```

### चरण 10: अपना पृष्ठ सत्यापित करें

प्रकाशन के बाद, अपनी सामग्री सत्यापित करें:

1. अपनी वेबसाइट के होमपेज पर जाएं
2. अपने सामग्री क्षेत्र पर नेविगेट करें
3. अपना नव निर्मित पृष्ठ देखें
4. इसे देखने के लिए क्लिक करें
5. जाँच करें:
   - [ ] सामग्री सही ढंग से प्रदर्शित होती है
   - [ ] छवियाँ दिखाई देती हैं
   - [ ] फ़ॉर्मेटिंग अच्छी लगती है
   - [ ] लिंक काम करते हैं
   - [ ] शीर्षक और विवरण सही है

## सामग्री के लिए एसईओ सर्वोत्तम अभ्यास

खोज इंजनों के लिए अपनी सामग्री अनुकूलित करें:

### शीर्षक
- मुख्य कीवर्ड शामिल करें
- 50-60 अक्षर
- प्रति पृष्ठ अद्वितीय

### मेटा विवरण
- स्वाभाविक रूप से कीवर्ड शामिल करें
- 150-160 अक्षर
- सम्मोहक और सटीक

### सामग्री
- स्वाभाविक रूप से लिखें, कीवर्ड स्टफिंग से बचें
- शीर्षकों (h2, h3) का उचित प्रयोग करें
- अन्य पृष्ठों के आंतरिक लिंक शामिल करें
- सभी छवियों पर वैकल्पिक टेक्स्ट का उपयोग करें
- लेखों के लिए 300+ शब्दों का लक्ष्य रखें

### URL संरचना
- URL को छोटा और वर्णनात्मक रखें
- शब्दों को अलग करने के लिए हाइफ़न का उपयोग करें
- विशेष पात्रों से बचें
- उदाहरण: `/about-our-company`

## अपनी सामग्री का प्रबंधन करना

### मौजूदा पृष्ठ संपादित करें

1. **सामग्री > पेज** पर जाएँ
2. सूची में अपना पृष्ठ ढूंढें
3. **संपादित करें** या पृष्ठ शीर्षक पर क्लिक करें
4. परिवर्तन करें
5. **अपडेट** पर क्लिक करें

### पृष्ठ हटाएँ

1. **सामग्री > पेज** पर जाएँ
2. अपना पेज ढूंढें
3. **हटाएं** पर क्लिक करें
4. विलोपन की पुष्टि करें

### प्रकाशन स्थिति बदलें

1. **सामग्री > पेज** पर जाएँ
2. पेज ढूंढें, **संपादित करें** पर क्लिक करें
3. ड्रॉपडाउन में स्थिति बदलें
4. **अपडेट** पर क्लिक करें

## सामग्री निर्माण में समस्या निवारण

### सामग्री दिखाई नहीं दे रही है

**लक्षण:** प्रकाशित पृष्ठ वेबसाइट पर दिखाई नहीं दे रहा है

**समाधान:**
1. प्रकाशन स्थिति जांचें: "प्रकाशित" होना चाहिए
2. प्रकाशन तिथि जांचें: वर्तमान या पुरानी होनी चाहिए
3. दृश्यता जांचें: "सार्वजनिक" होनी चाहिए
4. कैश साफ़ करें: एडमिन > टूल > कैश साफ़ करें
5. अनुमतियाँ जांचें: उपयोगकर्ता समूह के पास पहुंच होनी चाहिए### फ़ॉर्मेटिंग काम नहीं कर रही

**लक्षण:** HTML टैग या फ़ॉर्मेटिंग टेक्स्ट के रूप में दिखाई देते हैं

**समाधान:**
1. सत्यापित करें कि मॉड्यूल सेटिंग्स में HTML सक्षम है
2. उचित HTML सिंटैक्स का प्रयोग करें
3. सभी टैग बंद करें: `<p>Text</p>`
4. केवल अनुमत टैग का ही प्रयोग करें
5. HTML इकाइयों का उपयोग करें: `&lt;` के लिए `<`, `&amp;` के लिए `&`

### छवियाँ प्रदर्शित नहीं हो रही हैं

**लक्षण:** छवियाँ टूटा हुआ आइकन दिखाती हैं

**समाधान:**
1. सत्यापित करें कि छवि URL सही है
2. जांचें कि छवि फ़ाइल मौजूद है
3. छवि पर उचित अनुमतियाँ सत्यापित करें
4. इसके बजाय छवि को XOOPS पर अपलोड करने का प्रयास करें
5. बाहरी अवरोधन की जाँच करें (CORS की आवश्यकता हो सकती है)

### कैरेक्टर एन्कोडिंग मुद्दे

**लक्षण:** विशेष वर्ण अस्पष्ट प्रतीत होते हैं

**समाधान:**
1. फ़ाइल को UTF-8 एन्कोडिंग के रूप में सहेजें
2. सुनिश्चित करें कि पृष्ठ वर्णसेट UTF-8 है
3. HTML शीर्ष में जोड़ें: `<meta charset="UTF-8">`
4. वर्ड से कॉपी-पेस्ट करने से बचें (सादे टेक्स्ट का उपयोग करें)

## सामग्री वर्कफ़्लो सर्वोत्तम अभ्यास

### अनुशंसित प्रक्रिया

1. **पहले संपादक में लिखें:** व्यवस्थापक सामग्री संपादक का उपयोग करें
2. **प्रकाशन से पहले पूर्वावलोकन:** पूर्वावलोकन बटन पर क्लिक करें
3. **मेटाडेटा जोड़ें:** पूरा शीर्षक, विवरण, टैग
4. **पहले ड्राफ्ट के रूप में सहेजें:** काम खोने से बचने के लिए ड्राफ्ट के रूप में सहेजें
5. **अंतिम समीक्षा:** प्रकाशन से पहले दोबारा पढ़ें
6. **प्रकाशित करें:** तैयार होने पर प्रकाशित करें पर क्लिक करें
7. **सत्यापित करें:** लाइव साइट पर जांचें
8. **यदि आवश्यक हो तो संपादित करें:** शीघ्रता से सुधार करें

### संस्करण नियंत्रण

हमेशा बैकअप रखें:

1. **बड़े बदलावों से पहले:** नए संस्करण या बैकअप के रूप में सहेजें
2. **पुरानी सामग्री संग्रहित करें:** अप्रकाशित संस्करण रखें
3. **अपने ड्राफ्ट की तारीख:** स्पष्ट नामकरण का प्रयोग करें: "पेज-ड्राफ्ट-2025-01-28"
