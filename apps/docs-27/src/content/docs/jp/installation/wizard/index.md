---

XOOPSインストールウィザードは、データベースを設定し、管理アカウントを作成し、サイトを最初の使用に向けて準備する15ステップのプロセスを案内します。

## 始める前に

- [XOOPSをサーバーにアップロード](/xoops-docs/2.7/installation/ftp-upload/)したか、ローカル環境をセットアップした
- [要件を確認](/xoops-docs/2.7/installation/requirements/)した
- データベース認証情報を準備している

## ウィザーステップ

| ステップ | スクリーン | 実行内容 |
|------|--------|--------------|
| 1 | [言語選択](./step-01/) | インストール言語を選択 |
| 2 | [ウェルカム](./step-02/) | ライセンス契約 |
| 3 | [設定チェック](./step-03/) | PHP/サーバー環境チェック |
| 4 | [パス設定](./step-04/) | ルートパスとURLを設定 |
| 5 | [データベース接続](./step-05/) | データベースホスト、ユーザー、パスワードを入力 |
| 6 | [データベース設定](./step-06/) | データベース名とテーブルプリフィックスを設定 |
| 7 | [設定保存](./step-07/) | mainfile.phpを書き込む |
| 8 | [テーブル作成](./step-08/) | データベーススキーマを作成 |
| 9 | [初期設定](./step-09/) | サイト名、管理者メール |
| 10 | [データ挿入](./step-10/) | デフォルトデータを設定 |
| 11 | [サイト設定](./step-11/) | URL、タイムゾーン、言語 |
| 12 | [テーマ選択](./step-12/) | デフォルトテーマを選択 |
| 13 | [モジュールインストール](./step-13/) | バンドルされたモジュールをインストール |
| 14 | [ウェルカム](./step-14/) | インストール完了メッセージ |
| 15 | [クリーンアップ](./step-15/) | installフォルダを削除 |

:::caution[セキュリティ]
ウィザードを完了した後、**`install/`フォルダを削除またはリネームしてください** — ステップ15はこの手順を案内します。アクセス可能なままにしておくことはセキュリティリスクです。
:::
