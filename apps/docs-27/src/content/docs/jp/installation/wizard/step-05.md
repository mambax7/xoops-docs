このページはXOOPSが使用するデータベースユーザーアカウントに関する情報を収集します。

要求された情報を入力し、問題を修正した後、「続行」ボタンを選択して進めます。

![XOOPSインストーラーデータベース接続](/xoops-docs/2.7/img/installation/installer-05.png)

## このステップで収集されるデータ

### サーバー接続

#### データベース

現在、_mysql_が唯一利用可能なオプションです。

#### サーバーホスト名

データベースサーバーのホスト名。不確かな場合は、_localhost_がほとんどの場合に機能します。

#### ユーザー名

XOOPSがデータベースサーバーに接続するために使用するデータベースユーザーアカウント。

#### パスワード

上記で入力したデータベースユーザーアカウントのパスワード。

#### 永続的な接続を使用する

デフォルトは「いいえ」です。不確かな場合は、これを空白のままにしておきます。
