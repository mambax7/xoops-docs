---
title: "XOOPSのアップグレード"
description: "既存のXOOPSインストールを新しいバージョンに安全にアップグレードする方法。"
---

このセクションでは、既存のXOOPSサイトをアップグレードする手順を説明します。アップグレードウィザードはコアファイルを更新しながら、データ、モジュール、テーマを保持します。

## アップグレードパス

| ステップ | ページ | 実行すること |
|------|------|------------|
| 1 | [準備](./step-01/) | バックアップ、変更ログの確認、新リリースのダウンロード |
| 2 | [プリフライトチェック](./preflight/) | アップグレード実行前の互換性確認 |
| 3 | [アップグレード実行](./step-02/) | アップグレードウィザードの実行 |
| 4 | [トラブルシューティング](./step-03/) | アップグレード時の一般的な問題の解決 |
| 5 | [アップグレード後](./step-04/) | 検証、モジュール更新、キャッシュクリア |

:::tip
アップグレード前に、必ず**データベース全体のバックアップとファイルバックアップ**を作成してください。バックアップなしではアップグレードは復帰不可能です。
:::
