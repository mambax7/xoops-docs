## 要件

| コンポーネント | 最小                    | 推奨          |
|------------|-------------------------|---------------|
| PHP        | 8.2                    | 8.4+         |
| MySQL      | 5.7                     | 8.0+          |
| MariaDB    | 10.4                    | 10.11+        |
| ウェブサーバー | Apache 2.4 / Nginx 1.20 | 最新の安定版 |

## ダウンロード

[GitHub Releases](https://github.com/XOOPS/XoopsCore27/releases) から最新リリースをダウンロードします。

```bash
# または直接クローン
git clone https://github.com/XOOPS/XoopsCore27.git mysite
cd mysite
```

## インストール手順

1. **ファイルをアップロード** してウェブサーバードキュメントルート (例: `public_html/`) にアップロードします。
2. **MySQL データベースを作成** し、そのユーザーに完全な権限を持つ。
3. **ブラウザを開き** ドメインに移動します — XOOPS インストーラーが自動的に起動します。
4. **5 ステップウィザードに従う** — パスを構成し、テーブルを作成し、管理者アカウントをセットアップします。
5. **`install/` フォルダを削除** プロンプトが表示されたときに。これはセキュリティに必須です。

## インストールを確認

セットアップ後、以下にアクセスしてください:

- **フロントページ:** `https://yourdomain.com/`
- **管理パネル:** `https://yourdomain.com/xoops_data/` *(インストール中に選択したパス)*
