## XOOPS CMSについて

XOOPS は、管理者が優れたコンテンツと多くの優れた機能を備えた動的ウェブサイトを簡単に作成できるコンテンツ管理システム(CMS)およびウェブポータルプログラムです。小規模から大規模の動的コミュニティウェブサイト、社内ポータル、企業ポータル、ウェブログなど、その他多くの用途に最適なツールです。PHP対応のウェブサーバー(例:Apache または Nginx)と MySQL互換データベースを持つホストにインストールできます。

**メインプロジェクトウェブサイト:** [https://xoops.org](https://xoops.org)

XOOPS は GNU一般公開ライセンス(GPL)の条件で公開されており、自由に使用および変更できます。GPL のライセンス条件に従う限り、自由に再配布できます。

### XOOPS の意味

XOOPS は **eXtensible Object Oriented Portal System**(拡張可能なオブジェクト指向ポータルシステム)の頭字語です。当初はポータルシステムとして開始されましたが、現在は小規模な個人向けサイトから大規模エンタープライズインストールまでを対応する完全な機能を備えたコンテンツ管理システムです。

### 発音

XOOPS の標準的な発音は英語の規則に従い、`[zoo'ps]` と発音されます。

### 主な機能

**データベース駆動**

XOOPS は、ウェブベースのコンテンツ管理システムの実行に必要なデータを保存するためにリレーショナルデータベース(MySQL 5.7.8以降)を使用します。

**完全にモジュール化**

モジュールは XOOPS モジュール管理システムを通じてクリックするだけでインストール、アンインストール、有効化、または無効化できます。

**パーソナライゼーション**

登録ユーザーはプロフィールを編集、サイトテーマを選択、カスタムアバターをアップロードなどができます。

**ユーザー管理**

XOOPS は様々な条件でユーザーを検索、メール、テンプレートベースのプライベートメッセージングシステムを提供します。

**世界規模のサポート**

XOOPS は世界中のボランティアチームによって作成および保守されています。XOOPS コミュニティは複数の言語でサポートサイトを持っています。

**マルチバイト言語サポート**

XOOPS は日本語、簡体字中国語、繁体字中国語、韓国語、およびアラビア語(2.7.0で RTL サポートが追加)を含むマルチバイト言語を完全にサポートしています。

**多目的なグループパーミッション**

強力でユーザーフレンドリーなパーミッションシステムにより、管理者はグループごとにパーミッションを設定できます。

**テーマベースのスキン可能インターフェース**

テーマシステムにより、管理者とユーザーの両方がサイト全体の外観を変更できます。2.7.0配布には、最新の Bootstrap-5 および Tailwind ベースのテーマが含まれています。
