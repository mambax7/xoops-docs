---
title: "Criação de Tabela"
---

Esta página exibe os resultados da criação das tabelas XOOPS no banco de dados.

Após revisar e corrigir quaisquer problemas, selecione o botão "Continuar" para prosseguir.

## No Sucesso

A seção _Criação de tabelas do banco de dados_ mostra as tabelas do banco de dados que foram criadas.

![Criação de Tabela do Instalador XOOPS](/xoops-docs/2.7/img/installation/installer-08.png)

## Erros

Qualquer erro indicar problemas com permissão relacionados à conta de usuário do banco de dados MySQL e/ou banco de dados MySQL inserido em etapas anteriores. Você pode retornar a essas seções usando as opções Conexão do banco de dados e Configuração do banco de dados no menu.
