| Software | Descrição |
| --- | --- |
| 7Zip www.7zip.org | Programa de código aberto para zip/descompactar arquivos. |
| WinSCP www.WinSCP.net | WinSCP é um cliente SFTP e cliente FTP de código aberto para Windows. Sua função principal é a transferência segura de arquivos entre um computador local e remoto. |
| WAMP Server www.wampserver.com | WampServer é um ambiente de desenvolvimento web Windows. Permite criar aplicações web com Apache, PHP e banco de dados MySQL. Também vem com PHPMyAdmin para gerenciar facilmente seus bancos de dados. WampServer instala automaticamente (instalador) e seu uso é muito intuitivo. Você poderá ajustar seu servidor sem nem tocar nos arquivos de configuração. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP é uma distribuição Apache fácil de instalar contendo MySQL, PHP e Perl. XAMPP é realmente muito fácil de instalar e usar - basta fazer download, extrair e iniciar. |
| KeyPass www.keepass.info | KeePass é um gerenciador de senha de código aberto gratuito, que o ajuda a gerenciar suas senhas de forma segura. Você pode colocar todas as suas senhas em um banco de dados, que é bloqueado com uma chave mestra ou arquivo de chave. Portanto, você só precisa lembrar uma única senha mestra ou selecionar o arquivo de chave para desbloquear todo o banco de dados. Os bancos de dados são criptografados usando os melhores e mais seguros algoritmos de criptografia conhecidos (AES e Twofish). |
| Package Web Tools [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === Aplicativos USB ==== - 7zip, Filezilla, Kompozer, DataBase Browser, Opera, PicPick, Winmerge, Gimp, Notepad++, OikoCSSeditor, SubCommander, ToDoList, Toucan, Xenu, TeamViewer, Pidgin, FastImageOptimizer, VirtualWin e MiniServer (com último Apache - MySQL - PHP e Adminer para administrar o banco de dados) |
| Package Web Browsers [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === Aplicativos USB ==== - Firefox + extensões webdevelpper e XooFoo (versão estável e desenvolvimento), IETester 0.4.5, Opera (versão estável e desenvolvimento), OperaMobile Emulator, Safari, Google Chrome + extensão webdevelpper (versão estável e desenvolvimento), Maxthon (versão estável e desenvolvimento), SeaMonkey, Avant browser, Flock e Elinks |
