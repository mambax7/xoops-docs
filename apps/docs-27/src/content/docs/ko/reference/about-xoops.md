## XOOPS CMS 정보

XOOPS는 관리자가 훌륭한 콘텐츠와 많은 뛰어난 기능을 갖춘 동적 웹사이트를 쉽게 만들 수 있는 콘텐츠 관리 시스템(CMS)이자 웹 포털 프로그램입니다. 이는 크고 작은 동적 커뮤니티 웹사이트, 회사 내 포털, 기업 포털, 웹로그 등을 개발하는 데 이상적인 도구입니다. PHP 지원 웹 서버(예: Apache 또는 Nginx)와 MySQL 호환 데이터베이스가 있는 모든 호스트에 설치할 수 있습니다.

**주요 프로젝트 웹사이트:** [https://xoops.org](https://xoops.org)

XOOPS는 GNU 일반 공중 사용 허가서(GPL)의 조건에 따라 출시되며 무료로 사용하고 수정할 수 있습니다. GPL 배포 조건을 준수하는 한 재배포할 수 있습니다.

### XOOPS의 약자

XOOPS는 **eXtensible Object Oriented Portal System**의 약어입니다. 포털 시스템으로 시작했지만 이제는 대규모 기업 설치를 통해 소규모 개인 사이트를 서비스할 수 있는 완전한 기능을 갖춘 컨텐츠 관리 시스템입니다.

### 발음

XOOPS의 표준 발음은 영어 규칙을 따르며 `[zoo'ps]`으로 발음됩니다.

### 주요 기능

**데이터베이스 기반**

XOOPS는 관계형 데이터베이스(MySQL, 5.7.8 이상)를 사용하여 웹 기반 콘텐츠 관리 시스템을 실행하는 데 필요한 데이터를 저장합니다.

**완전히 모듈화됨**

XOOPS 모듈 관리 시스템을 통해 클릭 한 번으로 모듈을 설치, 제거, 활성화 또는 비활성화할 수 있습니다.

**개인화**

등록된 사용자는 자신의 프로필을 편집하고, 사이트 테마를 선택하고, 사용자 정의 아바타를 업로드하는 등의 작업을 수행할 수 있습니다.

**사용자 관리**

XOOPS는 다양한 기준, 이메일 및 템플릿 기반 개인 메시징 시스템을 통해 사용자 검색을 제공합니다.

**전 세계 지원**

XOOPS는 전 세계의 자원봉사자 팀에 의해 만들어지고 유지관리됩니다. XOOPS 커뮤니티에는 여러 언어로 지원 사이트가 있습니다.

**멀티바이트 언어 지원**

XOOPS는 일본어, 중국어 간체 및 번체, 한국어, 아랍어(2.7.0에 RTL 지원 추가)를 포함한 멀티바이트 언어를 완벽하게 지원합니다.

**다양한 그룹 권한**

강력하고 사용자 친화적인 권한 시스템을 통해 관리자는 그룹별로 권한을 설정할 수 있습니다.

**테마 기반 스킨 가능 인터페이스**

테마 시스템을 사용하면 관리자와 사용자 모두 전체 사이트의 모양을 변경할 수 있습니다. 2.7.0 배포판에는 최신 Bootstrap-5 및 Tailwind 기반 테마가 포함되어 있습니다.
