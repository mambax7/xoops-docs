---
title: "테이블 생성"
---

이 페이지에는 데이터베이스에 XOOPS 테이블을 생성한 결과가 표시됩니다.

문제를 검토하고 수정한 후 "계속" 버튼을 선택하여 계속 진행하세요.

## 성공 시

_데이터베이스 테이블 생성_ 섹션에는 생성된 데이터베이스 테이블이 표시됩니다.

![XOOPS Installer Table Creation](/xoops-docs/2.7/img/installation/installer-08.png)

## 오류

모든 오류는 이전 단계에서 입력한 MySQL 데이터베이스 사용자 계정 및/또는 MySQL 데이터베이스와 관련된 권한 문제를 나타냅니다. 메뉴의 데이터베이스 연결 및 데이터베이스 구성 옵션을 사용하여 해당 섹션으로 돌아갈 수 있습니다.
