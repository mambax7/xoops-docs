이 페이지에서는 새 사이트의 초기 테마에 대한 선택 사항을 수집합니다. 테마는 사이트의 시각적 특성을 제어합니다. 원하는 경우 언제든지 테마를 변경할 수 있습니다.

요청된 정보를 입력하고 문제를 수정한 후 "계속" 버튼을 선택하여 계속 진행하세요.

![XOOPS Installer Select Theme](/xoops-docs/2.7/img/installation/installer-12.png)

## 이 단계에서 수집된 데이터

### 기본 테마를 선택하세요

#### 기본 테마

목록에서 사이트의 초기 테마를 선택하세요. 각 테마를 선택하면 해당 테마의 스크린샷이 표시됩니다.

XOOPS 2.7.0은 기본적으로 여러 가지 프런트엔드 테마를 제공합니다.

* **기본값** — 사용자 정의의 시작점으로 유용한 최소 기준 테마입니다.
* **xbootstrap5** — 대부분의 새로운 사이트에 권장되는 Bootstrap 5 테마입니다.
* **xswatch5** — Bootswatch 5개 변형; Bootstrap 5 위에 스타일이 지정된 하위 테마 세트.
* **xtailwind** — Tailwind CSS 기반 테마 + 35개 팔레트와 Alpine.js 상호 작용 기능을 갖춘 DaisyUI
* **xtailwind2** — 선별된 팔레트와 세련된 모듈 스타일을 갖춘 예술 감독 형제 테마

관리 영역에서 언제든지 테마를 변경할 수 있으며, `themes/` 디렉토리에 추가 테마를 업로드할 수 있습니다.
