이 페이지는 XOOPS가 사용할 데이터베이스에 대한 정보를 수집합니다.

요청된 정보를 입력하고 문제를 수정한 후 "계속" 버튼을 선택하여 계속 진행하세요.

![XOOPS Installer Database Configuration](/xoops-docs/2.7/img/installation/installer-06.png)

## 이 단계에서 수집된 데이터

### 데이터베이스

#### 데이터베이스 이름

XOOPS가 사용해야 하는 호스트의 데이터베이스 이름입니다. 이전 단계에서 입력한 데이터베이스 사용자는 이 데이터베이스에 대한 모든 권한을 가지고 있어야 합니다. 설치 프로그램은 이 데이터베이스가 존재하지 않는 경우 생성을 시도합니다.

#### 테이블 접두사

이 접두사는 XOOPS에서 생성된 모든 새 테이블의 이름에 추가됩니다. 이는 데이터베이스가 다른 애플리케이션과 공유되는 경우 이름 충돌을 방지하는 데 도움이 됩니다. 또한 고유한 접두사는 테이블 이름을 추측하기 어렵게 만들어 보안상의 이점을 제공합니다. 확실하지 않다면 기본값을 유지하세요.

#### 데이터베이스 문자 집합

설치 프로그램의 기본값은 이모티콘과 보조 문자를 포함한 전체 유니코드 범위를 지원하는 `utf8mb4`입니다. 여기에서 다른 문자 집합을 선택할 수 있지만 `utf8mb4`은 거의 모든 언어 및 로케일에 권장되며 변경할 특별한 이유가 없는 한 그대로 유지해야 합니다.

#### 데이터베이스 대조

데이터 정렬 필드는 기본적으로 비어 있습니다. 비어 있으면 MySQL은 위에서 선택한 문자 집합에 대한 기본 데이터 정렬을 적용합니다(`utf8mb4`의 경우 이는 MySQL 버전에 따라 일반적으로 `utf8mb4_general_ci` 또는 `utf8mb4_0900_ai_ci`입니다). 예를 들어 기존 데이터베이스와 일치시키기 위해 특정 데이터 정렬이 필요한 경우 여기에서 선택하세요. 그렇지 않으면 공백으로 두는 것이 좋습니다.
