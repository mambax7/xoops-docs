이 페이지는 XOOPS가 사용할 데이터베이스 사용자 계정에 대한 정보를 수집합니다.

요청된 정보를 입력하고 문제를 수정한 후 "계속" 버튼을 선택하여 계속 진행하세요.

![XOOPS Installer Database Connection](/xoops-docs/2.7/img/installation/installer-05.png)

## 이 단계에서 수집된 데이터

### 서버 연결

#### 데이터베이스

현재 _mysql_이 유일하게 사용 가능한 옵션입니다.

#### 서버 호스트 이름

데이터베이스 서버의 호스트 이름입니다. 확실하지 않은 경우 대부분의 경우 _localhost_가 작동합니다.

#### 사용자 이름

XOOPS가 데이터베이스 서버에 연결하는 데 사용해야 하는 데이터베이스 사용자 계정입니다.

#### 비밀번호

위에서 입력한 데이터베이스 사용자 계정의 비밀번호입니다.

#### 영구 연결 사용

기본값은 '아니요'입니다. 확실하지 않은 경우 공백으로 남겨두세요.
