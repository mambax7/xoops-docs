XOOPS 설치 마법사는 데이터베이스를 구성하고, 관리자 계정을 생성하고, 처음 사용할 사이트를 준비하는 15단계 프로세스를 안내합니다.

## 시작하기 전에

- [서버에 XOOPS를 업로드](/xoops-docs/2.7/installation/ftp-upload/)했거나 로컬 환경을 설정했습니다.
- [요구사항을 확인]하셨습니다(/xoops-docs/2.7/installation/requirements/)
- 데이터베이스 자격 증명이 준비되었습니다.

## 마법사 단계

| 단계 | 화면 | 무슨 일이 일어나는가 |
|------|--------|--------------|
| 1 | [언어 선택](./step-01/) | 설치 언어 선택 |
| 2 | [환영합니다](./step-02/) | 라이센스 계약 |
| 3 | [구성 확인](./step-03/) | PHP/서버 환경 점검 |
| 4 | [경로설정](./step-04/) | 루트 경로 및 URL 설정 |
| 5 | [데이터베이스 연결](./step-05/) | 데이터베이스 호스트, 사용자, 비밀번호 입력 |
| 6 | [데이터베이스 구성](./step-06/) | 데이터베이스 이름 및 테이블 접두사 설정 |
| 7 | [구성 저장](./step-07/) | mainfile.php 작성 |
| 8 | [테이블 생성](./step-08/) | 데이터베이스 스키마 생성 |
| 9 | [초기 설정](./step-09/) | 사이트 이름, 관리자 이메일 |
| 10 | [데이터 삽입](./step-10/) | 기본 데이터 채우기 |
| 11 | [사이트 구성](./step-11/) | URL, 시간대, 언어 |
| 12 | [테마 선택](./step-12/) | 기본 테마 선택 |
| 13 | [모듈 설치](./step-13/) | 번들 모듈 설치 |
| 14 | [환영합니다](./step-14/) | 설치 완료 메시지 |
| 15 | [정리](./step-15/) | 설치 폴더 제거 |

:::주의사항[보안]
마법사를 완료한 후 **`install/` 폴더를 삭제하거나 이름을 바꿉니다**. 15단계에서 이를 안내합니다. 접근 가능한 상태로 놔두는 것은 보안상 위험합니다.
:::
