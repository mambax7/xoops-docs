이 페이지에서는 기본 관리자(AKA 웹마스터) 계정을 생성하는 데 사용되는 정보를 수집합니다.

요청된 정보를 입력하고 문제를 수정한 후 "계속" 버튼을 선택하여 계속 진행하세요.

![XOOPS Installer Initial Settings](/xoops-docs/2.7/img/installation/installer-09.png)

## 이 단계에서 수집된 데이터

### 관리자 계정

#### 관리자 로그인

이는 관리자가 XOOPS 시스템에 로그인하는 데 사용하는 사용자 이름입니다.

#### 관리자 이메일

이는 XOOPS 시스템이 관리자와 통신하는 데 사용하는 이메일 주소입니다. (참고로 이는 비밀번호 복구/재설정을 위해 필요합니다.)

#### 관리자 비밀번호

관리자 계정에 사용하는 비밀번호입니다.

#### 비밀번호 확인

이 필드를 사용하여 관리자 비밀번호를 확인하세요. _관리자 비밀번호_ 필드와 일치해야 합니다.

### 비밀번호 강도

이 섹션에는 _관리자 비밀번호_의 상대적 강도가 표시됩니다.

#### 비밀번호 생성기

이 섹션을 사용하여 계정에 대한 임의의 비밀번호를 생성하고 사용할 수 있습니다. 임의의 비밀번호를 표시하려면 _생성_을 선택하세요. 생성된 비밀번호를 _관리자 비밀번호_에 복사하려면 _복사_를 선택하세요.
