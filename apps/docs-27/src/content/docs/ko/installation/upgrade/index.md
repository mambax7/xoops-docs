---

이 섹션에서는 기존 XOOPS 사이트를 업그레이드하는 과정을 안내합니다. 업그레이드 마법사는 핵심 파일을 업데이트하는 동안 데이터, 모듈 및 테마를 유지합니다.

## 업그레이드 경로

| 단계 | 페이지 | 해야 할 일 |
|------|------|------------|
| 1 | [준비사항](./step-01/) | 백업, 변경 로그 검토, 새 릴리스 다운로드 |
| 2 | [비행 전 확인](./preflight/) | 업그레이드를 실행하기 전에 호환성 확인 |
| 3 | [업그레이드 실행](./step-02/) | 업그레이드 마법사 실행 |
| 4 | [문제 해결](./step-03/) | 일반적인 업그레이드 문제 해결 |
| 5 | [업그레이드 이후](./step-04/) | 확인, 모듈 업데이트, 캐시 지우기 |

:::팁
업그레이드하기 전에 항상 **전체 데이터베이스 백업 및 파일 백업**을 수행하세요. 백업 없이는 업그레이드를 되돌릴 수 없습니다.
:::
