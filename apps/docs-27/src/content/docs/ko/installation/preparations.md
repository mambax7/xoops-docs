title: "phpinfo"
---

이 단계는 선택 사항이지만 몇 시간의 좌절감을 쉽게 줄일 수 있습니다.

호스팅 시스템의 사전 설치 테스트로 매우 작지만 유용한 PHP 스크립트가 로컬로 생성되어 대상 시스템에 업로드됩니다.

PHP 스크립트는 단 한 줄입니다:

```php
<?php phpinfo();
```

텍스트 편집기를 사용하여 이 한 줄로 _info.php_라는 파일을 만듭니다.

다음으로 이 파일을 웹 루트에 업로드하세요.

![Filezilla info.php Upload](/xoops-docs/2.7/img/installation/filezilla-01-info.png)

브라우저에서 스크립트를 열어 스크립트에 액세스하세요(예: `http://example.com/info.php` 액세스). 모든 것이 올바르게 작동하면 다음과 같은 페이지가 표시됩니다.

![phpinfo() Example](/xoops-docs/2.7/img/installation/php-info.png)

참고: 일부 호스팅 서비스는 보안 조치로 _phpinfo()_ 기능을 비활성화할 수 있습니다. 그러한 경우 일반적으로 해당 내용에 대한 메시지를 받게 됩니다.

스크립트 출력은 문제 해결에 유용할 수 있으므로 복사본을 저장해 두는 것이 좋습니다.

테스트가 작동하면 설치를 진행하는 것이 좋습니다. _info.php_ 스크립트를 삭제하고 설치를 진행해야 합니다.

테스트가 실패하면 이유를 조사해 보세요! 이 간단한 테스트의 작동을 방해하는 문제가 무엇이든 **실제 설치가 작동하지 못하게 됩니다**.
