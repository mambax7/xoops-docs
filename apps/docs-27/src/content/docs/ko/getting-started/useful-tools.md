| 소프트웨어 | 설명 |
| --- | --- |
| 7Zip www.7zip.org | 파일을 압축/압축 해제하는 오픈 소스 프로그램입니다. |
| WinSCP www.WinSCP.net | WinSCP는 Windows용 오픈 소스 SFTP 클라이언트 및 FTP 클라이언트입니다. 주요 기능은 로컬 컴퓨터와 원격 컴퓨터 간의 안전한 파일 전송입니다. |
| WAMP 서버 www.wampserver.com | WampServer는 Windows 웹 개발 환경입니다. Apache, PHP 및 MySQL 데이터베이스를 사용하여 웹 애플리케이션을 만들 수 있습니다. 또한 데이터베이스를 쉽게 관리할 수 있도록 PHPMyAdmin이 함께 제공됩니다.  WampServer는 자동으로 설치되며(설치 프로그램) 사용법이 매우 직관적입니다. 설정 파일을 건드리지 않고도 서버를 조정할 수 있습니다. |
| XAMPP [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP는 MySQL, PHP 및 Perl이 포함된 Apache 배포판을 설치하기 쉽습니다. XAMPP는 설치 및 사용이 매우 쉽습니다. 다운로드하고 추출하고 시작하기만 하면 됩니다. |
| 키패스 www.keepass.info | KeePass는 안전한 방법으로 비밀번호를 관리하는 데 도움이 되는 무료 오픈 소스 비밀번호 관리자입니다. 하나의 마스터 키나 키 파일로 잠긴 하나의 데이터베이스에 모든 비밀번호를 넣을 수 있습니다. 따라서 하나의 마스터 비밀번호만 기억하거나 키 파일을 선택하면 전체 데이터베이스의 잠금을 해제할 수 있습니다. 데이터베이스는 현재 알려진 가장 우수하고 안전한 암호화 알고리즘(AES 및 Twofish)을 사용하여 암호화됩니다. |
| 패키지 웹 도구 [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === USB 앱 ==== - 7zip, Filezilla, Kompozer, DataBase Browser, Opera, PicPick, Winmerge, Gimp, Notepad++, OikoCSSeditor, SubCommander, ToDoList, Toucan, Xenu, TeamViewer, Pidgin, FastImageOptimizer, VirtualWin 및 MiniServer(마지막 Apache - MySQL - PHP 및 데이터베이스 관리를 위한 관리자 포함) |
| 패키지 웹 브라우저 [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === USB 앱 ==== - Firefox + webdevelopper 및 XooFoo 확장(안정 및 개발 버전), IETester 0.4.5, Opera(안정 및 개발 버전), OperaMobile Emulator, Safari, Google Chrome + webdevelopper 확장(안정 및 개발 버전), Maxthon(안정 및 개발 버전), SeaMonkey, Avant 브라우저, Flock 및 Elinks |
| 균일 서버 [http://uniformserver.com](http://uniformserver.com) | "XAMPP, WAMP 또는 EASYPHP 대신에 균일 서버를 사용하는 이유는 무엇입니까? 1. 보안 2. 경량(xampp 51MB, XAMPlite 28MB. 균일 서버 9.95MB). 3. 사용하기 더 쉽습니다. 4. 예를 들어 XAMPP에는 특히 Xoops 커뮤니티에 불필요한 기능이 많이 있습니다. 균일 서버의 작은 크기에도 불구하고 필요한 모든 것이 있습니다./ /마지막으로 균일 서버를 설치하면 도움이 됩니다. 버전: - MySQL 데이터베이스 - MySQL 데이터베이스를 관리하는 phpMyAdmin - PHP(XAMPP를 다운로드할 때 얻는 것보다 100개 이상의 버그 수정 포함) - 웹사이트 속도 향상
| XooFoo [http://goo.gl/LGx3V](http://goo.gl/LGx3V)의 유니폼 서버 | XOOPS에 최적화된 유니폼 서버 |
| XooFoo [http://goo.gl/Ke0LW](http://goo.gl/Ke0LW)의 미니 서버(5MB) | 정적 HTML 및 HTML 페이지 외에도 동적 웹 페이지를 허용하는 PHP 기능이 추가된 기본 휴대용 프로덕션 서버입니다. 동일한 PC 또는 USB 디스크에서 여러 서버를 실행할 수 있도록 설계되었으므로 자동으로 생성된 별도의 가상 드라이브가 필요합니다.  MySQL 지원이 활성화되어 이 서버의 기능이 더욱 향상됩니다. |
