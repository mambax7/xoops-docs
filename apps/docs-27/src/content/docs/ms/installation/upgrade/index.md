Bahagian ini memandu anda menaik taraf tapak XOOPS sedia ada. Wizard naik taraf mengekalkan data, modul dan tema anda semasa mengemas kini fail teras.## Naik taraf laluan| Langkah | Halaman | Apa yang perlu dilakukan |
|------|------|------------|
| 1 | [Persediaan](./step-01/) | Sandaran, semak log perubahan, muat turun keluaran baharu |
| 2 | [Semak Pra Penerbangan](./preflight/) | Sahkan keserasian sebelum menjalankan peningkatan |
| 3 | [Menjalankan Peningkatan](./step-02/) | Laksanakan wizard naik taraf |
| 4 | [Menyelesaikan masalah](./step-03/) | Selesaikan isu naik taraf biasa |
| 5 | [Selepas Naik Taraf](./step-04/) | Sahkan, kemas kini modul, kosongkan cache |:::petua
Sentiasa buat **sandaran pangkalan data penuh dan sandaran fail** sebelum menaik taraf. Peningkatan tidak dapat dipulihkan tanpa sandaran.
:::
