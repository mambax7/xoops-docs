title: "Menjalankan Peningkatan"
---
Sebelum menjalankan penaik taraf utama, pastikan anda telah menyelesaikan [Pemeriksaan Pralampu](preflight.md). UI peningkatan memerlukan prapenerbangan dijalankan sekurang-kurangnya sekali dan akan mengarahkan anda ke sana jika anda belum melakukannya.Lancarkan peningkatan dengan menghalakan penyemak imbas anda ke direktori _upgrade_ tapak anda:
```
text
http://example.com/upgrade/
```
Ini sepatutnya menunjukkan halaman seperti ini:![Permulaan Naik Taraf XOOPS](/XOOPS-docs/2.7/img/installation/upgrade-01.png)Pilih butang "Teruskan" untuk meneruskan.Setiap "Teruskan" maju melalui patch lain. Teruskan sehingga semua tampalan digunakan, dan halaman Kemas Kini Modul Sistem dibentangkan.![XOOPS Naik Taraf Gunaan Tampalan](/XOOPS-docs/2.7/img/installation/upgrade-05-applied.png)## Perkara 2.5.11 → 2.7.0 Naik Taraf TerpakaiApabila menaik taraf daripada XOOPS 2.5.11 kepada 2.7.0, peningkatan menggunakan tampung berikut. Setiap satu dibentangkan sebagai langkah berasingan dalam wizard supaya anda boleh mengesahkan perkara yang sedang diubah:1. **Alih keluar PHPMailer berpakatan usang.** Salinan gabungan PHPMailer di dalam modul Pelindung dipadamkan. PHPMailer kini dibekalkan melalui Komposer dalam `xoops_lib/vendor/`.
2. **Alih keluar folder HTMLPurifier usang.** Begitu juga, folder HTMLPurifier lama di dalam modul Protector dipadamkan. HTMLPurifier kini dibekalkan melalui Composer.
3. **Buat jadual `tokens`.** Jadual `tokens` baharu ditambahkan untuk storan token berskop generik. Jadual mempunyai lajur untuk id token, id pengguna, skop, cincang dan cap masa issued/expires/used, dan digunakan oleh ciri berasaskan token dalam XOOPS 2.7.0.
4. **Meluaskan `bannerclient.passwd`.** Lajur `bannerclient.passwd` diluaskan kepada `VARCHAR(255)` supaya ia boleh menyimpan cincang kata laluan moden (bcrypt, argon2) dan bukannya lajur sempit warisan.
5. **Tambah pilihan kuki sesi.** Dua pilihan baharu dimasukkan: `session_cookie_samesite` (untuk atribut kuki SameSite) dan `session_cookie_secure` (untuk memaksa kuki HTTPS sahaja). Lihat [Selepas Naik Taraf](ustep-04.md) untuk cara menyemak ini selepas peningkatan selesai.Tiada satu pun daripada langkah ini menyentuh data kandungan anda. Pengguna, siaran, imej dan data modul anda kekal tidak disentuh.## Memilih BahasaPengedaran XOOPS utama disertakan dengan sokongan bahasa Inggeris. Sokongan untuk tempat tambahan dibekalkan oleh [XOOPS Tapak sokongan setempat](https://XOOPS.org/modules/xoopspartners/). Sokongan ini boleh datang dalam bentuk pengedaran tersuai, atau fail tambahan untuk ditambahkan pada pengedaran utama.Terjemahan XOOPS dikekalkan pada [transifex](https://www.transifex.com/XOOPS/public/)Jika Penaik taraf XOOPS anda mempunyai sokongan bahasa tambahan, anda boleh menukar bahasa dengan memilih ikon bahasa dalam menu atas dan memilih bahasa lain.![XOOPS Naik Taraf Bahasa](/XOOPS-docs/2.7/img/installation/upgrade-02-change-language.png)
