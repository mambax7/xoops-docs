title: "Tetapan Laluan"
---
Halaman ini mengumpul maklumat tentang sistem fail dan alamat web yang XOOPS perlu beroperasi. Pemasang akan cuba meneka maklumat ini berdasarkan URL yang anda gunakan untuk mengaksesnya, dan lokasi skrip PHP yang sedang berjalan.Selepas menyemak dan membetulkan sebarang isu, pilih butang "Teruskan" untuk meneruskan.![XOOPS Tetapan Laluan Pemasang](/XOOPS-docs/2.7/img/installation/installer-04-02.png)## Data Dikumpul dalam Langkah Ini### XOOPS Laluan fizikal#### XOOPS mendokumenkan laluan fizikal akarLaluan fizikal ke direktori dokumen XOOPS (disediakan) WITHOUT slash mengekor#### XOOPS direktori fail dataLaluan fizikal ke direktori fail data XOOPS (boleh ditulis) WITHOUT slash mengekor. Cari folder daripada akar dokumen XOOPS untuk memastikannya selamat.#### XOOPS direktori perpustakaanLaluan fizikal ke direktori perpustakaan XOOPS WITHOUT slash mengekor. Cari folder daripada akar dokumen XOOPS untuk menjadikannya selamat.### Lokasi web#### Lokasi tapak web (URL)URL utama yang akan digunakan untuk mengakses pemasangan XOOPS anda. XOOPS akan menggunakan ini untuk membina semua URL di dalam XOOPS. Jika anda mahukan tapak _https_, pastikan anda menyatakannya. Jika anda want/don't mahukan _www._ terkemuka pada domain tapak anda, pastikan anda menyatakannya seperti yang anda mahukan.Ini akan lalai kepada URL yang digunakan untuk melancarkan pemasang.#### Domain Kuki untuk Tapak WebDomain untuk menetapkan kuki. Mungkin kosong, hos penuh daripada URL (www.example.com) atau domain berdaftar tanpa subdomain (example.com) untuk dikongsi merentas subdomain (www.example.com dan blog.example.com.)## RalatJika mana-mana laluan yang dimasukkan tidak dapat ditemui, ralat akan dipaparkan. Betulkan sebarang masalah sebelum meneruskan.![XOOPS Ralat Tetapan Laluan Pemasang](/XOOPS-docs/2.7/img/installation/installer-04-01.png)## BantuanAnda boleh melihat penerangan lanjutan semasa proses pemasangan. Memilih ikon "pemelihara hayat" di penjuru atas halaman menogol paparan perihalan lanjutan.![XOOPS Bantuan Tetapan Laluan Pemasang](/XOOPS-docs/2.7/img/installation/installer-04-03.png)
