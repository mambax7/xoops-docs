title: "Konfigurasi Tapak"
---
Halaman ini mengumpulkan maklumat yang digunakan untuk menetapkan keadaan awal tapak XOOPS baharu anda. Ini termasuk perkara seperti tajuk dan nilai teg meta HTML.Ini hanyalah nilai awal untuk tapak anda. Tetapan boleh ditukar pada bila-bila masa dalam pilihan pentadbiran tapak XOOPS baharu anda.Anda mungkin mahu menggunakan tetapan awal ini untuk mengehadkan perkara yang akan dilihat secara terbuka sehingga tapak anda disediakan sepenuhnya.Selepas memasukkan maklumat yang diminta dan membetulkan sebarang isu, pilih butang "Teruskan" untuk meneruskan.![XOOPS Konfigurasi Tapak Pemasang](/XOOPS-docs/2.7/img/installation/installer-11.png)## Data Dikumpul dalam Langkah Ini### Tetapan Umum#### Nama tapakNama, atau tajuk, untuk tapak anda.#### Slogan untuk tapak andaSlogan, atau sub-tajuk, untuk tapak anda.#### Mod nyahpepijatIni membolehkan anda memasuki tapak baharu anda dengan maklumat penyahpepijatan dipaparkan. Walaupun tapak web yang sedang berjalan sepatutnya ini dimatikan, ia boleh menjadi sangat berguna jika terdapat sebarang masalah dengan pemasangan. _Dayakan nyahpepijat (mod dalam talian)_ ialah nilai yang paling biasa digunakan.#### Matikan tapak anda?Pilih _ya_ untuk mematikan tapak anda. In this state, immediately after install only the newly created admin account will be able to access the site. Ini akan membolehkan anda mengkonfigurasi tapak anda mengikut kepuasan anda sebelum membenarkan akses awam.### Tag Meta dan PengakiTeg meta HTML mengandungi data yang menerangkan tapak. Sesetengah modul mungkin menetapkan teg meta khusus halaman. Nilai teg yang dimasukkan di sini akan digunakan jika tidak ditindih oleh nilai khusus halaman.#### Kata Kunci MetaSenarai kata kunci yang berkaitan dengan kandungan tapak. Ini mungkin memainkan peranan dalam SEO, tetapi gunakan dengan berhati-hati. "Spam" dengan kata kunci boleh memberi kesan negatif pada peletakan enjin carian.#### Penerangan MetaPenerangan ringkas tentang tapak anda.#### Pengarang MetaPengarang lalai tapak dalam bentuk nama, alamat e-mel, nama syarikat atau URL.#### Hak Cipta MetaKenyataan hak cipta untuk tapak anda.### Tetapan Maklumat Pengguna#### Benarkan pendaftaran pengguna baharu?Pilih ya untuk menerima pendaftaran pengguna baharu, atau tidak jika pendaftaran adalah tertutup.
