Halaman ini mengumpul maklumat tentang pangkalan data yang akan digunakan oleh XOOPS.Selepas memasukkan maklumat yang diminta dan membetulkan sebarang isu, pilih butang "Teruskan" untuk meneruskan.![XOOPS Konfigurasi Pangkalan Data Pemasang](/XOOPS-docs/2.7/img/installation/installer-06.png)## Data Dikumpul dalam Langkah Ini### Pangkalan data#### Nama pangkalan dataNama pangkalan data pada hos yang harus digunakan oleh XOOPS. Pengguna pangkalan data yang dimasukkan dalam langkah sebelumnya harus mempunyai semua keistimewaan pada pangkalan data ini. Pemasang akan cuba mencipta pangkalan data ini jika tidak wujud.#### Awalan jadualAwalan ini akan ditambahkan pada nama semua jadual baharu yang dibuat oleh XOOPS. Ini membantu mengelakkan konflik nama jika pangkalan data dikongsi dengan aplikasi lain. Awalan yang unik juga menjadikan nama jadual lebih sukar untuk diteka, yang mempunyai faedah keselamatan. Jika anda tidak pasti, hanya kekalkan lalai#### Set aksara pangkalan dataPemasang lalai kepada `utf8mb4`, yang menyokong julat Unicode penuh termasuk emoji dan aksara tambahan. Anda boleh memilih set aksara yang berbeza di sini, tetapi `utf8mb4` disyorkan untuk hampir semua bahasa dan tempat dan harus dibiarkan seperti sedia ada melainkan anda mempunyai sebab khusus untuk mengubahnya.#### Pengumpulan pangkalan dataMedan pengumpulan dibiarkan kosong secara lalai. Apabila kosong, MySQL menggunakan pengumpulan lalai untuk mana-mana set aksara yang dipilih di atas (untuk `utf8mb4` ini biasanya `utf8mb4_general_ci` atau `utf8mb4_0900_ai_ci`, bergantung pada versi MySQL). Jika anda memerlukan pengumpulan khusus — contohnya untuk memadankan pangkalan data sedia ada — pilihnya di sini. Jika tidak, membiarkannya kosong adalah pilihan yang disyorkan.
