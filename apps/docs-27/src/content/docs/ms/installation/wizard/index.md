Wizard pemasangan XOOPS membimbing anda melalui proses 15 langkah yang mengkonfigurasi pangkalan data anda, mencipta akaun pentadbir dan menyediakan tapak anda untuk kegunaan pertama.## Sebelum anda bermula- Anda telah [memuat naik XOOPS ke pelayan anda](/XOOPS-docs/2.7/installation/ftp-upload/) atau menyediakan persekitaran setempat
- Anda telah [mengesahkan keperluan](/XOOPS-docs/2.7/installation/requirements/)
- Anda mempunyai bukti kelayakan pangkalan data anda sedia## Langkah wizard| Langkah | Skrin | Apa yang berlaku |
|------|--------|--------------|
| 1 | [Pemilihan Bahasa](./step-01/) | Pilih bahasa pemasangan |
| 2 | [Selamat Datang](./step-02/) | Perjanjian lesen |
| 3 | [Semakan Konfigurasi](./step-03/) | Semakan persekitaran PHP/server |
| 4 | [Tetapan Laluan](./step-04/) | Tetapkan laluan akar dan URL |
| 5 | [Sambungan Pangkalan Data](./step-05/) | Masukkan hos pangkalan data, pengguna, kata laluan |
| 6 | [Konfigurasi Pangkalan Data](./step-06/) | Tetapkan nama pangkalan data dan awalan jadual |
| 7 | [Simpan Konfigurasi](./step-07/) | Tulis mainfile.php |
| 8 | [Pembuatan Jadual](./step-08/) | Cipta skema pangkalan data |
| 9 | [Tetapan Awal](./step-09/) | Nama tapak, e-mel pentadbir |
| 10 | [Sisipan Data](./step-10/) | Isi data lalai |
| 11 | [Konfigurasi Tapak](./step-11/) | URL, zon waktu, bahasa |
| 12 | [Pilih Tema](./step-12/) | Pilih tema lalai |
| 13 | [Pemasangan Modul](./step-13/) | Pasang modul yang digabungkan |
| 14 | [Selamat Datang](./step-14/) | Mesej lengkap pemasangan |
| 15 | [Pembersihan](./step-15/) | Alih keluar folder pemasangan |:::berhati-hati[Keselamatan]
Selepas melengkapkan wizard, **padam atau namakan semula folder `install/`** — langkah 15 membimbing anda melalui ini. Membiarkannya boleh diakses adalah risiko keselamatan.
:::
