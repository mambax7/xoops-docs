Halaman ini mengumpulkan pilihan anda untuk tema awal untuk tapak baharu anda. Tema mengawal watak visual tapak anda. Tema boleh ditukar pada bila-bila masa, jika mahu.Selepas memasukkan maklumat yang diminta dan membetulkan sebarang isu, pilih butang "Teruskan" untuk meneruskan.![XOOPS Pilih Tema](/XOOPS-docs/2.7/img/installation/installer-12.png)## Data Dikumpul dalam Langkah Ini### Pilih tema lalai#### Tema lalaiPilih tema awal untuk tapak anda daripada senarai. Tangkapan skrin bagi setiap tema ditunjukkan semasa ia dipilih.XOOPS 2.7.0 dihantar dengan beberapa tema bahagian hadapan di luar kotak:* **lalai** — tema garis dasar yang minimum, berguna sebagai titik permulaan untuk penyesuaian.
* **xbootstrap5** — Tema Bootstrap 5, disyorkan untuk kebanyakan tapak baharu.
* **xswatch5** — Bootswatch 5 varian; satu set subtema gaya di atas Bootstrap 5.
* **xtailwind** — Tema berasaskan CSS Tailwind + DaisyUI dengan 35 palet dan interaktiviti Alpine.js
* **xtailwind2** — tema adik-beradik terarah seni dengan palet terpilih dan penggayaan modul yang diperhalusiAnda boleh menukar tema pada bila-bila masa dari kawasan pentadbiran dan tema tambahan boleh dimuat naik ke direktori `themes/`.
