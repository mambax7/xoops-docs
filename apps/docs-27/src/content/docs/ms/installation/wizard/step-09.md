Halaman ini mengumpulkan maklumat yang digunakan untuk membuat akaun Pentadbir utama (AKA Webmaster).Selepas memasukkan maklumat yang diminta dan membetulkan sebarang isu, pilih butang "Teruskan" untuk meneruskan.![Tetapan Awal Pemasang XOOPS](/XOOPS-docs/2.7/img/installation/installer-09.png)## Data Dikumpul dalam Langkah Ini### Akaun pentadbir#### Log masuk pentadbirIni akan menjadi nama pengguna yang digunakan oleh pentadbir untuk log masuk ke sistem XOOPS.#### E-mel pentadbirIni alamat e-mel yang digunakan oleh sistem XOOPS untuk berkomunikasi dengan pentadbir. (Nota, ini diperlukan untuk kata laluan recovery/reset.)#### Kata laluan pentadbirIni adalah kata laluan untuk digunakan untuk akaun pentadbir,#### Sahkan kata laluanGunakan medan ini untuk mengesahkan kata laluan pentadbir. Ia mesti sepadan dengan medan _Kata laluan Admin_.### Kekuatan kata laluanBahagian ini memaparkan kekuatan relatif _kata laluan Admin_.#### Penjana Kata LaluanAnda boleh menggunakan bahagian ini untuk menjana dan menggunakan kata laluan rawak untuk akaun tersebut. Pilih _Generate_ untuk menunjukkan kata laluan rawak. Pilih _Copy_ untuk menyalin kata laluan yang dijana ke _Admin password_.
