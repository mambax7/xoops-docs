# XOOPS ภาพรวมแผงผู้ดูแลระบบ

คู่มือฉบับสมบูรณ์สำหรับการนำทางและใช้งานแดชบอร์ดผู้ดูแลระบบ XOOPS

## การเข้าถึงแผงผู้ดูแลระบบ

### เข้าสู่ระบบผู้ดูแลระบบ

เปิดเบราว์เซอร์ของคุณและไปที่:
```
http://your-domain.com/xoops/admin/
```
หรือถ้า XOOPS อยู่ในรูท:
```
http://your-domain.com/admin/
```
ป้อนข้อมูลรับรองผู้ดูแลระบบของคุณ:
```
Username: [Your admin username]
Password: [Your admin password]
```
### หลังจากเข้าสู่ระบบ

คุณจะเห็นแดชบอร์ดผู้ดูแลระบบหลัก:
```

    subgraph Dashboard["🖥️ Admin Dashboard"]
        direction TB
        Header["👤 Welcome, Administrator | Logout | My Account"]
        Title["📊 Dashboard"]

    end

    Header --> Title

    style Header fill:#1976d2,color:#fff

```

## ส่วนประกอบแดชบอร์ด

### แถบด้านบน

แถบด้านบนประกอบด้วยการควบคุมที่จำเป็น:

| องค์ประกอบ | วัตถุประสงค์ |
|---|---|
| **โลโก้แอดมิน** | คลิกเพื่อกลับสู่แดชบอร์ด |
| **ข้อความต้อนรับ** | แสดงชื่อผู้ดูแลระบบที่เข้าสู่ระบบ |
| **บัญชีของฉัน** | แก้ไขโปรไฟล์ผู้ดูแลระบบและรหัสผ่าน |
| **ช่วยเหลือ** | เอกสารการเข้าถึง |
| **ออกจากระบบ** | ออกจากระบบแผงผู้ดูแลระบบ |

### พื้นที่เนื้อหาหลัก

แสดงข้อมูลและการควบคุมสำหรับส่วนที่เลือก:

- แบบฟอร์มสำหรับการกำหนดค่า
- ตารางข้อมูลพร้อมรายการ
- แผนภูมิและสถิติ
- ปุ่มดำเนินการด่วน
- ข้อความช่วยเหลือและคำแนะนำเครื่องมือ

### วิดเจ็ตแดชบอร์ด

เข้าถึงข้อมูลสำคัญได้อย่างรวดเร็ว:

- **ข้อมูลระบบ:** เวอร์ชัน PHP, เวอร์ชัน MySQL, เวอร์ชัน XOOPS
- **สถิติด่วน:** จำนวนผู้ใช้ โพสต์ทั้งหมด โมดูลที่ติดตั้ง
- **กิจกรรมล่าสุด:** การเข้าสู่ระบบล่าสุด การเปลี่ยนแปลงเนื้อหา ข้อผิดพลาด
- **สถานะเซิร์ฟเวอร์:** CPU, หน่วยความจำ, การใช้งานดิสก์
- **การแจ้งเตือน:** การแจ้งเตือนของระบบ รอการอัปเดต

## ฟังก์ชั่นผู้ดูแลระบบหลัก

### การจัดการระบบ

**ตำแหน่ง:** ระบบ > [ตัวเลือกต่างๆ]

#### การตั้งค่า

กำหนดการตั้งค่าระบบพื้นฐาน:
```
System > Preferences > [Settings Category]
```
หมวดหมู่:
- การตั้งค่าทั่วไป (ชื่อไซต์ เขตเวลา)
- การตั้งค่าผู้ใช้ (การลงทะเบียน, โปรไฟล์)
- การตั้งค่าอีเมล (การกำหนดค่าSMTP)
- การตั้งค่าแคช (ตัวเลือกแคช)
- URL การตั้งค่า (URL ที่จำง่าย)
- เมตาแท็ก (การตั้งค่าSEO)

ดูการกำหนดค่าพื้นฐานและการตั้งค่าระบบ

#### ผู้ใช้ที่เป็นผู้ดูแลระบบ

จัดการบัญชีผู้ดูแลระบบ:
```
System > Admin Users
```
ฟังก์ชั่น:
- เพิ่มผู้ดูแลระบบใหม่
- แก้ไขโปรไฟล์ผู้ดูแลระบบ
- เปลี่ยนรหัสผ่านผู้ดูแลระบบ
- ลบบัญชีผู้ดูแลระบบ
- ตั้งค่าการอนุญาตของผู้ดูแลระบบ

### การจัดการเนื้อหา

**ตำแหน่ง:** เนื้อหา > [ตัวเลือกต่างๆ]

#### หน้า/บทความ

จัดการเนื้อหาไซต์:
```
Content > Pages (or your module)
```
ฟังก์ชั่น:
- สร้างหน้าใหม่
- แก้ไขเนื้อหาที่มีอยู่
- ลบหน้า
- เผยแพร่/ไม่เผยแพร่
- กำหนดหมวดหมู่
- จัดการการแก้ไข

#### หมวดหมู่

จัดระเบียบเนื้อหา:
```
Content > Categories
```
ฟังก์ชั่น:
- สร้างลำดับชั้นของหมวดหมู่
- แก้ไขหมวดหมู่
- ลบหมวดหมู่
- กำหนดให้กับเพจ

#### ความคิดเห็น

ความคิดเห็นของผู้ใช้ปานกลาง:
```
Content > Comments
```
ฟังก์ชั่น:
- ดูความคิดเห็นทั้งหมด
- อนุมัติความคิดเห็น
- แก้ไขความคิดเห็น
- ลบสแปม
- บล็อกผู้แสดงความคิดเห็น

### การจัดการผู้ใช้

**ตำแหน่ง:** ผู้ใช้ > [ตัวเลือกต่างๆ]

#### ผู้ใช้

จัดการบัญชีผู้ใช้:
```
Users > Users
```
ฟังก์ชั่น:
- ดูผู้ใช้ทั้งหมด
- สร้างผู้ใช้ใหม่
- แก้ไขโปรไฟล์ผู้ใช้
- ลบบัญชี
- รีเซ็ตรหัสผ่าน
- เปลี่ยนสถานะผู้ใช้
- มอบหมายให้กับกลุ่ม

#### ผู้ใช้ออนไลน์

ตรวจสอบผู้ใช้ที่ใช้งานอยู่:
```
Users > Online Users
```
การแสดง:
- ผู้ใช้ออนไลน์ในปัจจุบัน
- เวลากิจกรรมสุดท้าย
- ที่อยู่ IP
- ตำแหน่งของผู้ใช้ (หากกำหนดค่าไว้)

#### กลุ่มผู้ใช้

จัดการบทบาทผู้ใช้และการอนุญาต:
```
Users > Groups
```
ฟังก์ชั่น:
- สร้างกลุ่มที่กำหนดเอง
- ตั้งค่าการอนุญาตกลุ่ม
- กำหนดผู้ใช้ให้กับกลุ่ม
- ลบกลุ่ม

### การจัดการโมดูล

**ตำแหน่ง:** โมดูล > [ตัวเลือกต่างๆ]

#### โมดูล

ติดตั้งและกำหนดค่าโมดูล:
```
Modules > Modules
```
ฟังก์ชั่น:
- ดูโมดูลที่ติดตั้ง
- เปิด/ปิดโมดูล
- อัปเดตโมดูล
- กำหนดการตั้งค่าโมดูล
- ติดตั้งโมดูลใหม่
- ดูรายละเอียดโมดูล

#### ตรวจสอบการอัปเดต
```
Modules > Modules > Check for Updates
```
แสดง:
- การอัปเดตโมดูลที่มีอยู่
- บันทึกการเปลี่ยนแปลง
- ดาวน์โหลดและติดตั้งตัวเลือก

### การจัดการรูปลักษณ์

**ตำแหน่ง:** ลักษณะที่ปรากฏ > [ตัวเลือกต่างๆ]

#### ธีมส์

จัดการธีมของไซต์:
```
Appearance > Themes
```
ฟังก์ชั่น:
- ดูธีมที่ติดตั้ง
- ตั้งค่าธีมเริ่มต้น
- อัปโหลดธีมใหม่
- ลบธีม
- ดูตัวอย่างธีม
- การกำหนดค่าธีม

#### บล็อก

จัดการบล็อกเนื้อหา:
```
Appearance > Blocks
```
ฟังก์ชั่น:
- สร้างบล็อกที่กำหนดเอง
- แก้ไขเนื้อหาบล็อก
- จัดเรียงบล็อกบนหน้า
- ตั้งค่าการมองเห็นบล็อก
- ลบบล็อก
- กำหนดค่าการบล็อกแคช

#### เทมเพลต

จัดการเทมเพลต (ขั้นสูง):
```
Appearance > Templates
```
สำหรับผู้ใช้และนักพัฒนาขั้นสูง

### เครื่องมือระบบ

**ตำแหน่ง:** ระบบ > เครื่องมือ

#### โหมดการบำรุงรักษา

ป้องกันการเข้าถึงของผู้ใช้ระหว่างการบำรุงรักษา:
```
System > Maintenance Mode
```
กำหนดค่า:
- เปิด/ปิดการบำรุงรักษา
- ข้อความการบำรุงรักษาแบบกำหนดเอง
- ที่อยู่ IP ที่อนุญาต (สำหรับการทดสอบ)

#### การจัดการฐานข้อมูล
```
System > Database
```
ฟังก์ชั่น:
- ตรวจสอบความสอดคล้องของฐานข้อมูล
- เรียกใช้การอัพเดตฐานข้อมูล
- ซ่อมแซมโต๊ะ
- เพิ่มประสิทธิภาพฐานข้อมูล
- ส่งออกโครงสร้างฐานข้อมูล

#### บันทึกกิจกรรม
```
System > Logs
```
จอภาพ:
- กิจกรรมของผู้ใช้
- การดำเนินการด้านการบริหาร
- เหตุการณ์ของระบบ
- บันทึกข้อผิดพลาด

## การจัดการบัญชีผู้ใช้

### บัญชีของฉัน

เข้าถึงโปรไฟล์ผู้ดูแลระบบของคุณ:

1. คลิก "บัญชีของฉัน" ที่มุมขวาบน
2. แก้ไขข้อมูลโปรไฟล์:
   - ที่อยู่อีเมล
   - ชื่อจริง
   - ข้อมูลผู้ใช้
   - อวตาร

### เปลี่ยนรหัสผ่าน

เปลี่ยนรหัสผ่านผู้ดูแลระบบของคุณ:

1. ไปที่ **บัญชีของฉัน**
2. คลิก "เปลี่ยนรหัสผ่าน"
3. ป้อนรหัสผ่านปัจจุบัน
4. กรอกรหัสผ่านใหม่ (สองครั้ง)
5. คลิก "บันทึก"

**เคล็ดลับด้านความปลอดภัย:**
- ใช้รหัสผ่านที่รัดกุม (16+ ตัวอักษร)
- ประกอบด้วยตัวพิมพ์ใหญ่ ตัวพิมพ์เล็ก ตัวเลข สัญลักษณ์
- เปลี่ยนรหัสผ่านทุกๆ 90 วัน
- อย่าเปิดเผยข้อมูลประจำตัวของผู้ดูแลระบบ

### ออกจากระบบ

ออกจากระบบแผงผู้ดูแลระบบ:

1. คลิก "ออกจากระบบ" ที่มุมขวาบน
2. คุณจะถูกนำไปยังหน้าเข้าสู่ระบบ

## สถิติแผงผู้ดูแลระบบ

### สถิติแดชบอร์ด

ภาพรวมโดยย่อของการวัดไซต์:

| เมตริก | ค่า |
|--------|--------|
| ผู้ใช้ออนไลน์ | 12 |
| ผู้ใช้ทั้งหมด | 256 |
| กระทู้ทั้งหมด | 1,234 |
| ความคิดเห็นทั้งหมด | 5,678 |
| โมดูลทั้งหมด | 8 |

### สถานะของระบบ

ข้อมูลเซิร์ฟเวอร์และประสิทธิภาพ:

| ส่วนประกอบ | เวอร์ชัน/ค่า |
|----------|---------------|
| XOOPS เวอร์ชัน | 2.5.11 |
| PHP เวอร์ชัน | 8.2.x |
| เวอร์ชัน MySQL | 8.0.x |
| โหลดเซิร์ฟเวอร์ | 0.45, 0.42 |
| เวลาทำงาน | 45 วัน |

### กิจกรรมล่าสุด

ไทม์ไลน์ของเหตุการณ์ล่าสุด:
```
12:45 - Admin login
12:30 - New user registered
12:15 - Page published
12:00 - Comment posted
11:45 - Module updated
```
## ระบบการแจ้งเตือน

### การแจ้งเตือนผู้ดูแลระบบ

รับการแจ้งเตือนสำหรับ:

- การลงทะเบียนผู้ใช้ใหม่
- ความคิดเห็นที่รอการกลั่นกรอง
- ความพยายามเข้าสู่ระบบล้มเหลว
- ข้อผิดพลาดของระบบ
- มีการอัปเดตโมดูล
- ปัญหาฐานข้อมูล
- คำเตือนพื้นที่ดิสก์

กำหนดค่าการแจ้งเตือน:

**ระบบ > ค่ากำหนด > การตั้งค่าอีเมล**
```
Notify Admin on Registration: Yes
Notify Admin on Comments: Yes
Notify Admin on Errors: Yes
Alert Email: admin@your-domain.com
```
## งานผู้ดูแลระบบทั่วไป

### สร้างเพจใหม่

1. ไปที่ **เนื้อหา > หน้า** (หรือโมดูลที่เกี่ยวข้อง)
2. คลิก "เพิ่มหน้าใหม่"
3. กรอก:
   - ชื่อเรื่อง
   - เนื้อหา
   - คำอธิบาย
   - หมวดหมู่
   - ข้อมูลเมตา
4. คลิก "เผยแพร่"

### จัดการผู้ใช้

1. ไปที่ **ผู้ใช้ > ผู้ใช้**
2. ดูรายชื่อผู้ใช้ด้วย:
   - ชื่อผู้ใช้
   - อีเมล
   - วันที่ลงทะเบียน
   - เข้าสู่ระบบครั้งล่าสุด
   - สถานะ

3. คลิกชื่อผู้ใช้เพื่อ:
   - แก้ไขโปรไฟล์
   - เปลี่ยนรหัสผ่าน
   - แก้ไขกลุ่ม
   - บล็อก / ปลดบล็อกผู้ใช้

### กำหนดค่าโมดูล

1. ไปที่ **โมดูล > โมดูล**
2. ค้นหาโมดูลในรายการ
3. คลิกชื่อโมดูล
4. คลิก "การตั้งค่า" หรือ "การตั้งค่า"
5. กำหนดค่าตัวเลือกโมดูล
6. บันทึกการเปลี่ยนแปลง

### สร้างบล็อกใหม่

1. ไปที่ **รูปลักษณ์ > บล็อก**
2. คลิก "เพิ่มบล็อกใหม่"
3. ป้อน:
   - ชื่อบล็อก
   - บล็อกเนื้อหา (อนุญาต HTML)
   - ตำแหน่งบนหน้า
   - การมองเห็น (ทุกหน้าหรือเฉพาะเจาะจง)
   - โมดูล (ถ้ามี)
4. คลิก "ส่ง"

## ความช่วยเหลือจากแผงผู้ดูแลระบบ

เข้าถึงความช่วยเหลือจากแผงผู้ดูแลระบบ:

1. คลิกปุ่ม "ช่วยเหลือ" ในแถบด้านบน
2. ความช่วยเหลือตามบริบทสำหรับหน้าปัจจุบัน
3. ลิงก์ไปยังเอกสารประกอบ
4. คำถามที่พบบ่อย

## การปรับแต่งแผงผู้ดูแลระบบ

### ธีมผู้ดูแลระบบ

เลือกธีมอินเทอร์เฟซผู้ดูแลระบบ:

**ระบบ > ค่ากำหนด > การตั้งค่าทั่วไป**
```
Admin Theme: [Select theme]
```
ธีมที่มี:
- ค่าเริ่มต้น (เบา)
- โหมดมืด
- ธีมที่กำหนดเอง

### การปรับแต่งแดชบอร์ด

เลือกวิดเจ็ตที่จะปรากฏ:

**แดชบอร์ด > ปรับแต่ง**

เลือก:
- ข้อมูลระบบ
- สถิติ
- กิจกรรมล่าสุด
- ลิงค์ด่วน
- วิดเจ็ตที่กำหนดเอง

## สิทธิ์ของแผงผู้ดูแลระบบ

ระดับผู้ดูแลระบบที่แตกต่างกันมีสิทธิ์ที่แตกต่างกัน:

| บทบาท | ความสามารถ |
|---|---|
| **เว็บมาสเตอร์** | เข้าถึงฟังก์ชันผู้ดูแลระบบทั้งหมดโดยสมบูรณ์ |
| **ผู้ดูแลระบบ** | ฟังก์ชั่นผู้ดูแลระบบจำกัด |
| **ผู้ดำเนินรายการ** | การกลั่นกรองเนื้อหาเท่านั้น |
| **บรรณาธิการ** | การสร้างและแก้ไขเนื้อหา |

จัดการสิทธิ์:

**ระบบ > สิทธิ์**

## แนวทางปฏิบัติที่ดีที่สุดด้านความปลอดภัยสำหรับแผงผู้ดูแลระบบ

1. **รหัสผ่านที่รัดกุม:** ใช้รหัสผ่าน 16 ตัวอักษรขึ้นไป
2. **การเปลี่ยนแปลงปกติ:** เปลี่ยนรหัสผ่านทุกๆ 90 วัน
3. **การเข้าถึงการตรวจสอบ:** ตรวจสอบบันทึก "ผู้ใช้ผู้ดูแลระบบ" เป็นประจำ
4. **จำกัดการเข้าถึง:** เปลี่ยนชื่อโฟลเดอร์ผู้ดูแลระบบเพื่อความปลอดภัยเพิ่มเติม
5. **ใช้ HTTPS:** เข้าถึงผู้ดูแลระบบผ่าน HTTPS เสมอ
6. **IP ไวท์ลิสต์:** จำกัดการเข้าถึงของผู้ดูแลระบบสำหรับ IP ที่ระบุ
7. **การออกจากระบบปกติ:** การออกจากระบบเมื่อเสร็จสิ้น
8. **ความปลอดภัยของเบราว์เซอร์:** ล้างแคชของเบราว์เซอร์เป็นประจำ

ดูการกำหนดค่าความปลอดภัย

## การแก้ไขปัญหาแผงผู้ดูแลระบบ

### ไม่สามารถเข้าถึงแผงผู้ดูแลระบบได้

**วิธีแก้ปัญหา:**
1. ตรวจสอบข้อมูลการเข้าสู่ระบบ
2. ล้างแคชและคุกกี้ของเบราว์เซอร์
3. ลองใช้เบราว์เซอร์อื่น
4. ตรวจสอบว่าเส้นทางโฟลเดอร์ผู้ดูแลระบบถูกต้องหรือไม่
5. ตรวจสอบการอนุญาตไฟล์ในโฟลเดอร์ผู้ดูแลระบบ
6. ตรวจสอบการเชื่อมต่อฐานข้อมูลใน mainfile.php

### หน้าผู้ดูแลระบบว่างเปล่า

**วิธีแก้ปัญหา:**
```bash
# Check PHP errors
tail -f /var/log/apache2/error.log

# Enable debug mode temporarily
sed -i "s/define('XOOPS_DEBUG', 0)/define('XOOPS_DEBUG', 1)/" /var/www/html/xoops/mainfile.php

# Check file permissions
ls -la /var/www/html/xoops/admin/
```
### แผงผู้ดูแลระบบช้า

**วิธีแก้ปัญหา:**
1. ล้างแคช: **ระบบ > เครื่องมือ > ล้างแคช**
2. ปรับฐานข้อมูลให้เหมาะสม: **ระบบ > ฐานข้อมูล > ปรับให้เหมาะสม**
3. ตรวจสอบทรัพยากรเซิร์ฟเวอร์: `htop`
4. ตรวจสอบการสืบค้นที่ช้าใน MySQL

### โมดูลไม่ปรากฏ

**วิธีแก้ปัญหา:**
1. ตรวจสอบการติดตั้งโมดูล: **โมดูล > โมดูล**
2. ตรวจสอบการเปิดใช้งานโมดูล
3. ตรวจสอบสิทธิ์ที่ได้รับมอบหมาย
4. ตรวจสอบไฟล์โมดูลที่มีอยู่
5. ตรวจสอบบันทึกข้อผิดพลาด

## ขั้นตอนต่อไป

หลังจากทำความคุ้นเคยกับแผงผู้ดูแลระบบแล้ว:

1. สร้างหน้าแรกของคุณ
2. ตั้งค่ากลุ่มผู้ใช้
3. ติดตั้งโมดูลเพิ่มเติม
4. กำหนดการตั้งค่าพื้นฐาน
5. ดำเนินการรักษาความปลอดภัย
