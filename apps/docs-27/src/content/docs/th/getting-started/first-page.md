# การสร้างหน้าแรกของคุณใน XOOPS

เรียนรู้วิธีสร้าง จัดรูปแบบ และเผยแพร่เนื้อหาชิ้นแรกของคุณใน XOOPS

## ทำความเข้าใจเนื้อหา XOOPS

### เพจ/โพสต์คืออะไร?

ใน XOOPS เนื้อหาได้รับการจัดการผ่านโมดูล ประเภทเนื้อหาที่พบบ่อยที่สุดคือ:

| พิมพ์ | คำอธิบาย | ใช้กรณี |
|---|---|---|
| **หน้า** | เนื้อหาคงที่ | เกี่ยวกับเรา ติดต่อ บริการ |
| **โพสต์/บทความ** | เนื้อหาที่มีการประทับเวลา | ข่าว, โพสต์ในบล็อก |
| **หมวดหมู่** | การจัดระเบียบเนื้อหา | เนื้อหาที่เกี่ยวข้องกับกลุ่ม |
| **ความคิดเห็น** | ความคิดเห็นของผู้ใช้ | อนุญาตให้ผู้เยี่ยมชมโต้ตอบ |

คู่มือนี้ครอบคลุมถึงการสร้างหน้า/บทความพื้นฐานโดยใช้โมดูลเนื้อหาเริ่มต้น XOOPS'

## การเข้าถึงตัวแก้ไขเนื้อหา

### จากแผงผู้ดูแลระบบ

1. เข้าสู่แผงผู้ดูแลระบบ: `http://your-domain.com/xoops/admin/`
2. ไปที่ **เนื้อหา > หน้า** (หรือโมดูลเนื้อหาของคุณ)
3. คลิก "เพิ่มหน้าใหม่" หรือ "โพสต์ใหม่"

### ส่วนหน้า (หากเปิดใช้งาน)

หาก XOOPS ของคุณได้รับการกำหนดค่าให้อนุญาตการสร้างเนื้อหาส่วนหน้า:

1. เข้าสู่ระบบในฐานะผู้ใช้ที่ลงทะเบียน
2. ไปที่โปรไฟล์ของคุณ
3. ค้นหาตัวเลือก "ส่งเนื้อหา"
4. ทำตามขั้นตอนเดียวกันด้านล่าง

## คำแนะนำทีละขั้นตอน: การสร้างหน้าแรกของคุณ

### ขั้นตอนที่ 1: เข้าถึงตัวแก้ไขเนื้อหา

1. ในแผงผู้ดูแลระบบ คลิก **เนื้อหา > หน้า**
2. คลิก **"เพิ่มหน้าใหม่"** หรือ **"สร้าง"**
3. คุณจะเห็นโปรแกรมแก้ไขเนื้อหา

### ขั้นตอนที่ 2: ป้อนชื่อหน้า

ในช่อง "ชื่อเรื่อง" ป้อนชื่อเพจของคุณ:
```
Title: Welcome to Our Website
```
แนวทางปฏิบัติที่ดีที่สุดสำหรับชื่อ:
- มีความชัดเจนและสื่อความหมาย
- รวมคำหลักถ้าเป็นไปได้
- เหมาะสำหรับอักขระ 50-60 ตัว
- หลีกเลี่ยง ALL CAPS (อ่านยาก)
- เจาะจง (ไม่ใช่ "หน้า 1")

### ขั้นตอนที่ 3: เลือกหมวดหมู่

เลือกตำแหน่งที่จะจัดระเบียบเนื้อหานี้:
```
Category: [Dropdown ▼]
```
ตัวเลือกอาจรวมถึง:
- ทั่วไป
- ข่าว
- บล็อก
- ประกาศ
- บริการ

หากไม่มีหมวดหมู่ โปรดขอให้ผู้ดูแลระบบสร้างหมวดหมู่เหล่านั้น

### ขั้นตอนที่ 4: เขียนเนื้อหาของคุณ

คลิกในพื้นที่แก้ไขเนื้อหาแล้วพิมพ์ข้อความของคุณ

#### การจัดรูปแบบข้อความพื้นฐาน

ใช้แถบเครื่องมือแก้ไข:

| ปุ่ม | การกระทำ | ผลลัพธ์ |
|---|---|---|
| **ข** | ตัวหนา | **ข้อความตัวหนา** |
| *ฉัน* | ตัวเอียง | *ข้อความตัวเอียง* |
| <u>คุณ</u> | ขีดเส้นใต้ | <u>ข้อความที่ขีดเส้นใต้</u> |

#### การใช้ HTML

XOOPS อนุญาตแท็ก HTML ที่ปลอดภัย ตัวอย่างทั่วไป:
```html
<!-- Paragraphs -->
<p>This is a paragraph.</p>

<!-- Headings -->
<h1>Main Heading</h1>
<h2>Subheading</h2>

<!-- Lists -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<!-- Bold and Italic -->
<strong>Bold text</strong>
<em>Italic text</em>

<!-- Links -->
<a href="https://example.com">Link text</a>

<!-- Line breaks -->
<br>

<!-- Horizontal rule -->
<hr>
```
#### ตัวอย่าง HTML ปลอดภัย

**แท็กแนะนำ:**
- ย่อหน้า: `<p>`, `<br>`
- หัวข้อ: `<h1>` ถึง `<h6>`
- ข้อความ: `<strong>`, `<em>`, `<u>`
- รายการ: `<ul>`, `<ol>`, `<li>`
- ลิงก์: `<a href="">`
- บล็อกโควต: `<blockquote>`
- ตาราง: `<table>`, `<tr>`, `<td>`

**หลีกเลี่ยงแท็กเหล่านี้** (อาจถูกปิดใช้งานเพื่อความปลอดภัย):
- สคริปต์: `<script>`
- รูปแบบ: `<style>`
- Iframes: `<iframe>` (เว้นแต่จะกำหนดค่าไว้)
- แบบฟอร์ม: `<form>`, `<input>`

### ขั้นตอนที่ 5: เพิ่มรูปภาพ

#### ตัวเลือก 1: แทรกรูปภาพ URL

การใช้ตัวแก้ไข:

1. คลิกปุ่ม **แทรกรูปภาพ** (ไอคอนรูปภาพ)
2. ป้อนรูปภาพ URL: `https://example.com/image.jpg`
3. ป้อนข้อความแสดงแทน: "คำอธิบายรูปภาพ"
4. คลิก "แทรก"

HTML เทียบเท่า:
```html
<img src="https://example.com/image.jpg" alt="Description">
```
#### ตัวเลือก 2: อัปโหลดรูปภาพ

1. อัปโหลดรูปภาพไปที่ XOOPS ก่อน:
   - ไปที่ **เนื้อหา > ตัวจัดการสื่อ**
   - อัปโหลดภาพของคุณ
   - คัดลอกรูปภาพ URL

2. ในโปรแกรมแก้ไขเนื้อหา ให้แทรกโดยใช้ URL (ขั้นตอนข้างต้น)

#### แนวทางปฏิบัติที่ดีที่สุดสำหรับรูปภาพ

- ใช้ขนาดไฟล์ที่เหมาะสม (ปรับภาพให้เหมาะสม)
- ใช้ชื่อไฟล์ที่สื่อความหมาย
- รวมข้อความแสดงแทนเสมอ (การเข้าถึง)
- รูปแบบที่รองรับ: JPG, PNG, GIF, WebP
- ความกว้างที่แนะนำ: 600-800 พิกเซลสำหรับเนื้อหา

### ขั้นตอนที่ 6: ฝังสื่อ

#### ฝังวิดีโอจาก YouTube
```html
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  frameborder="0" allowfullscreen>
</iframe>
```
แทนที่ `VIDEO_ID` ด้วยวิดีโอ YouTube ID

**วิธีค้นหาวิดีโอ YouTube ID:**
1. เปิดวิดีโอบน YouTube
2. URL คือ: `https://www.youtube.com/watch?v=VIDEO_ID`
3. คัดลอก ID (อักขระหลัง `v=`)

#### ฝังวิดีโอจาก Vimeo
```html
<iframe src="https://player.vimeo.com/video/VIDEO_ID"
  width="640" height="360" frameborder="0"
  allow="autoplay; fullscreen" allowfullscreen>
</iframe>
```
### ขั้นตอนที่ 7: เพิ่มคำอธิบาย Meta

ในช่อง "คำอธิบาย" ให้เพิ่มข้อมูลสรุปโดยย่อ:
```
Description: Learn how to get started with our website.
This page provides an overview of our services and how we can help you.
```
**แนวทางปฏิบัติที่ดีที่สุดสำหรับคำอธิบาย Meta:**
- 150-160 ตัวอักษร
- ใส่คีย์เวิร์ดหลัก
- ควรสรุปเนื้อหาให้ถูกต้อง
- ใช้ในผลลัพธ์ของเครื่องมือค้นหา
- ทำให้น่าสนใจ (ผู้ใช้เห็นสิ่งนี้)

### ขั้นตอนที่ 8: กำหนดค่าตัวเลือกการเผยแพร่

#### สถานะการเผยแพร่

เลือกสถานะการตีพิมพ์:
```
Status: ☑ Published
```
ตัวเลือก:
- **เผยแพร่แล้ว:** ปรากฏต่อสาธารณะ
- **ฉบับร่าง:** เฉพาะผู้ดูแลระบบเท่านั้นที่มองเห็นได้
- **รอการตรวจสอบ:** รอการอนุมัติ
- **เก็บถาวรแล้ว:** ซ่อนไว้แต่เก็บไว้

#### การมองเห็น

กำหนดผู้ที่สามารถดูเนื้อหานี้:
```
Visibility: ☐ Public
           ☐ Registered Users Only
           ☐ Private (Admin Only)
```
#### วันที่ตีพิมพ์

ตั้งค่าเมื่อเนื้อหาปรากฏ:
```
Publish Date: [Date Picker] [Time]
```
ปล่อยให้เป็น "ตอนนี้" เพื่อเผยแพร่ทันที

#### ขออนุญาตแสดงความคิดเห็น

เปิดหรือปิดความคิดเห็นของผู้เยี่ยมชม:
```
Allow Comments: ☑ Yes
```
หากเปิดใช้งาน ผู้เยี่ยมชมสามารถเพิ่มคำติชมได้

### ขั้นตอนที่ 9: บันทึกเนื้อหาของคุณ

ตัวเลือกการบันทึกหลายรายการ:
```
[Publish Now]  [Save as Draft]  [Schedule]  [Preview]
```
- **เผยแพร่ทันที:** ทำให้มองเห็นได้ทันที
- **บันทึกเป็นฉบับร่าง:** เก็บไว้เป็นส่วนตัวในตอนนี้
- **กำหนดการ:** เผยแพร่ตามวัน/เวลาในอนาคต
- **ดูตัวอย่าง:** ดูลักษณะก่อนที่จะบันทึก

คลิกตัวเลือกของคุณ:
```
Click [Publish Now]
```
### ขั้นตอนที่ 10: ยืนยันเพจของคุณ

หลังจากเผยแพร่แล้ว ให้ตรวจสอบเนื้อหาของคุณ:

1. ไปที่หน้าแรกของเว็บไซต์ของคุณ
2. นำทางไปยังพื้นที่เนื้อหาของคุณ
3. ค้นหาเพจที่คุณสร้างขึ้นใหม่
4. คลิกเพื่อดู
5. ตรวจสอบ:
   - [ ] เนื้อหาแสดงอย่างถูกต้อง
   - [ ] ภาพปรากฏขึ้น
   - [ ] การจัดรูปแบบดูดี
   - [ ] ลิงค์ใช้งานได้
   - [ ] หัวข้อและคำอธิบายถูกต้อง

## SEO แนวทางปฏิบัติที่ดีที่สุดสำหรับเนื้อหา

เพิ่มประสิทธิภาพเนื้อหาของคุณสำหรับเครื่องมือค้นหา:

### ชื่อเรื่อง
- ใส่คีย์เวิร์ดหลัก
- 50-60 ตัวอักษร
- ไม่ซ้ำกันต่อหน้า

### คำอธิบายเมตา
- รวมคำหลักอย่างเป็นธรรมชาติ
- 150-160 ตัวอักษร
- น่าสนใจและแม่นยำ

### เนื้อหา
- เขียนอย่างเป็นธรรมชาติ หลีกเลี่ยงการใช้คำหลักในทางที่ผิด
- ใช้หัวข้อ (h2, h3) อย่างเหมาะสม
- รวมลิงค์ภายในไปยังหน้าอื่น ๆ
- ใช้ข้อความแสดงแทนในทุกภาพ
- ตั้งเป้าให้มีคำศัพท์มากกว่า 300 คำสำหรับบทความ

### URL โครงสร้าง
- เก็บ URL ให้สั้นและสื่อความหมาย
- ใช้ขีดคั่นเพื่อแยกคำ
- หลีกเลี่ยงอักขระพิเศษ
- ตัวอย่าง: `/about-our-company`

## การจัดการเนื้อหาของคุณ

### แก้ไขเพจที่มีอยู่

1. ไปที่ **เนื้อหา > หน้า**
2. ค้นหาเพจของคุณในรายการ
3. คลิก **แก้ไข** หรือชื่อหน้า
4. ทำการเปลี่ยนแปลง
5. คลิก **อัปเดต**

### ลบเพจ

1. ไปที่ **เนื้อหา > หน้า**
2. ค้นหาเพจของคุณ
3. คลิก **ลบ**
4. ยืนยันการลบ

### เปลี่ยนสถานะสิ่งพิมพ์

1. ไปที่ **เนื้อหา > หน้า**
2. ค้นหาเพจ คลิก **แก้ไข**
3. เปลี่ยนสถานะในรายการแบบเลื่อนลง
4. คลิก **อัปเดต**

## การแก้ไขปัญหาการสร้างเนื้อหา

### เนื้อหาไม่ปรากฏ

**อาการ:** หน้าที่เผยแพร่ไม่แสดงบนเว็บไซต์

**วิธีแก้ปัญหา:**
1. ตรวจสอบสถานะการตีพิมพ์: ควรเป็น "เผยแพร่แล้ว"
2. ตรวจสอบวันที่เผยแพร่: ควรเป็นปัจจุบันหรือในอดีต
3. ตรวจสอบการมองเห็น: ควรเป็น "สาธารณะ"
4. ล้างแคช: ผู้ดูแลระบบ > เครื่องมือ > ล้างแคช
5. ตรวจสอบสิทธิ์: กลุ่มผู้ใช้จะต้องมีสิทธิ์เข้าถึง

### การจัดรูปแบบไม่ทำงาน

**อาการ:** แท็ก HTML หรือการจัดรูปแบบปรากฏเป็นข้อความ

**วิธีแก้ปัญหา:**
1. ตรวจสอบว่า HTML เปิดใช้งานในการตั้งค่าโมดูล
2. ใช้ไวยากรณ์ HTML ที่เหมาะสม
3. ปิดแท็กทั้งหมด: `<p>Text</p>`
4. ใช้แท็กที่อนุญาตเท่านั้น
5. ใช้เอนทิตี HTML¤: `&lt;` สำหรับ `<`, `&amp;` สำหรับ `&`

### รูปภาพไม่แสดง

**อาการ:** รูปภาพแสดงไอคอนที่เสียหาย

**วิธีแก้ปัญหา:**
1. ตรวจสอบรูปภาพ URL ถูกต้อง
2. ตรวจสอบไฟล์ภาพที่มีอยู่
3. ตรวจสอบการอนุญาตที่เหมาะสมบนรูปภาพ
4. ลองอัปโหลดรูปภาพไปที่ XOOPS แทน
5. ตรวจสอบการบล็อกภายนอก (อาจต้องใช้ CORS)

### ปัญหาการเข้ารหัสอักขระ

**อาการ:** อักขระพิเศษปรากฏเป็นคำที่ไม่มีความหมาย

**วิธีแก้ปัญหา:**
1. บันทึกไฟล์เป็นการเข้ารหัส UTF-8
2. ตรวจสอบให้แน่ใจว่าชุดอักขระของหน้าเป็น UTF-8
3. เพิ่มในส่วนหัว HTML: `<meta charset="UTF-8">`
4. หลีกเลี่ยงการคัดลอกและวางจาก Word (ใช้ข้อความธรรมดา)

## แนวทางปฏิบัติที่ดีที่สุดสำหรับเวิร์กโฟลว์เนื้อหา

### ขั้นตอนที่แนะนำ

1. **เขียนในตัวแก้ไขก่อน:** ใช้โปรแกรมแก้ไขเนื้อหาของผู้ดูแลระบบ
2. **ดูตัวอย่างก่อนเผยแพร่:** คลิกปุ่มแสดงตัวอย่าง
3. **เพิ่มข้อมูลเมตา:** กรอกชื่อ คำอธิบาย แท็ก
4. **บันทึกเป็นฉบับร่างก่อน:** บันทึกเป็นฉบับร่างเพื่อหลีกเลี่ยงการสูญเสียงาน
5. **การตรวจสอบครั้งสุดท้าย:** อ่านซ้ำก่อนเผยแพร่
6. **เผยแพร่:** คลิกเผยแพร่เมื่อพร้อม
7. **ยืนยัน:** ตรวจสอบบนเว็บไซต์สด
8. **แก้ไขหากจำเป็น:** ทำการแก้ไขอย่างรวดเร็ว

### การควบคุมเวอร์ชัน

สำรองข้อมูลไว้เสมอ:

1. **ก่อนการเปลี่ยนแปลงครั้งใหญ่:** บันทึกเป็นเวอร์ชันใหม่หรือข้อมูลสำรอง
2. **เก็บเนื้อหาเก่า:** เก็บเวอร์ชันที่ยังไม่ได้เผยแพร่ไว้
3. **วันที่ฉบับร่างของคุณ:** ใช้การตั้งชื่อที่ชัดเจน: "Page-Draft-2025-01-28"
