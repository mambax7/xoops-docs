## ข้อกำหนด

| ส่วนประกอบ | ขั้นต่ำ | แนะนำ |
|------------|-------------------------|---------------|
| PHP | 8.2 | 8.4+ |
| MySQL | 5.7 | 8.0+ |
| MariaDB | 10.4 | 10.11+ |
| เว็บเซิร์ฟเวอร์ | อาปาเช่ 2.4 / Nginx 1.20 | ล่าสุดเสถียร |

## ดาวน์โหลด

ดาวน์โหลดรุ่นล่าสุดจาก [GitHub Releases](https://github.com/XOOPS/XoopsCore27/releases)
```bash
# Or clone directly
git clone https://github.com/XOOPS/XoopsCore27.git mysite
cd mysite
```
## ขั้นตอนการติดตั้ง

1. **อัปโหลดไฟล์** ไปยังรูทเอกสารของเว็บเซิร์ฟเวอร์ของคุณ (เช่น `public_html/`)
2. **สร้างฐานข้อมูล MySQL** และผู้ใช้ที่มีสิทธิ์การใช้งานเต็มรูปแบบ
3. **เปิดเบราว์เซอร์ของคุณ** และไปที่โดเมนของคุณ — โปรแกรมติดตั้ง XOOPS จะเริ่มทำงานโดยอัตโนมัติ
4. **ทำตามวิซาร์ด 5 ขั้นตอน** — กำหนดค่าเส้นทาง สร้างตาราง และตั้งค่าบัญชีผู้ดูแลระบบของคุณ
5. **ลบโฟลเดอร์ `install/`** เมื่อได้รับแจ้ง นี่เป็นข้อบังคับเพื่อความปลอดภัย

## ตรวจสอบการติดตั้ง

หลังจากตั้งค่าแล้ว ให้ไปที่:

- **หน้าแรก:** `https://yourdomain.com/`
- **แผงผู้ดูแลระบบ:** `https://yourdomain.com/xoops_data/` *(เส้นทางที่คุณเลือกระหว่างการติดตั้ง)*
