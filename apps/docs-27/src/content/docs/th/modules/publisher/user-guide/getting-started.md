#เริ่มต้นกับสำนักพิมพ์

> คำแนะนำทีละขั้นตอนในการตั้งค่าและใช้งานโมดูลข่าวสาร/บล็อกของผู้เผยแพร่

---

## ผู้จัดพิมพ์คืออะไร?

ผู้เผยแพร่โฆษณาเป็นโมดูลการจัดการเนื้อหาชั้นนำสำหรับ XOOPS ออกแบบมาสำหรับ:

- **ไซต์ข่าว** - เผยแพร่บทความพร้อมหมวดหมู่
- **บล็อก** - บล็อกส่วนตัวหรือผู้เขียนหลายคน``
- **เอกสารประกอบ** - จัดทำฐานความรู้
- **พอร์ทัลเนื้อหา** - เนื้อหาสื่อผสม
```
mermaid
graph LR
    A[Authors] --> B[Publisher]
    B --> C[Categories]
    B --> D[Articles]
    B --> E[Files]
    D --> F[Readers]
    E --> F
```

## ติดตั้งด่วน

### ขั้นตอนที่ 1: ติดตั้ง Publisher

1. ดาวน์โหลดจาก [GitHub](https://github.com/XoopsModules25x/publisher)
2. อัปโหลดไปที่ `modules/publisher/`
3. ไปที่ผู้ดูแลระบบ → โมดูล → ติดตั้ง

### ขั้นตอนที่ 2: สร้างหมวดหมู่
```
mermaid
graph TD
    A[News] --> B[Technology]
    A --> C[Sports]
    A --> D[Entertainment]
    B --> E[AI & ML]
    B --> F[Web Development]
```
1. ผู้ดูแลระบบ → ผู้เผยแพร่ → หมวดหมู่
2. คลิก "เพิ่มหมวดหมู่"
3. กรอก:
   - **ชื่อ**: ชื่อหมวดหมู่
   - **คำอธิบาย**: หมวดหมู่นี้ประกอบด้วยอะไรบ้าง
   - **รูปภาพ**: รูปภาพหมวดหมู่เสริม
4. กำหนดสิทธิ์ (ผู้ที่สามารถส่ง/ดูได้)
5. บันทึก

### ขั้นตอนที่ 3: กำหนดการตั้งค่า

1. ผู้ดูแลระบบ → ผู้เผยแพร่ → การตั้งค่า
2. การตั้งค่าที่สำคัญในการกำหนดค่า:

| การตั้งค่า | แนะนำ | คำอธิบาย |
|---------|-------------|-------------|
| รายการต่อหน้า | 10-20 | บทความเกี่ยวกับดัชนี |
| บรรณาธิการ | TinyMCE/CKEditor | โปรแกรมแก้ไขข้อความแบบ Rich Text |
| อนุญาตการให้คะแนน | ใช่ | ความคิดเห็นของผู้อ่าน |
| อนุญาตให้แสดงความคิดเห็น | ใช่ | การสนทนา |
| อนุมัติอัตโนมัติ | ไม่ | การควบคุมด้านบรรณาธิการ |

### ขั้นตอนที่ 4: สร้างบทความแรกของคุณ

1. เมนูหลัก → ผู้จัดพิมพ์ → ส่งบทความ
2. กรอกแบบฟอร์ม:
   - **หัวข้อ**: พาดหัวบทความ
   - **หมวดหมู่**: อยู่ที่ไหน
   - **สรุป**: คำอธิบายสั้นๆ
   - **เนื้อหา**: เนื้อหาบทความฉบับเต็ม
3. เพิ่มองค์ประกอบเสริม:
   - รูปภาพเด่น
   - ไฟล์แนบ
   - SEO การตั้งค่า
4. ส่งเพื่อตรวจสอบหรือเผยแพร่

## บทบาทของผู้ใช้
```
mermaid
flowchart TD
    subgraph "Permissions"
        A[Reader] --> |View| B[Published Content]
        C[Author] --> |Submit| D[Articles]
        E[Editor] --> |Approve| D
        F[Admin] --> |Full Control| G[Everything]
    end

    D --> E
    E --> B
```
### ผู้อ่าน
- ดูบทความที่เผยแพร่
- อัตราและแสดงความคิดเห็น
- ค้นหาเนื้อหา

### ผู้เขียน
- ส่งบทความใหม่
- แก้ไขบทความของตัวเอง
- แนบไฟล์

### บรรณาธิการ
- อนุมัติ / ปฏิเสธการส่ง
- แก้ไขบทความใด ๆ
- จัดการหมวดหมู่

### ผู้ดูแลระบบ
- การควบคุมโมดูลเต็มรูปแบบ
- กำหนดการตั้งค่า
- จัดการสิทธิ์

## การเขียนบทความ

### บรรณาธิการบทความ
```
┌─────────────────────────────────────────────────────┐
│ Title: [Your Article Title                        ] │
├─────────────────────────────────────────────────────┤
│ Category: [Select Category          ▼]              │
├─────────────────────────────────────────────────────┤
│ Summary:                                            │
│ ┌─────────────────────────────────────────────────┐ │
│ │ Brief description shown in listings...          │ │
│ └─────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────┤
│ Body:                                               │
│ ┌─────────────────────────────────────────────────┐ │
│ │ [B] [I] [U] [Link] [Image] [Code]               │ │
│ ├─────────────────────────────────────────────────┤ │
│ │                                                  │ │
│ │ Full article content goes here...               │ │
│ │                                                  │ │
│ └─────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────┤
│ [Submit] [Preview] [Save Draft]                     │
└─────────────────────────────────────────────────────┘
```
### แนวทางปฏิบัติที่ดีที่สุด

1. **ชื่อที่น่าสนใจ** - พาดหัวข่าวที่ชัดเจนและน่าดึงดูด
2. **สรุปดี** - ดึงดูดผู้อ่านให้คลิก
3. **เนื้อหาที่มีโครงสร้าง** - ใช้ส่วนหัว รายการ รูปภาพ
4. **การจัดหมวดหมู่ที่เหมาะสม** - ช่วยให้ผู้อ่านค้นหาเนื้อหา
5. **SEO การเพิ่มประสิทธิภาพ** - คำหลักในชื่อและเนื้อหา

## การจัดการเนื้อหา

### กระแสสถานะบทความ
```
mermaid
stateDiagram-v2
    [*] --> Draft: Save Draft
    Draft --> Submitted: Submit
    Submitted --> Published: Approve
    Submitted --> Rejected: Reject
    Rejected --> Draft: Revise
    Published --> Archived: Archive
    Published --> Draft: Unpublish
    Archived --> Published: Restore
```
### คำอธิบายสถานะ

| สถานะ | คำอธิบาย |
|--------|-------------|
| ร่าง | อยู่ระหว่างดำเนินการ |
| ส่งแล้ว | รอรีวิว |
| เผยแพร่แล้ว | ถ่ายทอดสดบนเว็บไซต์ |
| หมดอายุแล้ว | วันหมดอายุที่ผ่านมา |
| ถูกปฏิเสธ | ต้องการการแก้ไข |
| เก็บถาวร | ลบออกจากรายการ |

## การนำทาง

### กำลังเข้าถึงผู้เผยแพร่

- **เมนูหลัก** → ผู้จัดพิมพ์
- **โดยตรง URL**: `yoursite.com/modules/publisher/`

### หน้าสำคัญ

| หน้า | URL | วัตถุประสงค์ |
|------|-----|---------|
| ดัชนี | `/modules/publisher/` | รายการบทความ |
| หมวดหมู่ | `/modules/publisher/category.php?id=X` | หมวดหมู่บทความ |
| บทความ | `/modules/publisher/item.php?itemid=X` | บทความเดียว |
| ส่ง | `/modules/publisher/submit.php` | บทความใหม่ |
| ค้นหา | `/modules/publisher/search.php` | ค้นหาบทความ |
