---
title: "phpinfo.php"
---
ขั้นตอนนี้เป็นทางเลือก แต่ช่วยให้คุณประหยัดเวลาหงุดหงิดได้หลายชั่วโมง

จากการทดสอบระบบโฮสติ้งก่อนการติดตั้ง สคริปต์ PHP ที่มีขนาดเล็กมากแต่มีประโยชน์จะถูกสร้างขึ้นในเครื่อง และอัปโหลดไปยังระบบเป้าหมาย

สคริปต์ PHP มีเพียงบรรทัดเดียว:
```php
<?php phpinfo();
```
ใช้โปรแกรมแก้ไขข้อความสร้างไฟล์ชื่อ _info.php_ โดยมีบรรทัดเดียว

จากนั้น อัปโหลดไฟล์นี้ไปยังรูทเว็บของคุณ

![Filezilla info.php อัปโหลด](/xoops-docs/2.7/img/installation/filezilla-01-info.png)

เข้าถึงสคริปต์ของคุณโดยเปิดสคริปต์ในเบราว์เซอร์ของคุณ เช่น เข้าถึง `http://example.com/info.php` หากทุกอย่างทำงานอย่างถูกต้อง คุณจะเห็นหน้าเว็บในลักษณะนี้:

![phpinfo() ตัวอย่าง](/xoops-docs/2.7/img/installation/php-info.png)

หมายเหตุ: บริการโฮสติ้งบางแห่งอาจปิดใช้งานฟังก์ชัน _phpinfo()_ เพื่อเป็นมาตรการรักษาความปลอดภัย โดยปกติแล้วคุณจะได้รับข้อความแจ้งถึงผลกระทบดังกล่าว หากเป็นเช่นนั้น

ผลลัพธ์ของสคริปต์อาจมีประโยชน์สำหรับการแก้ไขปัญหา ดังนั้นให้พิจารณาบันทึกสำเนาไว้

หากการทดสอบใช้งานได้ คุณควรทำการติดตั้งให้ดี คุณควรลบสคริปต์ _info.php_ และดำเนินการติดตั้งต่อ

หากการทดสอบล้มเหลว ให้ตรวจสอบสาเหตุ! ไม่ว่าปัญหาใดก็ตามที่ทำให้การทดสอบง่ายๆ นี้ไม่ทำงาน **จะ** ป้องกันไม่ให้การติดตั้งจริงทำงานได้
