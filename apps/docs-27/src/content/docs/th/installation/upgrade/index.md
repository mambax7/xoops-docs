ส่วนนี้จะแนะนำคุณเกี่ยวกับการอัปเกรดไซต์ XOOPS ที่มีอยู่ วิซาร์ดการอัพเกรดจะรักษาข้อมูล โมดูล และธีมของคุณในขณะที่อัพเดตไฟล์หลัก

## เส้นทางการอัพเกรด

| ขั้นตอน | หน้า | จะทำอย่างไร |
|------|------|------------|
| 1 | [การเตรียมการ](./step-01/) | สำรองข้อมูล ตรวจสอบบันทึกการเปลี่ยนแปลง ดาวน์โหลดรุ่นใหม่ |
| 2 | [ตรวจสอบก่อนเที่ยวบิน](./preflight/) | ตรวจสอบความเข้ากันได้ก่อนที่จะรันการอัพเกรด |
| 3 | [เรียกใช้การอัปเกรด](./step-02/) | ดำเนินการตัวช่วยสร้างการอัพเกรด |
| 4 | [การแก้ไขปัญหา](./step-03/) | แก้ไขปัญหาการอัพเกรดทั่วไป |
| 5 | [หลังจากอัปเกรด](./step-04/) | ตรวจสอบ อัปเดตโมดูล ล้างแคช |

:::คำแนะนำ
ทำ **การสำรองฐานข้อมูลเต็มรูปแบบและสำรองไฟล์** ก่อนอัปเกรดเสมอ การอัปเกรดไม่สามารถย้อนกลับได้หากไม่มีการสำรองข้อมูล
::::::
