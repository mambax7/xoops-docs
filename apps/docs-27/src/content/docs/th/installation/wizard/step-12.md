หน้านี้รวบรวมตัวเลือกของคุณสำหรับธีมเริ่มต้นสำหรับไซต์ใหม่ของคุณ ธีมควบคุมลักษณะภาพของไซต์ของคุณ ธีมอาจมีการเปลี่ยนแปลงได้ตลอดเวลาหากต้องการ

หลังจากป้อนข้อมูลที่ร้องขอและแก้ไขปัญหาใด ๆ แล้ว ให้เลือกปุ่ม "ดำเนินการต่อ" เพื่อดำเนินการต่อ

![XOOPS ตัวติดตั้งเลือกธีม](/xoops-docs/2.7/img/installation/installer-12.png)

## ข้อมูลที่รวบรวมในขั้นตอนนี้

### เลือกธีมเริ่มต้น

#### ธีมเริ่มต้น

เลือกธีมเริ่มต้นสำหรับไซต์ของคุณจากรายการ ภาพหน้าจอของแต่ละธีมจะแสดงขึ้นเมื่อมีการเลือก

XOOPS 2.7.0 มาพร้อมกับธีมส่วนหน้าหลายแบบตั้งแต่แกะกล่อง:

* **ค่าเริ่มต้น** — ธีมพื้นฐานขั้นต่ำ ซึ่งมีประโยชน์เป็นจุดเริ่มต้นสำหรับการปรับแต่ง
* **xbootstrap5** — ธีม Bootstrap 5 แนะนำสำหรับไซต์ใหม่ส่วนใหญ่
* **xswatch5** — Bootswatch 5 รุ่น; ชุดของธีมย่อยที่มีสไตล์ที่ด้านบนของ Bootstrap 5
* **xtailwind** — ธีมที่ใช้ Tailwind CSS + DaisyUI พร้อม 35 จานสีและการโต้ตอบ Alpine.js
* **xtailwind2** — ธีมพี่น้องที่กำกับศิลป์พร้อมพาเล็ตที่ได้รับการดูแลจัดการและสไตล์โมดูลที่ได้รับการปรับปรุง

คุณสามารถเปลี่ยนธีมได้ตลอดเวลาจากพื้นที่การดูแลระบบ และสามารถอัปโหลดธีมเพิ่มเติมไปยังไดเร็กทอรี `themes/` ได้
