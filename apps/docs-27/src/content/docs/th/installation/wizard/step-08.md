title: "การสร้างตาราง"
---
หน้านี้แสดงผลลัพธ์ของการสร้างตาราง XOOPS ในฐานข้อมูล

หลังจากตรวจสอบและแก้ไขปัญหาแล้ว ให้เลือกปุ่ม "ดำเนินการต่อ" เพื่อดำเนินการต่อ

## ในความสำเร็จ

ส่วน _การสร้างตารางฐานข้อมูล_ จะแสดงตารางฐานข้อมูลที่ถูกสร้างขึ้น

![XOOPS การสร้างตารางตัวติดตั้ง](/xoops-docs/2.7/img/installation/installer-08.png)

## ข้อผิดพลาด

ข้อผิดพลาดใดๆ บ่งบอกถึงปัญหาเกี่ยวกับการอนุญาตที่เกี่ยวข้องกับบัญชีผู้ใช้ฐานข้อมูล MySQL และ/หรือฐานข้อมูล MySQL ที่ป้อนในขั้นตอนก่อนหน้า คุณสามารถกลับไปยังส่วนเหล่านั้นได้โดยใช้ตัวเลือกการเชื่อมต่อฐานข้อมูลและการกำหนดค่าฐานข้อมูลในเมนู
