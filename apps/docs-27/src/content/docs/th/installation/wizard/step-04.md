หน้านี้รวบรวมข้อมูลเกี่ยวกับระบบไฟล์และที่อยู่เว็บที่ XOOPS จำเป็นต้องดำเนินการ โปรแกรมติดตั้งจะพยายามเดาข้อมูลนี้ตาม URL ที่คุณใช้ในการเข้าถึง และตำแหน่งของสคริปต์ PHP ที่ทำงานอยู่

หลังจากตรวจสอบและแก้ไขปัญหาแล้ว ให้เลือกปุ่ม "ดำเนินการต่อ" เพื่อดำเนินการต่อ

![XOOPS การตั้งค่าเส้นทางตัวติดตั้ง](/xoops-docs/2.7/img/installation/installer-04-02.png)

## ข้อมูลที่รวบรวมในขั้นตอนนี้

### XOOPS เส้นทางทางกายภาพ

#### XOOPS เอกสารรูทฟิสิคัลพาธ

พาธฟิสิคัลไปยังไดเรกทอรีเอกสาร XOOPS (ให้บริการ) WITHOUT เครื่องหมายทับต่อท้าย

#### XOOPS ไดเรกทอรีไฟล์ข้อมูล

พาธฟิสิคัลไปยังไดเร็กทอรีไฟล์ข้อมูล XOOPS (เขียนได้) WITHOUT เครื่องหมายทับต่อท้าย ค้นหาโฟลเดอร์จากรูทเอกสาร XOOPS เพื่อให้มีความปลอดภัย

#### XOOPS ไดเรกทอรีไลบรารี

พาธฟิสิคัลไปยังไดเร็กทอรีไลบรารี XOOPS WITHOUT เครื่องหมายทับต่อท้าย ค้นหาโฟลเดอร์จากรูทเอกสาร XOOPS เพื่อให้มีความปลอดภัย

### ที่ตั้งเว็บ

#### ที่ตั้งเว็บไซต์ (URL)

URL หลักที่จะใช้ในการเข้าถึงการติดตั้ง XOOPS ของคุณ XOOPS จะใช้สิ่งนี้เพื่อสร้าง URL ทั้งหมดภายใน XOOPS หากคุณต้องการไซต์ _https_ อย่าลืมระบุไซต์นั้น หากคุณต้องการ/ไม่ต้องการ _www._ นำหน้าบนโดเมนไซต์ของคุณ อย่าลืมระบุตามที่คุณต้องการ

ซึ่งจะใช้ค่าเริ่มต้นเป็น URL ที่ใช้ในการเรียกใช้โปรแกรมติดตั้ง

#### โดเมนคุกกี้สำหรับเว็บไซต์

โดเมนสำหรับตั้งค่าคุกกี้ อาจว่างเปล่า โดยเป็นโฮสต์ทั้งหมดจาก URL (www.example.com¤) หรือโดเมนที่จดทะเบียนโดยไม่มีโดเมนย่อย (example.com) ที่จะแชร์ข้ามโดเมนย่อย (www.example.com และ blog.example.com)

## ข้อผิดพลาด

หากไม่พบเส้นทางที่ป้อน ข้อผิดพลาดจะปรากฏขึ้น แก้ไขปัญหาใด ๆ ก่อนดำเนินการต่อ

![XOOPS ข้อผิดพลาดการตั้งค่าเส้นทางตัวติดตั้ง](/xoops-docs/2.7/img/installation/installer-04-01.png)

## ช่วยด้วย

คุณสามารถดูคำอธิบายเพิ่มเติมได้ในระหว่างขั้นตอนการติดตั้ง การเลือกไอคอน "เครื่องช่วยชีวิต" ที่มุมด้านบนของหน้าจะเป็นการสลับการแสดงคำอธิบายเพิ่มเติม
