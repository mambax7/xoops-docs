# ایجاد اولین صفحه خود در XOOPS

با نحوه ایجاد، قالب بندی و انتشار اولین محتوای خود در XOOPS آشنا شوید.

## درک محتوای XOOPS

### Page/Post چیست؟

در XOOPS، محتوا از طریق ماژول ها مدیریت می شود. رایج ترین انواع محتوا عبارتند از:

| نوع | توضیحات | مورد استفاده |
|---|---|---|
| **صفحه** | محتوای ثابت | درباره ما، تماس، خدمات |
| **Post/Article** | محتوای مهر زمانی | اخبار، پست های وبلاگ |
| **دسته** | سازماندهی محتوا | مطالب مرتبط با گروه |
| **نظر** | بازخورد کاربر | اجازه تعامل با بازدید کننده |

این راهنما ایجاد یک page/article اساسی با استفاده از ماژول محتوای پیش فرض XOOPS را پوشش می دهد.

## دسترسی به ویرایشگر محتوا

### از پنل مدیریت

1. وارد پنل مدیریت شوید: `http://your-domain.com/xoops/admin/`
2. به **Content > Pages** (یا ماژول محتوای خود) بروید
3. روی «افزودن صفحه جدید» یا «پست جدید» کلیک کنید

### Frontend (در صورت فعال بودن)

اگر XOOPS شما به گونه ای پیکربندی شده است که اجازه ایجاد محتوای فرانت اند را بدهد:

1. به عنوان کاربر ثبت نام شده وارد شوید
2. به پروفایل خود بروید
3. به دنبال گزینه "Submit Content" باشید
4. همان مراحل زیر را دنبال کنید

## رابط ویرایشگر محتوا

ویرایشگر محتوا شامل:

```
┌─────────────────────────────────────┐
│ Content Editor                      │
├─────────────────────────────────────┤
│                                     │
│ Title: [________________]           │
│                                     │
│ Category: [Dropdown]                │
│                                     │
│ [B I U] [Link] [Image] [Video]    │
│ ┌─────────────────────────────────┐ │
│ │ Enter your content here...      │ │
│ │                                 │ │
│ │ You can use HTML tags here      │ │
│ └─────────────────────────────────┘ │
│                                     │
│ Description (Meta): [____________]  │
│                                     │
│ [Publish] [Save Draft] [Preview]   │
│                                     │
└─────────────────────────────────────┘
```

## راهنمای گام به گام: ایجاد اولین صفحه شما

### مرحله 1: به ویرایشگر محتوا دسترسی پیدا کنید

1. در پنل مدیریت، روی **Content > Pages** کلیک کنید.
2. روی **"افزودن صفحه جدید"** یا **"ایجاد"** کلیک کنید
3. ویرایشگر محتوا را خواهید دید

### مرحله 2: عنوان صفحه را وارد کنید

در قسمت "عنوان"، نام صفحه خود را وارد کنید:

```
Title: Welcome to Our Website
```

بهترین روش ها برای عناوین:
- واضح و توصیفی
- در صورت امکان کلمات کلیدی را وارد کنید
- 50-60 کاراکتر ایده آل
- از تمام حروف (سخت خواندن) اجتناب کنید
- خاص باشید (نه "صفحه 1")

### مرحله 3: دسته را انتخاب کنید

محل سازماندهی این محتوا را انتخاب کنید:

```
Category: [Dropdown ▼]
```

گزینه ها ممکن است شامل موارد زیر باشد:
- ژنرال
- اخبار
- وبلاگ
- اطلاعیه ها
- خدمات

اگر دسته‌ها وجود ندارند، از مدیر بخواهید آنها را ایجاد کند.

### مرحله 4: محتوای خود را بنویسید

در قسمت ویرایشگر محتوا کلیک کنید و متن خود را تایپ کنید.

#### قالب بندی متن پایه

از نوار ابزار ویرایشگر استفاده کنید:

| دکمه | اقدام | نتیجه |
|---|---|---|
| **ب** | پررنگ | **متن پررنگ** |
| *من* | کج | *متن ایتالیک* |
| <u>U</u> | زیر خط بکشید | <u>متن خط دار</u> |

#### استفاده از HTML

XOOPS به تگ های HTML ایمن اجازه می دهد. مثال های رایج:

```html
<!-- Paragraphs -->
<p>This is a paragraph.</p>

<!-- Headings -->
<h1>Main Heading</h1>
<h2>Subheading</h2>

<!-- Lists -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<!-- Bold and Italic -->
<strong>Bold text</strong>
<em>Italic text</em>

<!-- Links -->
<a href="https://example.com">Link text</a>

<!-- Line breaks -->
<br>

<!-- Horizontal rule -->
<hr>
```

#### نمونه های HTML ایمن

**برچسب های پیشنهادی:**
- پاراگراف ها: `<p>`، `<br>`
- سرفصل ها: `<h1>` تا `<h6>`
- متن: `<strong>`، `<em>`، `<u>`
- لیست ها: `<ul>`، `<ol>`، `<li>`
- پیوندها: `<a href="">`
- نقل قول های بلوک: `<blockquote>`
- جداول: `<table>`، `<tr>`، `<td>`

**از این برچسب ها اجتناب کنید** (ممکن است برای امنیت غیرفعال شود):
- اسکریپت ها: `<script>`
- سبک ها: `<style>`
- Iframes: `<iframe>` (مگر اینکه پیکربندی شده باشد)
- فرم ها: `<form>`، `<input>`

### مرحله 5: اضافه کردن تصاویر

#### گزینه 1: URL تصویر را درج کنید

با استفاده از ویرایشگر:

1. روی دکمه **درج تصویر** کلیک کنید (نماد تصویر)
2. آدرس تصویر را وارد کنید: `https://example.com/image.jpg`
3. متن جایگزین را وارد کنید: "توضیحات تصویر"
4. روی «درج» کلیک کنید

معادل HTML:

```html
<img src="https://example.com/image.jpg" alt="Description">
```

#### گزینه 2: آپلود تصویر

1. ابتدا تصویر را در XOOPS آپلود کنید:
   - به **Content > Media Manager** بروید
   - تصویر خود را آپلود کنید
   - آدرس تصویر را کپی کنید

2. در ویرایشگر محتوا، با استفاده از URL درج کنید (مراحل بالا)

#### بهترین تمرینات تصویر

- استفاده از اندازه فایل مناسب (بهینه سازی تصاویر)
- از نام فایل های توصیفی استفاده کنید
- همیشه متن جایگزین داشته باشید (دسترسی)
- فرمت های پشتیبانی شده: JPG، PNG، GIF، WebP
- عرض پیشنهادی: 600-800 پیکسل برای محتوا

### مرحله 6: جاسازی رسانه

#### ویدیو را از یوتیوب جاسازی کنید

```html
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  frameborder="0" allowfullscreen>
</iframe>
```

شناسه ویدیوی YouTube را جایگزین `VIDEO_ID` کنید.

**برای یافتن شناسه ویدیوی YouTube:**
1. ویدیو را در یوتیوب باز کنید
2. آدرس اینترنتی: `https://www.youtube.com/watch?v=VIDEO_ID` است
3. شناسه (نویسه های بعد از `v=`) را کپی کنید

#### ویدیو را از Vimeo جاسازی کنید
```html
<iframe src="https://player.vimeo.com/video/VIDEO_ID"
  width="640" height="360" frameborder="0"
  allow="autoplay; fullscreen" allowfullscreen>
</iframe>
```

### مرحله 7: توضیحات متا را اضافه کنید

در قسمت "توضیحات" یک خلاصه مختصر اضافه کنید:

```
Description: Learn how to get started with our website.
This page provides an overview of our services and how we can help you.
```

**بهترین شیوه های توضیحات متا:**
- 150-160 کاراکتر
- شامل کلمات کلیدی اصلی
- باید مطالب را به طور دقیق خلاصه کند
- در نتایج موتورهای جستجو استفاده می شود
- آن را جذاب کنید (کاربران این را می بینند)

### مرحله 8: تنظیمات انتشار را پیکربندی کنید

#### وضعیت انتشار

انتخاب وضعیت انتشار:

```
Status: ☑ Published
```

گزینه ها:
- **منتشر شده:** قابل مشاهده برای عموم
- **پیش نویس:** فقط برای مدیران قابل مشاهده است
- ** در انتظار بررسی: ** در انتظار تایید
- **بایگانی شده:** پنهان اما نگهداری می شود

#### دید

تعیین کنید چه کسانی می توانند این محتوا را ببینند:

```
Visibility: ☐ Public
           ☐ Registered Users Only
           ☐ Private (Admin Only)
```

#### تاریخ انتشار

زمانی را تنظیم کنید که محتوا قابل مشاهده شود:

```
Publish Date: [Date Picker] [Time]
```

برای انتشار فوراً به عنوان "اکنون" بگذارید.

#### اجازه دادن به نظرات

فعال یا غیرفعال کردن نظرات بازدیدکنندگان:

```
Allow Comments: ☑ Yes
```

اگر فعال باشد، بازدیدکنندگان می توانند بازخورد اضافه کنند.

### مرحله 9: محتوای خود را ذخیره کنید

چندین گزینه ذخیره:

```
[Publish Now]  [Save as Draft]  [Schedule]  [Preview]
```

- **اکنون منتشر کنید:** بلافاصله قابل مشاهده شوید
- **ذخیره به عنوان پیش نویس:** فعلا خصوصی نگه دارید
- **برنامه:** انتشار در آینده date/time
- **پیش نمایش:** قبل از ذخیره، ببینید چگونه به نظر می رسد

بر انتخاب خود کلیک کنید:

```
Click [Publish Now]
```

### مرحله 10: صفحه خود را تأیید کنید

پس از انتشار، محتوای خود را تأیید کنید:

1. به صفحه اصلی وب سایت خود بروید
2. به قسمت محتوای خود بروید
3. به دنبال صفحه تازه ایجاد شده خود بگردید
4. برای مشاهده کلیک کنید
5. بررسی کنید:
   - [ ] محتوا به درستی نمایش داده می شود
   - [ ] تصاویر ظاهر می شوند
   - [ ] قالب بندی خوب به نظر می رسد
   - [ ] پیوندها کار می کنند
   - [ ] عنوان و توضیحات درست است

## بهترین روش های سئو برای محتوا

محتوای خود را برای موتورهای جستجو بهینه کنید:

### عنوان
- شامل کلمه کلیدی اصلی
- 50-60 کاراکتر
- منحصر به فرد در هر صفحه

### توضیحات متا
- کلمه کلیدی را به طور طبیعی وارد کنید
- 150-160 کاراکتر
- قانع کننده و دقیق

### مطالب
- طبیعی بنویسید، از پر کردن کلمات کلیدی خودداری کنید
- از عناوین (h2, h3) به طور مناسب استفاده کنید
- شامل پیوندهای داخلی به صفحات دیگر
- از متن جایگزین در تمام تصاویر استفاده کنید
- بیش از 300 کلمه برای مقالات هدف گذاری کنید

### ساختار URL
- URL ها را کوتاه و توصیفی نگه دارید
- از خط فاصله برای جدا کردن کلمات استفاده کنید
- از شخصیت های خاص خودداری کنید
- مثال: `/about-our-company`

## مدیریت محتوای خود

### صفحه موجود را ویرایش کنید

1. به **محتوا > صفحات** بروید
2. صفحه خود را در لیست پیدا کنید
3. روی **ویرایش** یا عنوان صفحه کلیک کنید
4. ایجاد تغییرات
5. روی **به روز رسانی** کلیک کنید

### حذف صفحه

1. به **محتوا > صفحات** بروید
2. صفحه خود را پیدا کنید
3. روی **Delete** کلیک کنید
4. حذف را تایید کنید

### تغییر وضعیت انتشار

1. به **محتوا > صفحات** بروید
2. صفحه را پیدا کنید، روی **ویرایش** کلیک کنید
3. تغییر وضعیت در کشویی
4. روی **به روز رسانی** کلیک کنید

## عیب یابی ایجاد محتوا

### محتوا ظاهر نمی شود

** علامت: ** صفحه منتشر شده در وب سایت نشان داده نمی شود

**راه حل:**
1. بررسی وضعیت انتشار: باید "منتشر شده" باشد
2. تاریخ انتشار را بررسی کنید: باید فعلی یا گذشته باشد
3. قابلیت مشاهده را بررسی کنید: باید "عمومی" باشد
4. پاک کردن کش: Admin > Tools > Clear Cache
5. بررسی مجوزها: گروه کاربر باید دسترسی داشته باشد

### قالب بندی کار نمی کند

**علامت:** برچسب ها یا قالب بندی HTML به صورت متن ظاهر می شوند

**راه حل:**
1. بررسی کنید که HTML در تنظیمات ماژول فعال باشد
2. از نحو مناسب HTML استفاده کنید
3. بستن همه برچسب ها: `<p>Text</p>`
4. فقط از برچسب های مجاز استفاده کنید
5. از موجودیت های HTML استفاده کنید: `&lt;` برای `<`، `&amp;` برای `&`

### تصاویر نمایش داده نمی شوند

**علامت:** تصاویر نماد شکسته را نشان می دهند**راه حل:**
1. تأیید کنید URL تصویر درست است
2. بررسی کنید فایل تصویر وجود دارد
3. مجوزهای مناسب روی تصویر را بررسی کنید
4. به جای آن، تصویر را در XOOPS آپلود کنید
5. مسدود کردن خارجی را بررسی کنید (ممکن است به CORS نیاز باشد)

### مشکلات رمزگذاری کاراکتر

**علائم:** کاراکترهای خاص به صورت بیهوده ظاهر می شوند

**راه حل:**
1. فایل را به عنوان رمزگذاری UTF-8 ذخیره کنید
2. مطمئن شوید که مجموعه نویسه صفحه UTF-8 است
3. اضافه کردن به سر HTML: `<meta charset="UTF-8">`
4. از کپی پیست کردن از ورد خودداری کنید (از متن ساده استفاده کنید)

## بهترین شیوه های گردش کار محتوا

### فرآیند توصیه شده

1. **ابتدا در ویرایشگر بنویسید:** از ویرایشگر محتوای مدیریت استفاده کنید
2. **پیش نمایش قبل از انتشار:** روی دکمه پیش نمایش کلیک کنید
3. **افزودن فراداده:** عنوان کامل، توضیحات، برچسب ها
4. **اول ذخیره به عنوان پیش نویس:** ذخیره به عنوان پیش نویس برای جلوگیری از از دست دادن کار
5. **بررسی نهایی:** قبل از انتشار دوباره بخوانید
6. **انتشار:** پس از آماده شدن روی انتشار کلیک کنید
7. **تأیید:** در سایت زنده بررسی کنید
8. **ویرایش در صورت نیاز:** اصلاحات را به سرعت انجام دهید

### کنترل نسخه

همیشه پشتیبان تهیه کنید:

1. **قبل از تغییرات عمده:** به عنوان نسخه جدید یا نسخه پشتیبان ذخیره کنید
2. **محتوای قدیمی را بایگانی کنید:** نسخه های منتشر نشده را نگه دارید
3. **تاریخ پیش‌نویس‌ها:** از نام‌گذاری واضح استفاده کنید: "Page-Draft-2025-01-28"

## انتشار چندین صفحه

یک استراتژی محتوا ایجاد کنید:

```
Homepage
├── About Us
├── Services
│   ├── Service 1
│   ├── Service 2
│   └── Service 3
├── Blog
│   ├── Article 1
│   ├── Article 2
│   └── Article 3
├── Contact
```
