## الزامات

| جزء | حداقل | توصیه شده |
|------------|------------------------|---------------|
| PHP | 8.2 | 8.4+ |
| MySQL | 5.7 | 8.0+ |
| MariaDB | 10.4 | 10.11+ |
| وب سرور | Apache 2.4 / Nginx 1.20 | جدیدترین پایدار |

## دانلود کنید

آخرین نسخه را از [نسخه های GitHub](https://github.com/XOOPS/XoopsCore27/releases) دانلود کنید.

```bash
# Or clone directly
git clone https://github.com/XOOPS/XoopsCore27.git mysite
cd mysite
```

## مراحل نصب

1. **فایل ها** را در ریشه سند وب سرور خود آپلود کنید (به عنوان مثال `public_html/`).
2. **یک پایگاه داده MySQL** و یک کاربر با امتیازات کامل روی آن ایجاد کنید.
3. **مرورگر خود را باز کنید** و به دامنه خود بروید — نصب کننده XOOPS به طور خودکار شروع می شود.
4. **جادوگر 5 مرحله ای** را دنبال کنید - مسیرها را پیکربندی می کند، جداول ایجاد می کند و حساب مدیریت شما را تنظیم می کند.
5. **در صورت درخواست، پوشه `install/` را حذف کنید.** این برای امنیت اجباری است.

## نصب را تأیید کنید

پس از راه اندازی به آدرس زیر مراجعه کنید:

- **صفحه اول:** `https://yourdomain.com/`
- **پنل مدیریت:** `https://yourdomain.com/xoops_data/` *(مسیری که هنگام نصب انتخاب کردید)*
