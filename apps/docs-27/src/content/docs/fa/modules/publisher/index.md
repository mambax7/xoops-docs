## بررسی اجمالی

Publisher ماژول مدیریت محتوای قطعی برای XOOPS است که از SmartSection تکامل یافته و به غنی ترین راه حل وبلاگ و اخبار تبدیل شده است. ابزارهای جامعی برای ایجاد، سازماندهی و انتشار محتوا با پشتیبانی کامل از گردش کار ویرایشی فراهم می کند.

**نیازها:**
- XOOPS 2.5.10+
- PHP 7.1+ (PHP 8.x توصیه می شود)

## 🚀 شروع سریع

### 1. نصب

```bash
# Download from GitHub
git clone https://github.com/XoopsModules25x/publisher.git

# Copy to modules directory
cp -r publisher /path/to/xoops/htdocs/modules/
```

سپس از طریق XOOPS Admin → Modules → Install نصب کنید.

### 2. اولین دسته خود را ایجاد کنید

1. به **Admin → Publisher → Categories** بروید
2. روی **افزودن دسته** کلیک کنید
3. پر کنید:
   - **نام**: اخبار
   - **توضیحات**: آخرین اخبار و به روز رسانی ها
   - **تصویر**: آپلود تصویر دسته
4. ذخیره کنید

### 3. اولین مقاله خود را ایجاد کنید

1. به **مدیر → ناشر → مقالات** بروید
2. روی **افزودن مقاله** کلیک کنید
3. پر کنید:
   - **عنوان**: به سایت ما خوش آمدید
   - **دسته**: اخبار
   - **محتوا**: محتوای مقاله شما
4. تنظیم **وضعیت**: منتشر شده است
5. ذخیره کنید

## 🔄 مهاجرت

### از SmartSection

ناشر شامل یک ابزار انتقال داخلی است:

1. به **Admin → Publisher → Import بروید**
2. **SmartSection** را به عنوان منبع انتخاب کنید
3. گزینه های واردات را انتخاب کنید:
   - دسته بندی ها
   - مقالات
   - نظرات
4. روی **وارد کردن** کلیک کنید

### از ماژول اخبار1. به **Admin → Publisher → Import بروید**
2. **اخبار** را به عنوان منبع انتخاب کنید
3. دسته بندی نقشه
4. روی **وارد کردن** کلیک کنید
