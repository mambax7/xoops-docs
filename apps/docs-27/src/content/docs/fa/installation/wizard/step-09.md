title: "تنظیمات اولیه"
---
این صفحه اطلاعات مورد استفاده برای ایجاد حساب کاربری اصلی Administrator (AKA Webmaster) را جمع آوری می کند.

پس از وارد کردن اطلاعات درخواستی و رفع اشکال، دکمه "ادامه" را برای ادامه انتخاب کنید.

![تنظیمات اولیه XOOPS Installer](/xoops-docs/2.7/img/installation/installer-09.png)

## داده های جمع آوری شده در این مرحله

### حساب مدیر

#### ورود ادمین

این نام کاربری است که توسط ادمین برای ورود به سیستم XOOPS استفاده می شود.

#### ایمیل ادمین

این آدرس ایمیلی است که سیستم XOOPS برای ارتباط با مدیر استفاده می کند. (توجه داشته باشید، این برای رمز عبور recovery/reset. مورد نیاز است)

#### رمز عبور مدیریت

این رمز عبور برای حساب مدیر است،

#### رمز عبور را تایید کنید

از این فیلد برای تایید رمز عبور مدیر استفاده کنید. باید با قسمت _Admin password_ مطابقت داشته باشد.

### قدرت رمز عبور

این بخش قدرت نسبی _Admin Password_ را نمایش می دهد.

#### تولید کننده رمز عبور

می توانید از این بخش برای ایجاد و استفاده از رمز عبور تصادفی برای حساب کاربری استفاده کنید. برای نمایش رمز عبور تصادفی، _Generate_ را انتخاب کنید. برای کپی کردن رمز عبور ایجاد شده به _Admin Password__Copy_ را انتخاب کنید.
