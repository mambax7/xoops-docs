---
title: "Impostazione percorso"
---

Questa pagina raccoglie le informazioni sul file system e l'indirizzo web di cui XOOPS ha bisogno per operare. Il programma di installazione cercherà di indovinare questa informazione in base all'URL che stai utilizzando per accedervi e alle posizioni degli script PHP in esecuzione.

Dopo aver rivisto e corretto eventuali problemi, seleziona il pulsante "Continua" per procedere.

![Impostazioni percorso programma di installazione XOOPS](/xoops-docs/2.7/img/installation/installer-04-02.png)

## Dati raccolti in questo passaggio

### Percorsi fisici XOOPS

#### Percorso fisico della radice dei documenti XOOPS

Percorso fisico della directory documenti XOOPS (servita) SENZA barra finale

#### Directory file dati XOOPS

Percorso fisico della directory file dati XOOPS (scrivibile) SENZA barra finale. Individua la cartella al di fuori della radice del documento XOOPS per renderla sicura.

#### Directory libreria XOOPS

Percorso fisico della directory della libreria XOOPS SENZA barra finale. Individua la cartella al di fuori della radice del documento XOOPS per renderla sicura.

### Posizioni web

#### Posizione del sito web (URL)

URL principale che verrà utilizzato per accedere alla tua installazione XOOPS. XOOPS lo userà per costruire tutti gli URL all'interno di XOOPS. Se desideri un sito _https_, assicurati di specificarlo. Se desideri/non desideri un _www._ principale nel tuo dominio del sito, assicurati di specificarlo come desideri.

Questo imposterà automaticamente l'URL utilizzato per avviare il programma di installazione.

#### Dominio cookie per il sito web

Dominio per impostare i cookie. Potrebbe essere vuoto, l'host completo dall'URL (www.example.com), o il dominio registrato senza sottodomini (example.com) per condividere tra sottodomini (www.example.com e blog.example.com.)

## Errori

Se uno dei percorsi inseriti non può essere trovato, verrà visualizzato un errore. Correggi i problemi prima di continuare.

![Errore impostazioni percorso programma di installazione XOOPS](/xoops-docs/2.7/img/installation/installer-04-01.png)
