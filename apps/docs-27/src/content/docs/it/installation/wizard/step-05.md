Questa pagina raccoglie le informazioni sull'account utente del database che XOOPS utilizzerà.

Dopo aver inserito le informazioni richieste e aver corretto eventuali problemi, seleziona il pulsante "Continua" per procedere.

![Connessione database programma di installazione XOOPS](/xoops-docs/2.7/img/installation/installer-05.png)

## Dati raccolti in questo passaggio

### Connessione server

#### Database

Attualmente, _mysql_ è l'unica opzione disponibile.

#### Nome host server

Nome host del server di database. Se non sei sicuro, _localhost_ funziona nella maggior parte dei casi.

#### Nome utente

L'account utente di database che XOOPS dovrebbe utilizzare per connettersi al server di database.

#### Password

Password dell'account utente di database inserito sopra.

#### Usa connessione persistente

L'impostazione predefinita è "No". Lascia questo vuoto se non sei sicuro.
