title: "Creazione tabelle"
---

Questa pagina visualizza i risultati della creazione delle tabelle XOOPS nel database.

Dopo aver rivisto e corretto eventuali problemi, seleziona il pulsante "Continua" per procedere.

## Successo

La sezione _Creazione delle tabelle del database_ mostra le tabelle del database che sono state create.

![Creazione tabelle del programma di installazione XOOPS](/xoops-docs/2.7/img/installation/installer-08.png)

## Errori

Qualsiasi errore indica problemi con autorizzazioni relative all'account utente di MySQL e/o al database MySQL inserito nei passaggi precedenti. Puoi tornare a quelle sezioni utilizzando le opzioni Connessione database e Configurazione database nel menu.
