---
title: "Verifica configurazione"
---

_Verifica della configurazione del tuo server_ è una revisione dell'ambiente in cui viene eseguito il programma di installazione.

Se vengono segnalati problemi nella sezione _Requisiti_, questi devono essere corretti, altrimenti XOOPS _non_ funzionerà.

La sezione _Estensioni consigliate_ indica estensioni PHP aggiuntive che XOOPS o i moduli XOOPS potrebbero utilizzare per fornire funzioni aggiuntive. Puoi procedere con l'installazione, anche se le estensioni non sono disponibili. Ma, se puoi abilitare questi, è consigliato.

Dopo aver rivisto, seleziona il pulsante "Continua" per procedere.
