Questa pagina raccoglie le informazioni utilizzate per creare l'account amministratore primario (AKA webmaster).

Dopo aver inserito le informazioni richieste e aver corretto eventuali problemi, seleziona il pulsante "Continua" per procedere.

![Impostazioni iniziali del programma di installazione XOOPS](/xoops-docs/2.7/img/installation/installer-09.png)

## Dati raccolti in questo passaggio

### Account amministratore

#### Accesso amministratore

Questo sarà il nome utente utilizzato dall'amministratore per accedere al sistema XOOPS.

#### Email amministratore

Questo è l'indirizzo email utilizzato dal sistema XOOPS per comunicare con l'amministratore. (Nota, questo è richiesto per il recupero/ripristino della password.)

#### Password amministratore

Questa è la password da utilizzare per l'account amministratore,

#### Conferma password

Usa questo campo per confermare la password dell'amministratore. Deve corrispondere al campo _Password amministratore_.

### Forza della password

Questa sezione visualizza la relativa resistenza della _Password amministratore_.

#### Generatore password

Puoi usare questa sezione per generare e utilizzare una password casuale per l'account. Seleziona _Genera_ per mostrare una password casuale. Seleziona _Copia_ per copiare la password generata in _Password amministratore_.
