---
title: "Pulizia"
---

Per motivi di sicurezza, dovresti rimuovere la directory _install_ dal tuo sito. Il programma di installazione avrà rinominato la directory in _installremove_ seguito da una stringa casuale lunga.

Puoi usare un client FTP, come FileZilla per eliminare la directory.
