Questa sezione ti guida attraverso l'aggiornamento di un sito XOOPS esistente. La procedura guidata di aggiornamento preserva i tuoi dati, moduli e temi mentre aggiorna i file core.

## Percorso di aggiornamento

| Passaggio | Pagina | Cosa fare |
|------|------|------------|
| 1 | [Preparazioni](./step-01/) | Backup, rivedi changelog, scarica nuovo rilascio |
| 2 | [Controllo di preflight](./preflight/) | Verifica la compatibilità prima di eseguire l'aggiornamento |
| 3 | [Esecuzione aggiornamento](./step-02/) | Esegui la procedura guidata di aggiornamento |
| 4 | [Risoluzione problemi](./step-03/) | Risolvi i problemi comuni di aggiornamento |
| 5 | [Dopo l'aggiornamento](./step-04/) | Verifica, aggiorna moduli, cancella cache |

:::tip
Esegui sempre un **backup completo del database e backup dei file** prima dell'aggiornamento. L'aggiornamento è irreversibile senza un backup.
:::
