title: "Appendix 4: Strumenti Utili"

| Software | Descrizione |
| --- | --- |
| 7Zip www.7zip.org | Programma Open Source per comprimere/decomprimere file. |
| WinSCP www.WinSCP.net | WinSCP è un client SFTP e FTP open source per Windows. La sua funzione principale è il trasferimento sicuro di file tra un computer locale e uno remoto. |
| WAMP Server www.wampserver.com | WampServer è un ambiente di sviluppo web per Windows. Ti permette di creare applicazioni web con Apache, PHP e il database MySQL. Viene anche fornito con PHPMyAdmin per gestire facilmente i tuoi database. WampServer si installa automaticamente (installer), e il suo utilizzo è molto intuitivo. Potrai sintonizzare il tuo server senza neanche toccare i file di impostazioni. |
| XAMPP  [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP è una distribuzione Apache facile da installare contenente MySQL, PHP e Perl. XAMPP è davvero molto facile da installare e usare - scarica, estrai e avvia. |
| KeyPass www.keepass.info | KeePass è un gestore di password open source gratuito, che ti aiuta a gestire le tue password in modo sicuro. Puoi mettere tutte le tue password in un database, che è bloccato con una chiave master o un file chiave. Quindi devi solo ricordare una singola master password o selezionare il file chiave per sbloccare l'intero database. I database sono crittografati utilizzando i migliori e più sicuri algoritmi di crittografia attualmente noti (AES e Twofish). |
| Package Web Tools [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === USB Apps ==== - 7zip, Filezilla, Kompozer, DataBase Browser, Opera, PicPick, Winmerge, Gimp, Notepad++, OikoCSSeditor, SubCommander, ToDoList, Toucan, Xenu, TeamViewer, Pidgin, FastImageOptimizer, VirtualWin e MiniServer (con ultimo Apache - MySQL - PHP, e Adminer per amministrare il database) |
| Package Web Browsers [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === USB Apps ==== - Firefox + webdevelpper e estensioni XooFoo (versione stabile e di sviluppo), IETester 0.4.5, Opera (versione stabile e di sviluppo), OperaMobile Emulator, Safari, Google Chrome + estensione webdevelpper (versione stabile e di sviluppo), Maxthon (versione stabile e di sviluppo), SeaMonkey, browser Avant, Flock e Elinks |
| Uniform Server [http://uniformserver.com](http://uniformserver.com) | "Perché usare Uniform Server al posto di XAMPP o WAMP o EASYPHP? 1. Sicurezza 2. Leggero (xampp 51MB, XAMPlite 28MB. Uniform Server 9.95MB). 3. Più facile da usare. 4. XAMPP (per esempio) ha molte funzioni non necessarie, specialmente per la comunità Xoops. Nonostante la piccola dimensione di Uniform Server, ha tutto ciò di cui abbiamo bisogno./ /L'installazione di Uniform Server ti darà, nelle ultime versioni: - Database MySQL. - phpMyAdmin per amministrare il database MySQL. - Hosting Apache. - PHP (Ha oltre 100 bugfix in più rispetto a quello che ottieni quando scarichi XAMPP). - eAccelerator per rendere il sito web più veloce. |
| Uniform Server by XooFoo [http://goo.gl/LGx3V](http://goo.gl/LGx3V) | Uniform Server ottimizzato per XOOPS |
| Mini Server (solo 5MB) by XooFoo [http://goo.gl/Ke0LW](http://goo.gl/Ke0LW) | Un server di produzione portatile di base con la potenza aggiunta di PHP che consente pagine web dinamiche in aggiunta a pagine htm e html statiche. Progettato per consentire l'esecuzione di più server sulla stessa PC o unità USB, questi richiederanno un'unità virtuale separata creata automaticamente. Il supporto di MySQL è abilitato, migliorando ulteriormente le capacità di questo server. |
