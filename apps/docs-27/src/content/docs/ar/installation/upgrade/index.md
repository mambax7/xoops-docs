يرشدك هذا القسم خلال ترقية موقع XOOPS الحالي. يحافظ معالج الترقية على البيانات والوحدات والمواضيع الخاصة بك مع تحديث ملفات النواة.

## مسار الترقية

| الخطوة | الصفحة | ما يجب فعله |
|------|------|------------|
| 1 | [التحضيرات](./step-01/) | النسخ الاحتياطي ومراجعة سجل التغييرات وتنزيل الإصدار الجديد |
| 2 | [فحص ما قبل الترقية](./preflight/) | التحقق من التوافقية قبل تشغيل الترقية |
| 3 | [تشغيل الترقية](./step-02/) | تنفيذ معالج الترقية |
| 4 | [استكشاف الأخطاء](./step-03/) | حل المشاكل الشائعة للترقية |
| 5 | [بعد الترقية](./step-04/) | التحقق وتحديث الوحدات ومسح الذاكرة المؤقتة |

:::tip
قم دائماً بعمل **نسخة احتياطية كاملة للقاعدة البيانات والملفات** قبل الترقية. الترقية غير قابلة للعكس بدون نسخة احتياطية.
:::
