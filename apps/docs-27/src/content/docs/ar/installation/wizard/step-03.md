_فحص تكوين الخادم الخاص بك_ هو مراجعة للبيئة التي يعمل فيها المثبت.

إذا تم الإبلاغ عن أي مشاكل في قسم _المتطلبات_، فيجب تصحيح هذه المشاكل، وإلا فإن XOOPS _لن_ يعمل.

يشير قسم _المحاولات الموصى بها_ إلى ملحقات PHP الإضافية التي قد تستخدمها XOOPS أو وحدات XOOPS لتوفير وظائف إضافية. يمكنك المتابعة مع التثبيت، حتى لو لم تكن الملحقات متاحة. لكن، إذا كان بإمكانك تمكين هذه، فهذا هو الأفضل.

بعد المراجعة، حدد زر "متابعة" للمتابعة.
