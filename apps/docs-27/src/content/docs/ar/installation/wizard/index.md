يوجهك معالج تثبيت XOOPS عبر عملية مكونة من 15 خطوة تقوم بتكوين قاعدة البيانات الخاصة بك وإنشاء حساب المسؤول وتحضير موقعك للاستخدام الأول.

## قبل البدء

- لقد قمت [بتحميل XOOPS على الخادم الخاص بك](/xoops-docs/2.7/installation/ftp-upload/) أو قمت بإعداد بيئة محلية
- لقد قمت [بالتحقق من المتطلبات](/xoops-docs/2.7/installation/requirements/)
- لديك بيانات قاعدة البيانات الخاصة بك جاهزة
