تعرض هذه الصفحة نتائج إنشاء جداول XOOPS في قاعدة البيانات.

بعد المراجعة وتصحيح أي مشاكل، حدد زر "متابعة" للمتابعة.

## عند النجاح

يعرض قسم _إنشاء جداول قاعدة البيانات_ جداول قاعدة البيانات التي تم إنشاؤها.

![إنشاء جدول المثبت XOOPS](/xoops-docs/2.7/img/installation/installer-08.png)

## أخطاء

تشير أي أخطاء إلى مشاكل في الأذونات تتعلق بحساب مستخدم MySQL و/أو قاعدة بيانات MySQL المدخلة في الخطوات السابقة. يمكنك العودة إلى تلك الأقسام باستخدام خيارات اتصال قاعدة البيانات وتكوين قاعدة البيانات في القائمة.
