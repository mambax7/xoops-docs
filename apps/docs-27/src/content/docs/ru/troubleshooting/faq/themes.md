# Часто задаваемые вопросы - Темы

> Ответы на распространенные вопросы по темам и оформлению XOOPS.

---

## Как установить тему?

1. Загрузите тему в /themes/
2. Перейдите в Admin → System Settings
3. Выберите тему из списка
4. Нажмите "Save"

---

## Где найти темы?

Официальные источники:
- XOOPS репозиторий
- GitHub репозитории
- Сторонние дизайн-сайты

---

## Как отредактировать тему?

1. Перейдите в /themes/mytheme/
2. Отредактируйте CSS и шаблоны
3. Сохраните файлы
4. Очистите кэш браузера

---

## Как создать свою тему?

1. Скопируйте существующую тему
2. Переименуйте папку
3. Отредактируйте style.css
4. Модифицируйте шаблоны
5. Установите в Admin

---

## Почему тема не отображается?

Причины:
1. CSS файлы не загружаются
2. Проблема с синтаксисом HTML
3. Конфликт стилей
4. Кэш браузера
