# Часто задаваемые вопросы - Модули

> Ответы на распространенные вопросы по модулям XOOPS.

---

## Какой модуль мне нужен?

Выбор зависит от ваших потребностей:
- **Publisher** - Блог, новости, статьи
- **News** - Новостная лента
- **Downloads** - Управление загрузками
- **Gallery** - Галереи фотографий

---

## Как обновить модуль?

1. Загрузите новую версию
2. Перезапишите файлы
3. Перейдите в Admin → Modules
4. Нажмите "Update" если доступно
5. Следуйте инструкциям

---

## Почему модуль не работает?

Причины:
1. Неправильная установка
2. Недостаточные разрешения на файлы
3. Зависимости не установлены
4. Конфликт с другим модулем

---

## Как удалить модуль?

1. Перейдите в Admin → Modules
2. Найдите модуль
3. Нажмите "Uninstall"
4. Подтвердите удаление
5. Удалите папку вручную
