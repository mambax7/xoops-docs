| Программное обеспечение | Описание |
| --- | --- |
| 7Zip www.7zip.org | Программа с открытым исходным кодом для сжатия/распаковки файлов. |
| WinSCP www.WinSCP.net | WinSCP - это клиент SFTP с открытым исходным кодом и FTP-клиент для Windows. Его основная функция - безопасная передача файлов между локальным и удалённым компьютером. |
| WAMP Server www.wampserver.com | WampServer - это среда разработки веб-приложений Windows. Она позволяет создавать веб-приложения с Apache, PHP и базой данных MySQL. Он также поставляется с PHPMyAdmin для легкого управления вашими БД. WampServer устанавливается автоматически (установщик) и очень интуитивен в использовании. Вы сможете настраивать свой сервер, не касаясь файлов параметров. |
| XAMPP  [http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html) | XAMPP - это простой дистрибутив Apache, содержащий MySQL, PHP и Perl. XAMPP очень легко устанавливается и использует - просто загрузите, распакуйте и запустите. |
| KeyPass www.keepass.info | KeePass - это бесплатный менеджер паролей с открытым исходным кодом, который помогает вам управлять пароли безопасным способом. Вы можете поместить все свои пароли в одну БД, которая заблокирована одним главным ключом или файлом ключа. Поэтому вам нужно только помнить один главный пароль или выбрать файл ключа для разблокировки всей БД. БД шифруются с использованием лучших и наиболее безопасных алгоритмов шифрования, известных в настоящее время (AES и Twofish). |
| Package Web Tools [http://goo.gl/FMaFk](http://goo.gl/FMaFk) | === USB приложения ==== - 7zip, Filezilla, Kompozer, Database Browser, Opera, PicPick, Winmerge, Gimp, Notepad++, OikoCSSeditor, SubCommander, ToDoList, Toucan, Xenu, TeamViewer, Pidgin, FastImageOptimizer, VirtualWin и MiniServer (с последним Apache - MySQL - PHP и Adminer для администрирования БД) |
| Package Web Browsers [http://goo.gl/UXg8U](http://goo.gl/UXg8U) | === USB приложения ==== - Firefox + расширения webdevelopper и XooFoo (стабильная и версия разработки), IETester 0.4.5, Opera (стабильная и версия разработки), OperaMobile Emulator, Safari, Google Chrome + расширение webdevelopper (стабильная и версия разработки), Maxthon (стабильная и версия разработки), SeaMonkey, Avant browser, Flock и Elinks |
| Uniform Server [http://uniformserver.com](http://uniformserver.com) | "Почему использовать Uniform Server вместо XAMPP или WAMP или EASYPHP? 1. Безопасность 2. Лёгкий вес (xampp 51МБ, XAMPlite 28МБ. Uniform Server 9,95МБ). 3. Легче в использовании. 4. XAMPP (например) имеет много ненужных функций, особенно для сообщества Xoops. Несмотря на небольшой размер Uniform Server, он имеет всё, что нам нужно./ /Установка Uniform Server даст вам в последние версии: - БД MySQL. - PHPMyAdmin для администрирования БД MySQL. - Хостинг Apache. - PHP (имеет более 100 исправлений ошибок больше, чем то, что вы получаете при загрузке XAMPP). - eAccelerator, чтобы ускорить работу веб-сайта. |
| Uniform Server от XooFoo [http://goo.gl/LGx3V](http://goo.gl/LGx3V) | Uniform Server оптимизирован для XOOPS |
| Mini Server (всего 5МБ) от XooFoo [http://goo.gl/Ke0LW](http://goo.gl/Ke0LW) | Базовый портативный производственный сервер с добавленной мощью PHP, позволяющей динамические веб-страницы в дополнение к статическим htm и html страницам. Разработан для запуска нескольких серверов на одном ПК или USB-дисках, они будут требовать отдельного виртуального диска, автоматически созданного. Поддержка MySQL включена, что дальше расширяет возможности этого сервера. |
