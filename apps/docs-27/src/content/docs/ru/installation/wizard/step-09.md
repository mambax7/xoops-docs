title: "Начальные настройки"

Эта страница собирает информацию, используемую для создания основной учетной записи администратора (также известной как Webmaster).

После ввода запрашиваемой информации и исправления любых проблем нажмите кнопку "Продолжить" для продолжения.

![XOOPS Installer Initial Settings](/xoops-docs/2.7/img/installation/installer-09.png)

## Данные, собираемые на этом этапе

### Учетная запись администратора

#### Логин администратора

Это будет имя пользователя, используемое администратором для входа в систему XOOPS.

#### Email администратора

Это адрес электронной почты, используемый системой XOOPS для связи с администратором. (Примечание: это необходимо для восстановления/сброса пароля.)

#### Пароль администратора

Это пароль, который будет использоваться для учетной записи администратора.

#### Подтверждение пароля

Используйте это поле для подтверждения пароля администратора. Оно должно совпадать с полем _Пароль администратора_.

### Надежность пароля

Этот раздел отображает относительную надежность _Пароля администратора_.

#### Генератор пароля

Вы можете использовать этот раздел для создания и использования случайного пароля для учетной записи. Нажмите _Создать_ для отображения случайного пароля. Нажмите _Копировать_ для копирования созданного пароля в _Пароль администратора_.
