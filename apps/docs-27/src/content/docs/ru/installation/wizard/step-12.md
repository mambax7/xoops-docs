Эта страница собирает ваш выбор для начальной темы вашего нового сайта. Темы определяют визуальный характер вашего сайта. Темы можно изменять в любое время, если захотите.

После ввода запрашиваемой информации и исправления любых проблем нажмите кнопку "Продолжить" для продолжения.

![XOOPS Installer Select Theme](/xoops-docs/2.7/img/installation/installer-12.png)

## Данные, собираемые на этом этапе

### Выберите тему по умолчанию

#### Тема по умолчанию

Выберите начальную тему для вашего сайта из списка. Снимок экрана каждой темы показывается при её выборе.

XOOPS 2.7.0 поставляется с несколькими фронтенд-темами прямо из коробки:

* **default** — минимальная базовая тема, полезная в качестве отправной точки для кастомизации.
* **xbootstrap5** — тема Bootstrap 5, рекомендуется для большинства новых сайтов.
* **xswatch5** — варианты Bootswatch 5; набор стилизованных подтем поверх Bootstrap 5.
* **xtailwind** — тема на основе Tailwind CSS + DaisyUI с 35 палитрами и интерактивностью Alpine.js
* **xtailwind2** — направляемая сестринская тема с кураторскими палитрами и уточненным стилем модулей

Вы можете изменить тему в любое время из области администрирования, и дополнительные темы можно загружать в каталог `themes/`.
