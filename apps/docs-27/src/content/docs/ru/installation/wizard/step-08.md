Эта страница показывает результаты создания таблиц XOOPS в базе данных.

После проверки и исправления любых проблем нажмите кнопку "Продолжить" для продолжения.

## При успехе

Раздел _Создание таблиц базы данных_ показывает таблицы БД, которые были созданы.

## Ошибки

Любые ошибки указывают на проблемы с разрешениями, связанные с учетной записью пользователя MySQL и/или базой данных MySQL, введенными на предыдущих этапах. Вы можете вернуться в эти разделы, используя опции Подключение к БД и Конфигурация БД в меню.
