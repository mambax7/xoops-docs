---
title: "Конфигурация сайта"
---

Эта страница собирает информацию, используемую для установки начального состояния вашего нового сайта XOOPS. Это включает такие вещи, как заголовки и значения HTML метатегов.

Это только начальные значения для вашего сайта. Параметры можно изменить в любое время в административных настройках вашего нового сайта XOOPS.

Вы можете захотеть использовать эти начальные параметры, чтобы ограничить то, что будет видно публично, пока ваш сайт полностью не будет настроен.

После ввода запрашиваемой информации и исправления любых проблем нажмите кнопку "Продолжить" для продолжения.

## Данные, собираемые на этом этапе

### Общие параметры

#### Имя сайта

Имя или заголовок вашего сайта.

#### Слоган сайта

Слоган или подзаголовок для вашего сайта.

#### Режим отладки

Это позволяет вам входить в ваш новый сайт с отображаемой информацией отладки. Хотя работающий веб-сайт должен иметь это отключено, это может быть очень полезно, если возникнут проблемы с установкой. _Включить отладку (встроенный режим)_ — это наиболее часто используемое значение.

#### Отключить ваш сайт?

Выберите _да_ для отключения вашего сайта. В таком состоянии сразу после установки только новая учетная запись администратора сможет получить доступ к сайту. Это позволит вам настроить ваш сайт в соответствии с вашими требованиями перед открытием общественного доступа.

### Метатеги и подвал

HTML метатеги содержат данные, описывающие сайт. Некоторые модули могут устанавливать метатеги для конкретной страницы. Значения тегов, введенные здесь, будут использоваться, если не переопределены значениями для конкретной страницы.

#### Ключевые слова метатега

Список ключевых слов, связанных с содержимым сайта. Это может сыграть роль в SEO, но используйте осторожно. "Спам" ключевыми словами может иметь отрицательный эффект на размещение в поисковых системах.

#### Описание метатега

Краткое описание вашего сайта.

#### Автор метатега

Автор сайта по умолчанию в виде имени, адреса электронной почты, названия компании или URL.

#### Авторское право метатега

Заявление об авторских правах для вашего сайта.

### Параметры информации о пользователе

#### Разрешить новые регистрации пользователей?

Выберите да для принятия новых регистраций пользователей или нет, если регистрация будет приватной.
