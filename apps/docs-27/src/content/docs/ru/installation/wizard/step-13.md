Эта страница собирает ваш выбор для модулей, которые нужно установить в вашем новом сайте. Модули можно добавлять или удалять в любое время, если захотите.

После ввода запрашиваемой информации и исправления любых проблем нажмите кнопку "Продолжить" для продолжения.

![XOOPS Installer Modules Installation](/xoops-docs/2.7/img/installation/installer-13.png)

## Данные, собираемые на этом этапе

### Доступные модули

Выберите "Да" для установки модуля. Вы можете не выбрать ни один, выбрать любые или все модули.

Это модули, включенные в стандартную установку XOOPS 2.7.0.

* Private Messaging — _расширенный обмен личными сообщениями между пользователями_
* User Profile — _расширенные/кастомные профили пользователя_
* Protector — _расширенная безопасность сайта, контроль спама и смягчение атак_
* DebugBar — _панель инструментов отладки для разработки и staging на основе Symfony VarDumper; полезна для разработчиков модулей и диагностики производственных проблем, но обычно не требуется на public-facing live сайте_

Модуль _System_ всегда устанавливается и здесь не указывается в качестве опции.

#### Примечание

Вы можете добавлять модули в каталог модулей при загрузке файлов XOOPS перед запуском установки, и они также будут доступны на этом этапе.

#### Сильная рекомендация

**Все сайты XOOPS** должны установить модуль _Protector_.
