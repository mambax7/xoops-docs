## Данные, собираемые на этом этапе

### База данных

#### Имя базы данных

Имя базы данных на хосте, которую должен использовать XOOPS. Пользователь БД, введенный на предыдущем этапе, должен иметь все права на эту базу данных. Установщик попытается создать эту базу данных, если она не существует.

#### Префикс таблиц

Этот префикс будет добавлен к именам всех новых таблиц, создаваемых XOOPS. Это помогает избежать конфликтов имен, если база данных используется совместно с другими приложениями. Уникальный префикс также затрудняет угадывание имен таблиц, что имеет преимущества для безопасности. Если вы не уверены, просто оставьте значение по умолчанию.

#### Кодировка символов БД

Установщик по умолчанию использует `utf8mb4`, который поддерживает полный диапазон Unicode, включая эмодзи и дополнительные символы. Вы можете выбрать здесь другую кодировку, но `utf8mb4` рекомендуется для практически всех языков и локалей и должна оставаться без изменений, если у вас нет определенной причины её менять.

#### Сопоставление символов БД

Поле сопоставления по умолчанию оставляется пустым. Когда пусто, MySQL применяет сопоставление по умолчанию для выбранной выше кодировки (для `utf8mb4` это обычно `utf8mb4_general_ci` или `utf8mb4_0900_ai_ci`, в зависимости от версии MySQL). Если вам нужно определенное сопоставление — например, для согласования с существующей базой данных — выберите его здесь. В противном случае рекомендуется оставить поле пустым.
