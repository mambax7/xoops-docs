Эта страница собирает информацию об учетной записи пользователя базы данных, которую будет использовать XOOPS.

После ввода запрашиваемой информации и исправления любых проблем нажмите кнопку "Продолжить" для продолжения.

![XOOPS Installer Database Connection](/xoops-docs/2.7/img/installation/installer-05.png)

## Данные, собираемые на этом этапе

### Подключение к серверу

#### База данных

В настоящее время _mysql_ — единственный доступный вариант.

#### Имя хоста сервера БД

Имя хоста сервера базы данных. Если вы не уверены, _localhost_ работает в большинстве случаев.

#### Имя пользователя

Учетная запись пользователя базы данных, которую XOOPS должен использовать для подключения к серверу БД.

#### Пароль

Пароль учетной записи пользователя базы данных, введенной выше.

#### Использовать постоянное подключение

По умолчанию 'Нет'. Оставьте это поле пустым, если вы не уверены.
