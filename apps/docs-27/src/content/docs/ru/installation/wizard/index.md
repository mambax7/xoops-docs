Мастер установки XOOPS проведет вас через 15-этапный процесс, который настроит вашу базу данных, создаст учетную запись администратора и подготовит ваш сайт к первому использованию.

## Перед началом

- Вы [загрузили XOOPS на сервер](/xoops-docs/2.7/installation/ftp-upload/) или установили локальное окружение
- Вы [проверили требования](/xoops-docs/2.7/installation/requirements/)
- Вы имеете учетные данные для доступа к базе данных

## Этапы мастера

| Этап | Экран | Что происходит |
|------|--------|--------------|
| 1 | [Выбор языка](./step-01/) | Выберите язык установки |
| 2 | [Добро пожаловать](./step-02/) | Соглашение о лицензии |
| 3 | [Проверка конфигурации](./step-03/) | Проверка окружения PHP/сервера |
| 4 | [Установка путей](./step-04/) | Установите корневой путь и URL |
| 5 | [Подключение к БД](./step-05/) | Введите хост БД, пользователя, пароль |
| 6 | [Конфигурация БД](./step-06/) | Установите имя БД и префикс таблиц |
| 7 | [Сохранение конфигурации](./step-07/) | Запишите mainfile.php |
| 8 | [Создание таблиц](./step-08/) | Создайте схему базы данных |
| 9 | [Начальные настройки](./step-09/) | Имя сайта, email администратора |
| 10 | [Вставка данных](./step-10/) | Заполните данные по умолчанию |
| 11 | [Конфигурация сайта](./step-11/) | URL, часовой пояс, язык |
| 12 | [Выбор темы](./step-12/) | Выберите тему по умолчанию |
| 13 | [Установка модулей](./step-13/) | Установите встроенные модули |
| 14 | [Добро пожаловать](./step-14/) | Сообщение об успешной установке |
| 15 | [Очистка](./step-15/) | Удалите папку установки |

:::caution[Безопасность]
После завершения мастера **удалите или переименуйте папку `install/`** — этап 15 поможет вам это сделать. Оставлять её доступной — это угроза безопасности.
:::
