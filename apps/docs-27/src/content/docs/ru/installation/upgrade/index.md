Этот раздел содержит пошаговое обновление существующего сайта XOOPS. Мастер обновления сохраняет ваши данные, модули и темы при обновлении основных файлов.

## Путь обновления

| Шаг | Страница | Что делать |
|------|------|------------|
| 1 | [Preparations](./step-01/) | Резервная копия, просмотрите журнал изменений, загрузите новый выпуск |
| 2 | [Preflight Check](./preflight/) | Проверьте совместимость перед запуском обновления |
| 3 | [Running the Upgrade](./step-02/) | Выполните мастер обновления |
| 4 | [Troubleshooting](./step-03/) | Решение типовых проблем с обновлением |
| 5 | [After the Upgrade](./step-04/) | Проверьте, обновите модули, очистите кэш |

:::tip
Всегда создавайте **полную резервную копию базы данных и файлов** перед обновлением. Обновление необратимо без резервной копии.
:::
