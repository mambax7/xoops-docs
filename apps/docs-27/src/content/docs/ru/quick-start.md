## Требования

| Компонент  | Минимум                 | Рекомендуемо   |
|------------|-------------------------|---------------|
| PHP        | 8.2                    | 8.4+         |
| MySQL      | 5.7                     | 8.0+          |
| MariaDB    | 10.4                    | 10.11+        |
| Веб-сервер | Apache 2.4 / Nginx 1.20 | Последняя стабильная |

## Загрузка

Загрузите последний релиз с [GitHub Releases](https://github.com/XOOPS/XoopsCore27/releases).

```bash
# Или клонируйте напрямую
git clone https://github.com/XOOPS/XoopsCore27.git mysite
cd mysite
```

## Шаги установки

1. **Загрузите файлы** в корневую папку документов веб-сервера (например `public_html/`).
2. **Создайте базу данных MySQL** и пользователя с полными привилегиями на неё.
3. **Откройте браузер** и перейдите на ваш домен — установщик XOOPS запустится автоматически.
4. **Следуйте мастеру из 5 шагов** — он настроит пути, создаст таблицы и установит ваш аккаунт администратора.
5. **Удалите папку `install/`** когда будет предложено. Это обязательно для безопасности.

## Проверка установки

После установки посетите:

- **Главная страница:** `https://yourdomain.com/`
- **Панель администратора:** `https://yourdomain.com/xoops_data/` *(путь, выбранный вами при установке)*
