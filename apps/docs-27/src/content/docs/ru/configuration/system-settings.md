# Параметры системы XOOPS

Полное руководство по настройке параметров системы XOOPS для оптимальной функциональности.

## Доступ к параметрам системы

**Панель администратора > Система > Параметры**

Основные разделы:
1. Общие параметры
2. Параметры пользователя
3. Параметры электронной почты
4. Параметры кэша
5. Параметры URL
6. Параметры SEO

## Общие параметры

### Информация о сайте

```
Имя сайта: Мой сайт XOOPS
Описание сайта: Описание вашего сайта
Девиз сайта: Краткий девиз или слоган

URL сайта: http://your-domain.com/xoops/
Язык по умолчанию: English
Тема по умолчанию: [Выберите тему]
```

### Информация администратора

```
Имя администратора: Администратор сайта
Электронная почта администратора: admin@your-domain.com
Адрес поддержки: support@your-domain.com
```

### Региональные параметры

```
Часовой пояс: Europe/Moscow
Формат даты: %Y-%m-%d
Формат времени: %H:%M:%S
Первый день недели: Monday
```

## Параметры модулей

### Порядок модулей

Управляйте отображением модулей в меню:

**Система > Параметры > Модули**

Для каждого модуля:
```
Вес: [1-999]  (меньше = выше в списке)
Отображение в меню: Да/Нет
Видимость: Все/Группы
```

### Отключение модулей

Отключите ненужные модули для улучшения производительности:

1. **Система > Модули > Модули**
2. Найдите модуль
3. Нажмите название
4. Установите статус "Неактивный"
5. Сохраните

## Управление блоками

### Создание блоков

**Внешний вид > Блоки**

Новый блок:
```
Название блока: [Название]
Содержимое: [HTML или текст]
Видимость: Все страницы / Конкретные страницы
Позиция: Слева / Справа / Центр
Видно для групп: [Выберите группы]
```

### Видимость блока

Контролируйте, где отображается блок:

```
По умолчанию: Скрыт
На главной: Видимо
На странице модуля: Видимо
На конкретных страницах: Выберите
```

## Параметры тем

### Выбор темы

**Внешний вид > Темы**

Установите тему по умолчанию:

1. Выберите тему из списка
2. Нажмите "Установить по умолчанию"
3. Тема применяется всем пользователям

### Конфигурация темы

Некоторые темы имеют параметры конфигурации:

```
Ширина макета: [Фиксированная/Адаптивная]
Цветовая схема: [Светлая/Тёмная]
Размер шрифта: [Маленький/Средний/Крупный]
```

### Кэширование

```

Тип кэша: File Cache (APCu/Memcache если доступны)
Время жизни кэша: 3600 сек.
```

### Оптимизация

```
Сжимать вывод: Да
Минифицировать CSS: Да
Минифицировать JavaScript: Да
```

### Ограничение доступа

```
Максимум неудачных входов: 5
Время блокировки: 15 минут
Требуемая длина пароля: 8 символов
Срок действия пароля: 90 дней
```

### Защита

```
Требовать HTTPS: Да
Защита CSRF: Да
Защита XSS: Да

```

### Мета-теги

### Карта сайта


## Сохранение параметров

После изменения параметров:

1. Нажмите **"Сохранить"** в конце страницы
2. Подождите подтверждения
3. Параметры сохранены

## Дефрагментация таблиц

Периодически оптимизируйте таблицы БД:

**Система > Инструменты > Оптимизировать БД**

Или вручную:

```bash
mysql -u xoops_user -p xoops_db << EOF
OPTIMIZE TABLE xoops_users;
OPTIMIZE TABLE xoops_posts;
OPTIMIZE TABLE xoops_config;
EOF
```

## Экспорт и импорт конфигурации

### Резервная копия конфигурации

```bash
# Экспортировать параметры системы
mysqldump -u xoops_user -p xoops_db xoops_config > config_backup.sql
```

### Восстановление конфигурации

```bash
# Импортировать параметры
mysql -u xoops_user -p xoops_db < config_backup.sql
```
