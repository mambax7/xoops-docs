## Статус

Принято

## Контекст

XOOPS расширяется через модули. Нужна структурированная система для их управления.

## Решение

Реализовать стандартизированную систему модулей с жизненным циклом загрузки, установки и активации.

## Обоснование

- Модульная архитектура
- Независимые расширения
- Управление зависимостями
- Контролируемое расширение

## Компоненты

- XoopsModule - Представление модуля
- ModuleHandler - Управление модулями
- xoops_version.php - Метаинформация модуля
- install/remove - Жизненный цикл

## Примеры

```php
$moduleHandler = xoops_getHandler('module');
$module = $moduleHandler->getByDirname('news');
```

---

*Система модулей обеспечивает основу для расширяемости XOOPS.*
