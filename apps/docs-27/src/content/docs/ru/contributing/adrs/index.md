Записи об архитектурных решениях (ADR) документируют важные архитектурные решения, принятые в проекте XOOPS, и их обоснование.

## Что такое ADR?

ADR — это записи, которые фиксируют важное архитектурное решение, принятое в контексте проекта.

## Как использовать ADR

При возникновении архитектурного вопроса:
1. Создайте новую запись ADR
2. Обсудите решение с командой
3. Задокументируйте решение и его обоснование
4. Оставьте запись для будущих разработчиков
