---
title: "ADR-002: Миграция на PSR-4"
description: "Решение использовать автозагрузчик PSR-4"
---

## Статус

Принято

## Контекст

XOOPS использовал пользовательский автозагрузчик. PSR-4 — это стандарт PHP для автозагрузки классов.

## Решение

Мигрировать на автозагрузчик PSR-4 для всех классов.

## Обоснование

- Соответствие стандартам PHP
- Совместимость с Composer
- Лучшая производительность
- Широкая поддержка инструментов

## Последствия

- Переструктурирование кода
- Обновление пространств имен
- Совместимость с инструментами анализа

## Примеры

```php
namespace Xoops\Core;

class XoopsObject
{
    // Класс теперь находится в файле:
    // /htdocs/class/Xoops/Core/XoopsObject.php
}
```

---

*PSR-4 обеспечивает стандартный способ организации кода PHP.*
