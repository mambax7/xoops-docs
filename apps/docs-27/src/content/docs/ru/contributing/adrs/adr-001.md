title: "ADR-001: Использование PHP 7.4+"
description: "Решение требовать PHP 7.4+ для XOOPS"
---

## Статус

Принято

## Контекст

XOOPS 2.5 требовал PHP 5.3+. Множество новых функций и улучшений производительности были добавлены в PHP 7.4.

## Решение

Требовать PHP 7.4 как минимальную версию для XOOPS 2.5.11+.

## Обоснование

- Современные функции типизации
- Лучшая производительность
- Долгосрочная поддержка сообществом
- Прекращена поддержка PHP < 7.4

## Последствия

- Повышение требований к хостингу
- Возможность использования новых функций PHP
- Лучшая безопасность

## Связанная информация

- [PHP 7.4 릴리스 정보](https://www.php.net/releases/7_4_0.php)
