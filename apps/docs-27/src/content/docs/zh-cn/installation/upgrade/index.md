title: “升级XOOPS”
description: “如何将现有的XOOPS安装安全地升级到新版本。”
---

本部分将引导您升级现有的 XOOPS 站点。升级向导在更新核心文件时会保留您的数据、模区块和主题。

## 升级路径

|步骤|页 |该怎么办 |
|------|------|------------|
| 1 | [Preparations](./step-01/) |备份、查看变更日志、下载新版本 |
| 2 | [Preflight Check](./preflight/) |运行升级之前验证兼容性 |
| 3 | [Running the Upgrade](./step-02/) |执行升级向导|
| 4 | [Troubleshooting](./step-03/) |解决常见升级问题 |
| 5 | [After the Upgrade](./step-04/) |验证、更新模区块、清除缓存 |

:::提示
升级前务必进行**完整数据库备份和文件备份**。如果没有备份，升级是不可逆的。
:::
