此页面收集有关 XOOPS 将使用的数据库用户帐户的信息。

输入所需信息并更正任何问题后，选择“继续”按钮继续。

![XOOPS Installer Database Connection](/XOOPS-docs/2.7/img/installation/installer-05.png)

## 本步骤收集的数据

### 服务器连接

#### 数据库

目前，_mysql_ 是唯一可用的选项。

#### 服务器主机名

数据库服务器的主机名。如果您不确定，_localhost_ 在大多数情况下都有效。

#### 用户名

XOOPS 用于连接到数据库服务器的数据库用户帐户。

#### 密码

上面输入的数据库用户帐户的密码。

#### 使用持久连接

默认为“否”。如果您不确定，请将此留空。
