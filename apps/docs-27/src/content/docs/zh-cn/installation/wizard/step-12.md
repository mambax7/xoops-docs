此页面收集您对新网站初始主题的选择。主题控制网站的视觉特征。如果需要，主题可以随时更改。

输入所需信息并更正任何问题后，选择“继续”按钮继续。

![XOOPS Installer Select Theme](/XOOPS-docs/2.7/img/installation/installer-12.png)

## 本步骤收集的数据

### 选择默认主题

#### 默认主题

从列表中为您的网站选择一个初始主题。每个主题在选择时都会显示一个屏幕截图。

XOOPS 2.7.0 附带了几个开箱即用的前-end主题：

* **默认** — 最小基线主题，可用作自定义的起点。
* **xbootstrap5** — Bootstrap 5 主题，推荐用于大多数新网站。
* **xswatch5** — Bootswatch 5 变体； Bootstrap 5 之上的一组样式 sub-themes。
* **xtailwind** — Tailwind CSS-based 主题 + DaisyUI 具有 35 个调色板和 Alpine.js 交互性
* **xtailwind2** — art-directed兄弟主题，具有精选的调色板和精致的模区块样式

您可以随时从管理区域更改主题，并且可以将其他主题上传到`themes/`目录。
