此页面收集您对要安装在新站点中的模区块的选择。如果需要，可以随时添加或删除模区块。

输入所需信息并更正任何问题后，选择“继续”按钮继续。

![XOOPS Installer Modules Installation](/XOOPS-docs/2.7/img/installation/installer-13.png)

## 本步骤收集的数据

### 可用模区块

选择“是”安装模区块。您可以不选择任何模区块、选择任何模区块或选择所有模区块。

这些是标准 XOOPS 2.7.0 安装中包含的模区块。

* 私人消息传递 - _增强用户之间的私人消息传递_
* 用户个人资料 - _enhanced/custom 用户个人资料_
* 保护器 - _增强站点安全性、垃圾邮件控制和攻击缓解_
* DebugBar - _基于 Symfony VarDumper 的开发和分段调试工具栏；对于模区块开发人员和诊断生产问题很有用，但通常在公共-facing实时站点上不需要_

_System_ 模区块始终已安装，此处未作为选项列出。

#### 注意

您可以在开始安装之前上传XOOPS文件时将模区块添加到modules目录中，它们也将在此步骤中可用。

####强烈推荐

**所有XOOPS站点**应安装_Protector_模区块。
