## 关于XOOPS CMS

XOOPS 是一个内容管理系统 (CMS) 和 Web 门户程序，允许管理员轻松创建具有丰富内容和许多出色功能的动态网站。它是开发小型到大型动态社区网站、内部-company门户、企业门户、博客等的理想工具。它可以安装在任何具有 PHP-capable Web 服务器（例如 Apache 或 Nginx）和 MySQL-compatible 数据库的主机上。

**主要项目网站：** [https://XOOPS.org](https://XOOPS.org)

XOOPS 根据GNU 通用公共权限证 (GPL) 的条款发布，可免费使用和修改。只要您遵守GPL的分发条款，您就可以重新分发它。

### XOOPS 代表什么

XOOPS 是**可扩展面向对象门户系统**的缩写。虽然它最初是一个门户系统，但现在它是一个功能齐全的内容管理系统，可以通过大型企业安装为小型个人网站提供服务。

### 发音

XOOPS的标准发音遵循英语规则，发音为`[zoo'ps]`。

### 主要特点

**数据库-driven**

XOOPS使用关系数据库（MySQL，5.7.8或更高版本）来存储运行网络-based内容管理系统所需的数据。

**完全模区块化**

单击XOOPS模区块管理系统即可安装、卸载、激活或停用模区块。

**个性化**

注册用户可以编辑他们的个人资料、选择网站主题、上传自定义头像等等。

**用户管理**

XOOPS 通过各种条件、电子邮件和模板-based 私人消息系统提供跨用户搜索。

**全球支持**

XOOPS 由来自世界各地的志愿者团队创建和维护。 XOOPS 社区拥有多种语言的支持站点。

**多字节语言支持**

XOOPS 完全支持多字节语言，包括日语、简体中文、繁体中文、韩语和阿拉伯语（2.7.0 中添加了 RTL 支持）。

**多功能群组权限**

强大的用户-friendly权限系统允许管理员按组设置权限。

**主题-based可换肤界面**

主题系统允许管理员和用户更改整个站点的外观。 2.7.0 发行版附带现代 Bootstrap-5 和 Tailwind-based 主题。
