## Επισκόπηση

Το Publisher είναι η οριστική ενότητα διαχείρισης περιεχομένου για το XOOPS, που εξελίχθηκε από το SmartSection για να γίνει η πιο πλούσια σε δυνατότητες λύση ιστολογίου και ειδήσεων. Παρέχει ολοκληρωμένα εργαλεία για τη δημιουργία, οργάνωση και δημοσίευση περιεχομένου με πλήρη υποστήριξη ροής εργασιών σύνταξης.

**Απαιτήσεις:**
- XOOPS 2.5.10+
- PHP 7,1+ (PHP 8,x συνιστάται)

## 🌟 Βασικά χαρακτηριστικά

## # Διαχείριση περιεχομένου
- **Κατηγορίες & Υποκατηγορίες** - Ιεραρχική οργάνωση περιεχομένου
- **Επεξεργασία εμπλουτισμένου κειμένου** - Υποστηρίζονται πολλοί WYSIWYG επεξεργαστές

- **Διαχείριση προσχέδιο** - Αποθήκευση εργασιών σε εξέλιξη

- **Τέσσερα βασικά πρότυπα** - Πολλαπλές διατάξεις οθόνης
- **Προσαρμοσμένα πρότυπα** - Δημιουργήστε τα δικά σας σχέδια

- **Σχεδίαση με απόκριση** - Έξοδος έτοιμη για φορητές συσκευές

## # Αλληλεπίδραση χρήστη
- **Βαθμολογίες** - Σύστημα αξιολόγησης άρθρου
- **Σχόλια** - Συζητήσεις αναγνωστών
- **Κοινωνική κοινή χρήση** - Κοινή χρήση σε κοινωνικά δίκτυα

## # Δικαιώματα
- **Έλεγχος υποβολής** - Ποιος μπορεί να υποβάλει άρθρα

## 🚀 Γρήγορη εκκίνηση

## # 1. Εγκατάσταση

```bash
# Download from GitHub
git clone https://github.com/XoopsModules25x/publisher.git

# Copy to modules directory
cp -r publisher /path/to/xoops/htdocs/modules/
```

Στη συνέχεια, εγκαταστήστε μέσω XOOPS Admin → Modules → Install.

## # 2. Δημιουργήστε την πρώτη σας κατηγορία

1. Μεταβείτε στο **Διαχειριστής → Εκδότης → Κατηγορίες**
2. Κάντε κλικ στο **Προσθήκη κατηγορίας**
3. Συμπληρώστε:
   - **Όνομα**: Ειδήσεις
   - **Περιγραφή**: Τελευταία νέα και ενημερώσεις
   - **Εικόνα**: Μεταφόρτωση εικόνας κατηγορίας
4. Αποθήκευση

## # 3. Δημιουργήστε το πρώτο σας άρθρο

1. Μεταβείτε στο **Διαχειριστής → Εκδότης → Άρθρα**
2. Κάντε κλικ στο **Προσθήκη άρθρου**
3. Συμπληρώστε:
   - **Τίτλος**: Καλώς ήρθατε στον ιστότοπό μας
   - **Κατηγορία**: Ειδήσεις
   - **Περιεχόμενο**: Το περιεχόμενο του άρθρου σας
4. Ορισμός **Κατάσταση**: Δημοσιεύτηκε
5. Αποθήκευση

## 🔄 Μετανάστευση

## # Από το SmartSection

Ο Publisher περιλαμβάνει ένα ενσωματωμένο εργαλείο μετεγκατάστασης:

1. Μεταβείτε στο **Διαχειριστής → Εκδότης → Εισαγωγή**
2. Επιλέξτε **SmartSection** ως πηγή
3. Επιλέξτε επιλογές εισαγωγής:
   - Κατηγορίες
   - Άρθρα
   - Σχόλια
4. Κάντε κλικ στο **Εισαγωγή**

## # Από την ενότητα ειδήσεων

1. Μεταβείτε στο **Διαχειριστής → Εκδότης → Εισαγωγή**
2. Επιλέξτε **Ειδήσεις** ως πηγή
3. Κατηγορίες χαρτών
4. Κάντε κλικ στο **Εισαγωγή**
