# Δημιουργία της πρώτης σας σελίδας στο XOOPS

Μάθετε πώς να δημιουργείτε, να μορφοποιείτε και να δημοσιεύετε το πρώτο σας περιεχόμενο στο XOOPS.

## Κατανόηση του περιεχομένου XOOPS

## # Τι είναι το Page/Post;

Στο XOOPS, η διαχείριση του περιεχομένου γίνεται μέσω λειτουργικών μονάδων. Οι πιο συνηθισμένοι τύποι περιεχομένου είναι:

| Τύπος | Περιγραφή | Περίπτωση χρήσης |
|---|---|---|
| **Σελίδα** | Στατικό περιεχόμενο | Σχετικά με εμάς, Επικοινωνία, Υπηρεσίες |
| **Post/Article** | Περιεχόμενο με χρονική σφραγίδα | Ειδήσεις, αναρτήσεις ιστολογίου |
| **Κατηγορία** | Οργάνωση περιεχομένου | Περιεχόμενο σχετικό με την ομάδα |
| **Σχόλιο** | Σχόλια χρηστών | Να επιτρέπεται η αλληλεπίδραση επισκεπτών |

Αυτός ο οδηγός καλύπτει τη δημιουργία μιας βασικής ενότητας page/article με τη χρήση της προεπιλεγμένης μονάδας περιεχομένου XOOPS.

## Πρόσβαση στον Επεξεργαστή Περιεχομένου

## # Από τον Πίνακα Διαχειριστή

1. Συνδεθείτε στον πίνακα διαχείρισης: `http://your-domain.com/XOOPS/admin/`
2. Μεταβείτε στο **Περιεχόμενο > Σελίδες** (ή στη μονάδα περιεχομένου σας)
3. Κάντε κλικ στην επιλογή "Προσθήκη νέας σελίδας" ή "Νέα ανάρτηση"

## # Frontend (αν είναι ενεργοποιημένο)

Εάν το XOOPS σας έχει ρυθμιστεί ώστε να επιτρέπει τη δημιουργία περιεχομένου διεπαφής:

1. Συνδεθείτε ως εγγεγραμμένος χρήστης
2. Μεταβείτε στο προφίλ σας
3. Αναζητήστε την επιλογή "Υποβολή περιεχομένου".
4. Ακολουθήστε τα ίδια βήματα παρακάτω

## Διεπαφή επεξεργασίας περιεχομένου

## Οδηγός βήμα προς βήμα: Δημιουργία της πρώτης σας σελίδας

## # Βήμα 1: Πρόσβαση στον Επεξεργαστή Περιεχομένου

1. Στον πίνακα διαχειριστή, κάντε κλικ στο **Περιεχόμενο > Σελίδες**
2. Κάντε κλικ στο **"Προσθήκη νέας σελίδας"** ή **"Δημιουργία"**
3. Θα δείτε το πρόγραμμα επεξεργασίας περιεχομένου

## # Βήμα 2: Εισαγάγετε τον τίτλο της σελίδας

Στο πεδίο "Τίτλος", εισαγάγετε το όνομα της σελίδας σας:

```
Title: Welcome to Our Website
```

Βέλτιστες πρακτικές για τίτλους:
- Ξεκάθαρο και περιγραφικό
- Συμπεριλάβετε λέξεις-κλειδιά αν είναι δυνατόν
- 50-60 χαρακτήρες ιδανικό
- Αποφύγετε το ALL CAPS (δύσκολα διαβασμένο)
- Να είστε συγκεκριμένοι (όχι "Σελίδα 1")

## # Βήμα 3: Επιλέξτε Κατηγορία

Επιλέξτε πού θα οργανωθεί αυτό το περιεχόμενο:

```
Category: [Dropdown ▼]
```

Οι επιλογές μπορεί να περιλαμβάνουν:
- Στρατηγός
- Ειδήσεις
- Ιστολόγιο
- Ανακοινώσεις
- Υπηρεσίες

Εάν οι κατηγορίες δεν υπάρχουν, ζητήστε από τον διαχειριστή να τις δημιουργήσει.

## # Βήμα 4: Γράψτε το περιεχόμενό σας

Κάντε κλικ στην περιοχή επεξεργασίας περιεχομένου και πληκτρολογήστε το κείμενό σας.

### # Βασική μορφοποίηση κειμένου

Χρησιμοποιήστε τη γραμμή εργαλείων του επεξεργαστή:

| Κουμπί | Δράση | Αποτέλεσμα |
|---|---|---|
| **Β** | Τολμηρός | **Έντονο κείμενο** |
| *Ι* | Πλάγια | *Πλάγιο κείμενο* |
| <u>U</u> | Υπογράμμιση | <u>Υπογραμμισμένο κείμενο</u> |

### # Χρήση HTML

Το XOOPS επιτρέπει ασφαλείς ετικέτες HTML. Συνηθισμένα παραδείγματα:

```html
<!-- Paragraphs -->
<p>This is a paragraph.</p>

<!-- Headings -->
<h1>Main Heading</h1>
<h2>Subheading</h2>

<!-- Lists -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<!-- Bold and Italic -->
<strong>Bold text</strong>
<em>Italic text</em>

<!-- Links -->
<a href="https://example.com">Link text</a>

<!-- Line breaks -->
<br>

<!-- Horizontal rule -->
<hr>
```

### # Ασφαλές HTML Παραδείγματα

**Προτεινόμενες ετικέτες:**
- Παράγραφοι: `<p>`, `<br>`
- Επικεφαλίδες: `<h1>` έως `<h6>`
- Κείμενο: `<strong>`, `<em>`, `<u>`
- Λίστες: `<ul>`, `<ol>`, `<li>`
- Σύνδεσμοι: `<a href="">`
- Εισαγωγικά: `<blockquote>`
- Πίνακες: `<table>`, `<tr>`, `<td>`

**Αποφύγετε αυτές τις ετικέτες** (μπορεί να είναι απενεργοποιημένες για ασφάλεια):
- Σενάρια: `<script>`
- Στυλ: `<style>`
- Iframes: `<iframe>` (εκτός εάν έχουν ρυθμιστεί)
- Έντυπα: `<form>`, `<input>`

## # Βήμα 5: Προσθήκη εικόνων

### # Επιλογή 1: Εισαγωγή εικόνας URL

Χρησιμοποιώντας το πρόγραμμα επεξεργασίας:

1. Κάντε κλικ στο κουμπί **Εισαγωγή εικόνας** (εικονίδιο εικόνας)
2. Εισαγάγετε την εικόνα URL: `https://example.com/image.jpg`
3. Εισαγάγετε εναλλακτικό κείμενο: "Περιγραφή εικόνας"
4. Κάντε κλικ στο "Εισαγωγή"

HTML ισοδύναμο:

```html
<img src="https://example.com/image.jpg" alt="Description">
```

### # Επιλογή 2: Μεταφόρτωση εικόνας

1. Μεταφορτώστε πρώτα την εικόνα στο XOOPS:
   - Μεταβείτε στο **Περιεχόμενο > Διαχείριση πολυμέσων**
   - Ανεβάστε την εικόνα σας
   - Αντιγράψτε την εικόνα URL

2. Στο πρόγραμμα επεξεργασίας περιεχομένου, εισαγάγετε χρησιμοποιώντας το URL (παραπάνω βήματα)

### # Βέλτιστες πρακτικές εικόνας

- Χρησιμοποιήστε κατάλληλα μεγέθη αρχείων (βελτιστοποίηση εικόνων)
- Χρησιμοποιήστε περιγραφικά ονόματα αρχείων
- Να συμπεριλαμβάνεται πάντα εναλλακτικό κείμενο (προσβασιμότητα)
- Υποστηριζόμενες μορφές: JPG, PNG, GIF, WebP
- Προτεινόμενο πλάτος: 600-800 pixel για περιεχόμενο

## # Βήμα 6: Ενσωμάτωση πολυμέσων

### # Ενσωματώστε βίντεο από το YouTube

```html
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  frameborder="0" allowfullscreen>
</iframe>
```

Αντικαταστήστε το `VIDEO_ID` με το αναγνωριστικό βίντεο YouTube.

**Για να βρείτε το αναγνωριστικό βίντεο YouTube:**
1. Ανοίξτε το βίντεο στο YouTube
2. URL είναι: `https://www.youtube.com/watch?v=VIDEO_ID`
3. Αντιγράψτε το αναγνωριστικό (χαρακτήρες μετά το `v=`)

### # Ενσωματώστε βίντεο από το Vimeo

```html
<iframe src="https://player.vimeo.com/video/VIDEO_ID"
  width="640" height="360" frameborder="0"
  allow="autoplay; fullscreen" allowfullscreen>
</iframe>
```

## # Βήμα 7: Προσθήκη Meta Description

Στο πεδίο "Περιγραφή", προσθέστε μια σύντομη περίληψη:

```
Description: Learn how to get started with our website.
This page provides an overview of our services and how we can help you.
```

**Βέλτιστες πρακτικές μετα-περιγραφής:**
- 150-160 χαρακτήρες
- Συμπεριλάβετε κύριες λέξεις-κλειδιά
- Θα πρέπει να συνοψίζει με ακρίβεια το περιεχόμενο
- Χρησιμοποιείται στα αποτελέσματα των μηχανών αναζήτησης
- Κάντε το συναρπαστικό (οι χρήστες βλέπουν αυτό)

## # Βήμα 8: Διαμόρφωση επιλογών δημοσίευσης

### # Κατάσταση δημοσίευσης

Επιλέξτε κατάσταση δημοσίευσης:

```
Status: ☑ Published
```

Επιλογές:
- **Δημοσιεύτηκε:** Ορατό στο κοινό
- **Πρόχειρο:** Ορατό μόνο στους διαχειριστές
- **Εκκρεμεί έλεγχος:** Εν αναμονή έγκρισης
- **Αρχειοθετήθηκε:** Κρυφό αλλά διατηρήθηκε

### # Ορατότητα

Ορίστε ποιος μπορεί να δει αυτό το περιεχόμενο:

```
Visibility: ☐ Public
           ☐ Registered Users Only
           ☐ Private (Admin Only)
```

### # Ημερομηνία δημοσίευσης

Ορίστε πότε το περιεχόμενο γίνεται ορατό:

```
Publish Date: [Date Picker] [Time]
```

Αφήστε ως "Τώρα" για δημοσίευση αμέσως.

### # Να επιτρέπονται τα σχόλια

Ενεργοποίηση ή απενεργοποίηση σχολίων επισκεπτών:

```
Allow Comments: ☑ Yes
```

Εάν είναι ενεργοποιημένο, οι επισκέπτες μπορούν να προσθέσουν σχόλια.

## # Βήμα 9: Αποθηκεύστε το περιεχόμενό σας

Πολλαπλές επιλογές αποθήκευσης:

```
[Publish Now]  [Save as Draft]  [Schedule]  [Preview]
```

- **Δημοσίευση τώρα:** Κάντε άμεσα ορατό
- **Αποθήκευση ως πρόχειρο:** Διατηρήστε ιδιωτικό προς το παρόν
- **Πρόγραμμα:** Δημοσίευση στο μέλλον date/time
- **Προεπισκόπηση:** Δείτε πώς φαίνεται πριν την αποθήκευση

Κάντε κλικ στην επιλογή σας:

```
Click [Publish Now]
```

## # Βήμα 10: Επαληθεύστε τη σελίδα σας

Μετά τη δημοσίευση, επαληθεύστε το περιεχόμενό σας:

1. Μεταβείτε στην αρχική σελίδα του ιστότοπού σας
2. Πλοηγηθείτε στην περιοχή περιεχομένου σας
3. Αναζητήστε τη σελίδα που δημιουργήσατε πρόσφατα
4. Κάντε κλικ για να το δείτε
5. Ελέγξτε:
   - [ ] Το περιεχόμενο εμφανίζεται σωστά
   - [ ] Εμφανίζονται εικόνες
   - [ ] Η μορφοποίηση φαίνεται καλή
   - [ ] Οι σύνδεσμοι λειτουργούν
   - [ ] Ο τίτλος και η περιγραφή είναι σωστά

## Αντιμετώπιση προβλημάτων Δημιουργία περιεχομένου

## # Το περιεχόμενο δεν εμφανίζεται

**Σύμπτωμα:** Η δημοσιευμένη σελίδα δεν εμφανίζεται στον ιστότοπο

**Λύση:**
1. Ελέγξτε την κατάσταση δημοσίευσης: Πρέπει να είναι "Δημοσιευμένο"
2. Ελέγξτε την ημερομηνία δημοσίευσης: Πρέπει να είναι τρέχουσα ή προηγούμενη
3. Ελέγξτε την ορατότητα: Πρέπει να είναι "Δημόσιο"
4. Εκκαθάριση προσωρινής μνήμης: Διαχειριστής > Εργαλεία > Εκκαθάριση προσωρινής μνήμης
5. Έλεγχος δικαιωμάτων: Η ομάδα χρηστών πρέπει να έχει πρόσβαση

## # Η μορφοποίηση δεν λειτουργεί

**Σύμπτωμα:** HTML ετικέτες ή μορφοποίηση εμφανίζονται ως κείμενο

**Λύση:**
1. Βεβαιωθείτε ότι το HTML είναι ενεργοποιημένο στις ρυθμίσεις της μονάδας
2. Χρησιμοποιήστε τη σωστή σύνταξη HTML
3. Κλείστε όλες τις ετικέτες: `<p>Text</p>`
4. Χρησιμοποιήστε μόνο επιτρεπόμενες ετικέτες
5. Χρησιμοποιήστε οντότητες HTML: `&lt;` για `<`, `&amp;` για `&`

## # Οι εικόνες δεν εμφανίζονται

**Σύμπτωμα:** Οι εικόνες δείχνουν σπασμένο εικονίδιο

**Λύση:**
1. Βεβαιωθείτε ότι η εικόνα URL είναι σωστή
2. Ελέγξτε εάν υπάρχει αρχείο εικόνας
3. Επαληθεύστε τα σωστά δικαιώματα στην εικόνα
4. Δοκιμάστε να ανεβάσετε εικόνα στο XOOPS
5. Ελέγξτε για εξωτερικό μπλοκάρισμα (μπορεί να χρειαστεί CORS)

## # Ζητήματα κωδικοποίησης χαρακτήρων

**Σύμπτωμα:** Οι ειδικοί χαρακτήρες εμφανίζονται ως ασυναρτησίες

**Λύση:**
1. Αποθηκεύστε το αρχείο ως κωδικοποίηση UTF-8
2. Βεβαιωθείτε ότι το σύνολο χαρακτήρων σελίδας είναι UTF-8
3. Προσθήκη στην κεφαλή HTML: `<meta charset="UTF-8">`
4. Αποφύγετε την αντιγραφή-επικόλληση από το Word (χρησιμοποιήστε απλό κείμενο)

## Βέλτιστες πρακτικές ροής εργασίας περιεχομένου

## # Προτεινόμενη διαδικασία

1. **Γράψτε πρώτα στο πρόγραμμα επεξεργασίας:** Χρησιμοποιήστε το πρόγραμμα επεξεργασίας περιεχομένου διαχειριστή
2. **Προεπισκόπηση πριν από τη δημοσίευση:** Κάντε κλικ στο κουμπί Προεπισκόπηση
3. **Προσθήκη μεταδεδομένων:** Πλήρης τίτλος, περιγραφή, ετικέτες
4. **Αποθήκευση ως πρόχειρο πρώτα:** Αποθήκευση ως πρόχειρο για να αποφύγετε την απώλεια εργασίας
5. **Τελική αναθεώρηση: ** Ξαναδιάβασε πριν από τη δημοσίευση
6. **Δημοσίευση:** Κάντε κλικ στο Δημοσίευση όταν είναι έτοιμο
7. **Επαλήθευση:** Έλεγχος σε ζωντανή τοποθεσία
8. **Επεξεργαστείτε εάν χρειάζεται:** Κάντε γρήγορα διορθώσεις

## # Έλεγχος έκδοσης

Διατηρείτε πάντα αντίγραφα ασφαλείας:

1. **Πριν από σημαντικές αλλαγές:** Αποθήκευση ως νέα έκδοση ή αντίγραφο ασφαλείας
2. **Αρχειοθέτηση παλαιού περιεχομένου:** Διατήρηση μη δημοσιευμένων εκδόσεων
3. **Ημερομηνία των προσχεδίων σας:** Χρησιμοποιήστε σαφή ονομασία: "Page-Draft-2025-01-28"

## Δημοσίευση πολλαπλών σελίδων

Δημιουργήστε μια στρατηγική περιεχομένου:

```
Homepage
├── About Us
├── Services
│   ├── Service 1
│   ├── Service 2
│   └── Service 3
├── Blog
│   ├── Article 1
│   ├── Article 2
│   └── Article 3
├── Contact
└── FAQ
```

Δημιουργήστε σελίδες για να ακολουθήσετε αυτήν τη δομή.

## Επόμενα βήματα

Αφού δημιουργήσετε την πρώτη σας σελίδα:

1. Ρυθμίστε λογαριασμούς χρηστών
2. Εγκαταστήστε πρόσθετες μονάδες
3. Εξερευνήστε τις δυνατότητες διαχειριστή
4. Διαμορφώστε τις ρυθμίσεις
5. Βελτιστοποιήστε με ρυθμίσεις απόδοσης
