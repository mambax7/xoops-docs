title: "Εργαλεία του Εμπορίου"

Υπάρχουν πολλά πράγματα που χρειάζονται για να προσαρμόσετε και να διατηρήσετε έναν ιστότοπο XOOPS που πρέπει να συμβούν εκτός του XOOPS ή γίνονται πιο εύκολα εκεί.

Αυτή είναι μια λίστα τύπων εργαλείων που μπορεί να θέλετε να έχετε διαθέσιμα, μαζί με ορισμένες προτάσεις για συγκεκριμένα εργαλεία που οι webmasters XOOPS βρήκαν χρήσιμα.

## Συντάκτες

Οι συντάκτες είναι μια πολύ προσωπική επιλογή και οι άνθρωποι μπορούν να παθιαστούν με το αγαπημένο τους. Θα παρουσιάσουμε μόνο μερικές από τις πολλές δυνατότητες.

Για χρήση του XOOPS, θα χρειαστείτε ένα πρόγραμμα επεξεργασίας για να τροποποιήσετε ορισμένες επιλογές διαμόρφωσης καθώς και να προσαρμόσετε ένα θέμα για τον ιστότοπό σας. Για αυτές τις χρήσεις, μπορεί να είναι πολύ χρήσιμο να έχετε ένα πρόγραμμα επεξεργασίας που μπορεί να λειτουργεί με πολλά αρχεία ταυτόχρονα, να μπορεί να αναζητά και να αντικαθιστά σε πολλά αρχεία και να παρέχει επισήμανση σύνταξης. Μπορείτε να χρησιμοποιήσετε ένα πολύ απλό πρόγραμμα επεξεργασίας χωρίς περιττά στοιχεία, αλλά θα εργαστείτε πολύ πιο σκληρά για να ολοκληρώσετε ορισμένες εργασίες.

Το **PhpStorm** από την _JetBrains_ είναι ένα IDE (ενσωματωμένο περιβάλλον ανάπτυξης) ειδικά προσαρμοσμένο για PHP ανάπτυξη ιστού. Η _JetBrains_ βοήθησε πολύ στη χορηγία του XOOPS και τα προϊόντα του είναι αγαπημένα για πολλούς προγραμματιστές. Είναι ένα εμπορικό προϊόν και μπορεί να είναι απαγορευτικό για ορισμένους νέους webmasters, αλλά ο χρόνος που μπορεί να εξοικονομήσει το κάνει ελκυστικό για έμπειρους προγραμματιστές.

Το **Visual Studio Code** είναι ένα δωρεάν πρόγραμμα επεξεργασίας πηγαίου κώδικα πολλαπλών πλατφορμών από τη Microsoft. Διαθέτει υποστήριξη, είτε ενσωματωμένη είτε μέσω επεκτάσεων, για τις βασικές τεχνολογίες Ιστού όπως HTML, JavaScript και PHP, γεγονός που το καθιστά κατάλληλο για χρήση XOOPS.

Το **Notepad++** είναι ένας δωρεάν, διαχρονικός υποψήφιος σε αυτήν την κατηγορία για Windows, με πιστούς χρήστες.

Το **Meld** δεν είναι πρόγραμμα επεξεργασίας, αλλά συγκρίνει αρχεία κειμένου που εμφανίζουν διαφορές και επιτρέπει τη συγχώνευση αλλαγών επιλεκτικά και την πραγματοποίηση μικρών επεξεργασιών. Είναι πολύ χρήσιμο όταν συγκρίνετε αρχεία διαμόρφωσης, πρότυπα θεμάτων και φυσικά κώδικα PHP.

## FTP Πελάτης

Πρωτόκολλο μεταφοράς αρχείων (FTP,) ή μια παραλλαγή του, χρησιμοποιείται για τη μετακίνηση αρχείων από έναν υπολογιστή σε άλλο. Οι περισσότερες εγκαταστάσεις XOOPS θα χρειαστούν έναν πελάτη FTP για τη μετακίνηση αρχείων που προέρχονται από τη διανομή XOOPS σε ένα κεντρικό σύστημα όπου θα αναπτυχθεί η τοποθεσία.

Το **FileZilla** είναι ένας δωρεάν και ισχυρός πελάτης FTP που είναι διαθέσιμος για τις περισσότερες πλατφόρμες. Η συνέπεια μεταξύ πλατφορμών το έκανε την επιλογή για τα FTP παραδείγματα σε αυτό το βιβλίο.

Το **PuTTY** είναι ένας δωρεάν πελάτης SSH, χρήσιμος για την πρόσβαση του Shell σε διακομιστή, καθώς και για την παροχή δυνατοτήτων μεταφοράς αρχείων με το SCP

Το **WinSCP** είναι ένας πελάτης FTP/SFTP/SCP για συστήματα Windows.

## MySQL/MariaDB

Η βάση δεδομένων περιέχει όλο το περιεχόμενο του ιστότοπού σας, τις διαμορφώσεις που προσαρμόζουν τον ιστότοπό σας, τις πληροφορίες σχετικά με τους χρήστες του ιστότοπού σας και πολλά άλλα. Η προστασία και η διατήρηση αυτών των πληροφοριών μπορεί να είναι ευκολότερη με ορισμένα πρόσθετα εργαλεία που ασχολούνται ειδικά με τη βάση δεδομένων.

Το **phpMyAdmin** είναι το πιο δημοφιλές εργαλείο που βασίζεται στο web για εργασία με το MySQL databases, including making one-off backups.

Το **BigDump** είναι ένα δώρο θεού για περιορισμένους λογαριασμούς φιλοξενίας, όπου βοηθά στην επαναφορά μεγάλων αντιγράφων ασφαλείας βάσεων δεδομένων, αποφεύγοντας ταυτόχρονα το χρονικό όριο και τους περιορισμούς μεγέθους.

Το **srdb**, Search Replace DB for XOOPS είναι μια προσαρμογή XOOPS του [Search and Replace DB](https://github.com/interconnectit/Search-Replace-DB) από το interconnect/it. Είναι ιδιαίτερα χρήσιμο να αλλάζετε διευθύνσεις URL και αναφορές συστήματος αρχείων στο MySQL data when you are moving a site.

## Στοίβες προγραμματιστών

Ορισμένες πλατφόρμες, όπως το Ubuntu, έχουν ενσωματωμένη ολόκληρη τη στοίβα που απαιτείται για την εκτέλεση του XOOPS, ενώ άλλες χρειάζονται κάποιες προσθήκες.

Οι **WAMP** και **Uniform Server Zero** είναι στοίβες all-in-one για Windows.

**XAMPP**, μια στοίβα all-in-one από το Apache Friends, είναι διαθέσιμη για πολλές πλατφόρμες.

Το **bitnami** προσφέρει ένα ευρύ φάσμα προκατασκευασμένων στοίβων εφαρμογών, συμπεριλαμβανομένων εικόνων εικονικής μηχανής και κοντέινερ. Οι προσφορές τους μπορούν να αποτελέσουν πολύτιμο πόρο για τη γρήγορη δοκιμή εφαρμογών (συμπεριλαμβανομένου του XOOPS) ή διαφόρων τεχνολογιών Ιστού. Μπορούν να είναι κατάλληλα για παραγωγή αλλά και για αναπτυξιακή χρήση.

Το **Docker** είναι μια πλατφόρμα κοντέινερ εφαρμογών, που χρησιμοποιείται για τη δημιουργία και εκτέλεση κοντέινερ για την υλοποίηση προσαρμοσμένων περιβαλλόντων. 

Το **Devilbox** είναι μια στοίβα ανάπτυξης που βασίζεται εύκολα στο Docker. Προσφέρει ένα ευρύ φάσμα εκδόσεων για όλα τα στοιχεία στοίβας και επιτρέπει στους προγραμματιστές να δοκιμάζουν σε ένα αναπαραγώγιμο και κοινόχρηστο περιβάλλον.
