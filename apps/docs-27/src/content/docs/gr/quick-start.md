## Απαιτήσεις

| Συστατικό | Ελάχιστο | Συνιστάται |
|------------|-------------------------|---------------|
| PHP | 8.2 | 8,4+ |
| MySQL      | 5.7                     | 8.0+          |
| MariaDB | 10.4 | 10.11+ |
| Διακομιστής Ιστού | Apache 2.4 / Nginx 1.20 | Τελευταίο σταθερό |

## Λήψη

Κάντε λήψη της πιο πρόσφατης έκδοσης από το [GitHub Releases](https://github.com/XOOPS/XoopsCore27/releases).

```bash
# Or clone directly
git clone https://github.com/XOOPS/XoopsCore27.git mysite
cd mysite
```

## Βήματα εγκατάστασης

1. **Μεταφορτώστε αρχεία** στη ρίζα εγγράφων του διακομιστή ιστού σας (π.χ. `public_html/`).
2. **Δημιουργήστε ένα MySQL database** and a user with full privileges on it.
3. **Ανοίξτε το πρόγραμμα περιήγησής σας** και πλοηγηθείτε στον τομέα σας — το πρόγραμμα εγκατάστασης XOOPS ξεκινά αυτόματα.
4. **Ακολουθήστε τον οδηγό 5 βημάτων** — διαμορφώνει διαδρομές, δημιουργεί πίνακες και ρυθμίζει τον λογαριασμό διαχειριστή σας.
5. **Διαγράψτε το φάκελο `install/`** όταν σας ζητηθεί. Αυτό είναι υποχρεωτικό για λόγους ασφάλειας.

## Επαληθεύστε την εγκατάσταση

Μετά τη ρύθμιση, επισκεφτείτε:

- **Μπροστινή σελίδα:** `https://yourdomain.com/`
- **Πίνακας διαχειριστή:** `https://yourdomain.com/xoops_data/` *(διαδρομή που επιλέξατε κατά την εγκατάσταση)*
