---
title: "phpinfo"
---

Αυτό το βήμα είναι προαιρετικό, αλλά μπορεί εύκολα να σας εξοικονομήσει ώρες απογοήτευσης.

Ως δοκιμή προεγκατάστασης του συστήματος φιλοξενίας, δημιουργείται τοπικά ένα πολύ μικρό, αλλά χρήσιμο σενάριο PHP και μεταφορτώνεται στο σύστημα προορισμού.

Το σενάριο PHP είναι μόνο μία γραμμή:

```php
<?php phpinfo();
```

Χρησιμοποιώντας ένα πρόγραμμα επεξεργασίας κειμένου, δημιουργήστε ένα αρχείο με το όνομα _info.php_ με αυτή τη μία γραμμή.

Στη συνέχεια, ανεβάστε αυτό το αρχείο στη ρίζα ιστού σας.

![Filezilla info.php Upload](/xoops-docs/2.7/img/installation/filezilla-01-info.png)

Αποκτήστε πρόσβαση στο σενάριό σας ανοίγοντάς το στο πρόγραμμα περιήγησής σας, δηλαδή αποκτώντας πρόσβαση στο `http://example.com/info.php`. Εάν όλα λειτουργούν σωστά, θα πρέπει να δείτε μια σελίδα κάπως έτσι:

![phpinfo() Παράδειγμα](/xoops-docs/2.7/img/installation/php-info.png)

Σημείωση: ορισμένες υπηρεσίες φιλοξενίας ενδέχεται να απενεργοποιήσουν τη λειτουργία _phpinfo()_ ως μέτρο ασφαλείας. Συνήθως θα λάβετε ένα μήνυμα για το σκοπό αυτό, εάν αυτό ισχύει.

Το αποτέλεσμα του σεναρίου μπορεί να είναι χρήσιμο για την αντιμετώπιση προβλημάτων, επομένως σκεφτείτε να αποθηκεύσετε ένα αντίγραφό του.

Εάν η δοκιμή λειτουργεί, θα πρέπει να πάτε για την εγκατάσταση. Θα πρέπει να διαγράψετε το σενάριο _info.php_ και να προχωρήσετε στην εγκατάσταση.

Εάν το τεστ αποτύχει, διερευνήστε γιατί! Οποιοδήποτε πρόβλημα εμποδίζει αυτήν την απλή δοκιμή να λειτουργήσει **θα** εμποδίσει μια πραγματική εγκατάσταση να λειτουργήσει.
