Αυτή η σελίδα συλλέγει τις πληροφορίες που χρησιμοποιούνται για τη δημιουργία του κύριου λογαριασμού διαχειριστή (AKA Webmaster).

Αφού εισαγάγετε τις πληροφορίες που ζητήσατε και διορθώσετε τυχόν προβλήματα, επιλέξτε το κουμπί "Συνέχεια" για να προχωρήσετε.

![XOOPS Αρχικές ρυθμίσεις εγκατάστασης](/xoops-docs/2.7/img/installation/installer-09.png)

## Δεδομένα που συλλέγονται σε αυτό το βήμα

## # Λογαριασμός διαχειριστή

### # Είσοδος διαχειριστή

Αυτό θα είναι το όνομα χρήστη που χρησιμοποιεί ο διαχειριστής για να συνδεθείτε στο σύστημα XOOPS.

### # Ηλεκτρονικό ταχυδρομείο διαχειριστή

Αυτή είναι η διεύθυνση email που χρησιμοποιείται από το σύστημα XOOPS για την επικοινωνία με τον διαχειριστή. (Σημείωση, αυτό απαιτείται για τον κωδικό πρόσβασης recovery/reset.)

### # Κωδικός πρόσβασης διαχειριστή

Αυτός είναι ο κωδικός πρόσβασης που πρέπει να χρησιμοποιήσετε για τον λογαριασμό διαχειριστή,

### # Επιβεβαίωση κωδικού πρόσβασης

Χρησιμοποιήστε αυτό το πεδίο για να επιβεβαιώσετε τον κωδικό πρόσβασης διαχειριστή. Πρέπει να ταιριάζει με το πεδίο _Κωδικός πρόσβασης διαχειριστή_.

## # Ισχύς κωδικού πρόσβασης

Αυτή η ενότητα εμφανίζει τη σχετική ισχύ του _Κωδικού πρόσβασης διαχειριστή_.

### # Δημιουργία κωδικών πρόσβασης

Μπορείτε να χρησιμοποιήσετε αυτήν την ενότητα για να δημιουργήσετε και να χρησιμοποιήσετε έναν τυχαίο κωδικό πρόσβασης για τον λογαριασμό. Επιλέξτε _Generate_ για να εμφανίσετε έναν τυχαίο κωδικό πρόσβασης. Επιλέξτε _Αντιγραφή_ για να αντιγράψετε τον κωδικό πρόσβασης που δημιουργήθηκε στον _Κωδικός πρόσβασης διαχειριστή_.
