---
title: "Δημιουργία πίνακα"
---

Αυτή η σελίδα εμφανίζει τα αποτελέσματα της δημιουργίας των πινάκων XOOPS στη βάση δεδομένων.

Αφού ελέγξετε και διορθώσετε τυχόν προβλήματα, επιλέξτε το κουμπί "Συνέχεια" για να προχωρήσετε.

## Για την επιτυχία

Η ενότητα _Δημιουργία πινάκων βάσης δεδομένων_ εμφανίζει τους πίνακες βάσης δεδομένων που δημιουργήθηκαν.

![XOOPS Δημιουργία πίνακα εγκατάστασης](/xoops-docs/2.7/img/installation/installer-08.png)

## Σφάλματα

Τυχόν σφάλματα υποδεικνύουν προβλήματα με την άδεια που σχετίζονται με το MySQL database user account and/or MySQL database entered in previous steps. You can return to those sections using the Database connection and Database configuration options in the menu.
