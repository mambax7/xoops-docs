Αυτή η σελίδα συλλέγει τις πληροφορίες σχετικά με τη βάση δεδομένων που θα χρησιμοποιήσει το XOOPS.

Αφού εισαγάγετε τις πληροφορίες που ζητήσατε και διορθώσετε τυχόν προβλήματα, επιλέξτε το κουμπί "Συνέχεια" για να προχωρήσετε.

## Δεδομένα που συλλέγονται σε αυτό το βήμα

## # Βάση δεδομένων

### # Όνομα βάσης δεδομένων

Το όνομα της βάσης δεδομένων στον κεντρικό υπολογιστή που πρέπει να χρησιμοποιεί το XOOPS. Ο χρήστης της βάσης δεδομένων που εισήχθη στο προηγούμενο βήμα θα πρέπει να έχει όλα τα δικαιώματα σε αυτήν τη βάση δεδομένων. Το πρόγραμμα εγκατάστασης θα προσπαθήσει να δημιουργήσει αυτήν τη βάση δεδομένων εάν δεν υπάρχει.

### # Πρόθεμα πίνακα

Αυτό το πρόθεμα θα προστεθεί στα ονόματα όλων των νέων πινάκων που δημιουργήθηκαν από το XOOPS. Αυτό βοηθά στην αποφυγή διενέξεων ονομάτων εάν η βάση δεδομένων είναι κοινόχρηστη με άλλες εφαρμογές. Ένα μοναδικό πρόθεμα κάνει επίσης πιο δύσκολο να μαντέψει κανείς τα ονόματα των πινάκων, κάτι που έχει πλεονεκτήματα ασφάλειας. Εάν δεν είστε σίγουροι, απλώς διατηρήστε την προεπιλογή

### # Σύνολο χαρακτήρων βάσης δεδομένων

Το πρόγραμμα εγκατάστασης έχει οριστεί από προεπιλογή `utf8mb4 `, το οποίο υποστηρίζει την πλήρη σειρά Unicode, συμπεριλαμβανομένων των emoji και των συμπληρωματικών χαρακτήρων. Μπορείτε να επιλέξετε ένα διαφορετικό σύνολο χαρακτήρων εδώ, αλλά το ` utf8mb4` συνιστάται για όλες σχεδόν τις γλώσσες και τις τοπικές ρυθμίσεις και θα πρέπει να παραμείνει ως έχει, εκτός εάν έχετε συγκεκριμένο λόγο να το αλλάξετε.

### # Συγκέντρωση βάσεων δεδομένων

Το πεδίο ταξινόμησης παραμένει κενό από προεπιλογή. Όταν είναι κενό, MySQL applies the default collation for whichever character set was selected above (for `utf8mb4 ` this is typically ` utf8mb4_general_ci ` or ` utf8mb4_0900_ai_ci`, depending on the MySQL version). If you need a specific collation — for example to match an existing database — select it here. Otherwise, leaving it blank is the recommended choice.
