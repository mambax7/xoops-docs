Αυτή η σελίδα συλλέγει την επιλογή σας για τις ενότητες που θα εγκατασταθούν στον νέο σας ιστότοπο. Οι ενότητες μπορούν να προστεθούν ή να αφαιρεθούν ανά πάσα στιγμή, εάν το επιθυμείτε.

Αφού εισαγάγετε τις πληροφορίες που ζητήσατε και διορθώσετε τυχόν προβλήματα, επιλέξτε το κουμπί "Συνέχεια" για να προχωρήσετε.

![XOOPS Εγκατάσταση μονάδων εγκατάστασης](/xoops-docs/2.7/img/installation/installer-13.png)

## Δεδομένα που συλλέγονται σε αυτό το βήμα

## # Διαθέσιμες ενότητες

Επιλέξτε "Ναι" για να εγκαταστήσετε μια μονάδα. Μπορείτε να επιλέξετε καμία, καμία ή όλες τις ενότητες.

Αυτές είναι οι μονάδες που περιλαμβάνονται σε μια τυπική εγκατάσταση XOOPS 2.7.0.

* Προσωπικά μηνύματα - _βελτιωμένα προσωπικά μηνύματα μεταξύ χρηστών_
* Προφίλ χρήστη - _enhanced/custom προφίλ χρήστη_
* Protector - _βελτιωμένη ασφάλεια ιστότοπου, έλεγχοι ανεπιθύμητων μηνυμάτων και μετριασμός επιθέσεων_
* DebugBar - _μια γραμμή εργαλείων εντοπισμού σφαλμάτων ανάπτυξης και σταδιοποίησης που βασίζεται στο Symfony VarDumper. χρήσιμο για προγραμματιστές λειτουργικών μονάδων και για τη διάγνωση προβλημάτων παραγωγής, αλλά συνήθως δεν χρειάζεται σε δημόσιο ιστότοπο_

Η λειτουργική μονάδα _System_ είναι πάντα εγκατεστημένη και δεν αναφέρεται εδώ ως επιλογή.

### # Σημείωση

Μπορείτε να προσθέσετε λειτουργικές μονάδες στον κατάλογο λειτουργιών κατά τη μεταφόρτωση των αρχείων XOOPS πριν από την έναρξη της εγκατάστασης, και θα είναι επίσης διαθέσιμες σε αυτό το βήμα.

### # Ισχυρή σύσταση

**Όλες οι τοποθεσίες XOOPS** θα πρέπει να εγκαταστήσουν τη λειτουργική μονάδα _Protector_.
