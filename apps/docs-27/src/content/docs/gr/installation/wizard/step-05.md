Αυτή η σελίδα συλλέγει τις πληροφορίες σχετικά με τον λογαριασμό χρήστη της βάσης δεδομένων που θα χρησιμοποιήσει το XOOPS.

Αφού εισαγάγετε τις πληροφορίες που ζητήσατε και διορθώσετε τυχόν προβλήματα, επιλέξτε το κουμπί "Συνέχεια" για να προχωρήσετε.

![XOOPS Σύνδεση βάσης δεδομένων εγκατάστασης](/xoops-docs/2.7/img/installation/installer-05.png)

## Δεδομένα που συλλέγονται σε αυτό το βήμα

## # Σύνδεση διακομιστή

### # Βάση δεδομένων

Επί του παρόντος, η _mysql_ είναι η μόνη διαθέσιμη επιλογή.

### # Όνομα κεντρικού υπολογιστή διακομιστή

Όνομα κεντρικού υπολογιστή του διακομιστή βάσης δεδομένων. Εάν δεν είστε σίγουροι, το _localhost_ λειτουργεί στις περισσότερες περιπτώσεις.

### # Όνομα χρήστη

Ο λογαριασμός χρήστη της βάσης δεδομένων που πρέπει να χρησιμοποιήσει το XOOPS για να συνδεθεί στον διακομιστή βάσης δεδομένων.

### # Κωδικός πρόσβασης

Ο κωδικός πρόσβασης του λογαριασμού χρήστη της βάσης δεδομένων που εισήχθη παραπάνω.

### # Χρησιμοποιήστε μόνιμη σύνδεση

Η προεπιλογή είναι "Όχι". Αφήστε αυτό το κενό εάν δεν είστε σίγουροι.
