---
title: "Έλεγχος διαμόρφωσης"
---

Ο _Έλεγχος της διαμόρφωσης του διακομιστή σας_ είναι μια επισκόπηση του περιβάλλοντος στο οποίο εκτελείται το πρόγραμμα εγκατάστασης.

Εάν αναφερθούν προβλήματα στην ενότητα _Απαιτήσεις_, αυτά πρέπει να διορθωθούν, διαφορετικά το XOOPS δεν θα λειτουργεί.

Η ενότητα _Συνιστώμενες επεκτάσεις_ υποδεικνύει πρόσθετες επεκτάσεις PHP τις οποίες μπορεί να χρησιμοποιήσουν οι μονάδες XOOPS ή XOOPS για την παροχή πρόσθετων λειτουργιών. Μπορείτε να προχωρήσετε στην εγκατάσταση, ακόμα κι αν δεν υπάρχουν διαθέσιμες επεκτάσεις. Αλλά, εάν μπορείτε να τα ενεργοποιήσετε, συνιστάται.

Μετά τον έλεγχο, επιλέξτε το κουμπί "Συνέχεια" για να προχωρήσετε.
