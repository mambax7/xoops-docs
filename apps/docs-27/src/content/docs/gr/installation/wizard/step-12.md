Αυτή η σελίδα συλλέγει την επιλογή σας για το αρχικό θέμα για τον νέο σας ιστότοπο. Τα θέματα ελέγχουν τον οπτικό χαρακτήρα του ιστότοπού σας. Τα θέματα μπορούν να αλλάξουν ανά πάσα στιγμή, εάν το επιθυμείτε.

Αφού εισαγάγετε τις πληροφορίες που ζητήσατε και διορθώσετε τυχόν προβλήματα, επιλέξτε το κουμπί "Συνέχεια" για να προχωρήσετε.

## Δεδομένα που συλλέγονται σε αυτό το βήμα

## # Επιλέξτε το προεπιλεγμένο θέμα

### # Προεπιλεγμένο θέμα

Επιλέξτε ένα αρχικό θέμα για τον ιστότοπό σας από τη λίστα. Ένα στιγμιότυπο οθόνης για κάθε θέμα εμφανίζεται καθώς έχει επιλεγεί.

Το XOOPS 2.7.0 αποστέλλεται με πολλά θέματα στο μπροστινό μέρος εκτός συσκευασίας:

* **προεπιλογή** — ένα ελάχιστο βασικό θέμα, χρήσιμο ως σημείο εκκίνησης για προσαρμογή.
* **xbootstrap5** — Θέμα Bootstrap 5, συνιστάται για τους περισσότερους νέους ιστότοπους.
* **xswatch5** — Bootswatch 5 παραλλαγές. ένα σύνολο υποθεμάτων με στυλ πάνω από το Bootstrap 5.
* **xtailwind** — Θέμα που βασίζεται σε Tailwind CSS + DaisyUI με 35 παλέτες και διαδραστικότητα Alpine.js
* **xtailwind2** — καλλιτεχνικό θέμα αδερφού με επιμελημένες παλέτες και εκλεπτυσμένο στυλ ενότητας

Μπορείτε να αλλάξετε το θέμα ανά πάσα στιγμή από την περιοχή διαχείρισης και επιπλέον θέματα μπορούν να φορτωθούν στον κατάλογο `themes/`.
