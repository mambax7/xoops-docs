Tato stránka shromažďuje váš výběr pro počáteční téma pro váš nový web. Motivy řídí vizuální charakter vašeho webu. Motivy lze v případě potřeby kdykoli změnit.

Po zadání požadovaných informací a opravě případných problémů pokračujte kliknutím na tlačítko „Pokračovat“.

![XOOPS Installer Select Theme](/xoops-docs/2.7/img/installation/installer-12.png)

## Údaje shromážděné v tomto kroku

### Vyberte výchozí motiv

#### Výchozí motiv

Ze seznamu vyberte počáteční motiv pro svůj web. Při výběru každého motivu se zobrazí snímek obrazovky.

XOOPS 2.7.0 se dodává s několika front-end tématy hned po vybalení:

* **výchozí** — minimální základní téma, užitečné jako výchozí bod pro přizpůsobení.
* **xbootstrap5** — Motiv Bootstrap 5, doporučený pro většinu nových webů.
* **xswatch5** — Bootswatch 5 variant; sada stylizovaných podtémat nad Bootstrap 5.
* **xtailwind** — Motiv založený na Tailwind CSS + DaisyUI s 35 paletami a interaktivitou Alpine.js
* **xtailwind2** — umělecky zaměřený sourozenecký motiv s upravenými paletami a rafinovaným stylem modulů

Motiv můžete kdykoli změnit v oblasti administrace a další motivy lze nahrát do adresáře `themes/`.
