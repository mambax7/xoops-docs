Tato část vás provede upgradem stávajícího webu XOOPS. Průvodce upgradem zachová vaše data, moduly a motivy při aktualizaci základních souborů.

## Upgrade cesta

| Krok | Strana | Co dělat |
|------|------|------------|
| 1 | [Přípravky](./step-01/) | Zálohování, kontrola seznamu změn, stahování nového vydání |
| 2 | [Předletová kontrola](./preflight/) | Před spuštěním upgradu ověřte kompatibilitu |
| 3 | [Spuštění upgradu](./step-02/) | Spusťte průvodce upgradem |
| 4 | [Odstraňování problémů](./step-03/) | Vyřešte běžné problémy s upgradem |
| 5 | [Po upgradu](./step-04/) | Ověřit, aktualizovat moduly, vymazat mezipaměť |

:::tip
Před upgradem vždy proveďte **úplnou zálohu databáze a souborů**. Upgrade je nevratný bez zálohy.
:::
